# 事实性描述专项审阅规则

## 适用范围

政策背景说明、行业现状描述、研究对象现状介绍、历史沿革、统计数据呈现、表格/图表说明、案例事实介绍、现实问题陈述、制度安排描述、调研材料中的客观事实摘要。

## 事实性描述的五项核心任务

1. **提供研究背景**：说明问题出现于何种现实场景
2. **提供研究对象信息**：说明对象的基本情况
3. **提供证据基础**：为后续分析提供原始材料
4. **提供制度/政策框架**：说明规则环境
5. **提供案例事实**：为案例分析建立事实基础

## 六大核心审查维度

### D1：准确性
- **核心问题**：事实本身是否准确？
- **检查**：时间/地点/主体/数字/概念/事件关系 / 是否把推断写成事实
- **风险**：年份错/口径混/主体名不准/用模糊说法掩盖不确定性
- **动作**：核实数据 / 校对时间 / 改写不确定表述
- **批注**："此处事实判断建议再次核对时间与主体。"

### D2：来源可追溯性
- **核心问题**：读者能否知道事实从哪里来？
- **检查**：是否有明确来源 / 来源是否权威 / 是否存在"据相关资料显示"
- **风险**：无来源 / 来源不权威 / 来源与对象不匹配
- **动作**：补充注释 / 更换来源 / 明确出处
- **批注**："来源尚不明确，建议补充可追溯出处。"

### D3：完整性
- **核心问题**：是否遗漏了理解事实所必需的前提和范围？
- **检查**：是否交代场景 / 时间区间 / 比较对象 / 统计范围
- **风险**：只报数字不说口径 / 只摘有利信息 / 断章取义
- **动作**：补充前提/范围/比较基准/关键背景
- **批注**："缺少必要的范围或口径说明。"

### D4：边界性
- **核心问题**：有没有越过事实可支持的边界？
- **检查**：是否以偏概全 / 相关写成因果 / 短期变化写成长期趋势
- **风险**：从局部推整体 / 从现象跳到原因 / 政策文本直接等同执行效果
- **动作**：降低判断强度 / 加限定词 / 改因果为相关
- **批注**："现有事实更适合支持趋势性判断。"

### D5：解释性
- **核心问题**：是否只"摆出事实"却没说明其对论题的意义？
- **风险**：数据堆砌 / 列政策条文但不说作用 / 事实和论点间缺桥梁句
- **动作**：增加解释句 / 增加桥梁句 / 删无关事实
- **批注**："数据已呈现，但与中心判断的关系仍可再解释一步。"

### D6：规范性
- **核心问题**：呈现方式是否符合学术写作规范？
- **检查**：图表来源 / 数据单位 / 时间格式 / 数字表达 / 表格与正文一致性
- **风险**：图表无来源 / 正文与表格数据不一致 / 政策名称混乱
- **动作**：统一格式 / 补图表来源 / 核对表图正文一致性
- **批注**："图表信息建议补充来源与统计口径说明。"

---

## 专项风险库

| 风险 | 表现 | 动作 |
|---|---|---|
| 背景堆砌 | 大量背景但与主线关系弱 | 删无关背景，收束到问题 |
| 事实空泛化 | "近年来""显著"等高频抽象词 | 具体化，数字化 |
| 来源模糊化 | "有资料表明""据统计"无出处 | 补注释，明确机构年份 |
| 数据陈列化 | 只列数据没有解释 | 增加解释，加比较基线 |
| 事实越界推断 | 从局部推整体，现象跳因果 | 收束判断，降低强度 |
| 案例失真 | 案例像新闻故事，关键事实不全 | 补关键事实，说明代表性 |

---

## 问题分级

- **S级**：核心事实错误 / 核心数据不可信 / 支撑结论的事实无来源
- **A级**：大面积来源模糊 / 数据与论点长期脱节 / 以偏概全反复出现
- **B级**：信息不完整 / 解释不足 / 表述强度偏高
- **C级**：数字格式不统一 / 图表来源位置不规范

---

## 复审清单

1. 核心事实是否已核准
2. 来源是否已补齐
3. 时间、数字、主体是否统一
4. 背景是否更凝练
5. 数据是否被解释
6. 是否还存在以偏概全
7. 是否与研究问题更紧密连接
