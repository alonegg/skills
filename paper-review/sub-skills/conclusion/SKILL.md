# 结论与建议专项审阅规则

## 适用范围

研究结论、研究发现总结、实践/政策/管理建议、理论启示、实践启示、局限与不足、未来展望、结尾性综合判断段。

## 结论与建议的六项核心任务

1. **回应研究问题**：结论必须回到一开始提出的问题
2. **提炼研究发现**：不重复正文，层次化概括关键发现
3. **控制结论边界**：与证据强度匹配
4. **提出可操作建议**：有对象、路径、条件、目标
5. **说明局限**：对象/样本/方法/材料/结论适用范围
6. **形成收束**：让整篇论文完成闭环

## 七大核心审查维度

### D1：回应研究问题的程度
- **核心问题**：结论是否真正回到了题目和研究问题？
- **风险**：结论与问题脱节 / 只回应部分问题 / 关键维度消失
- **动作**：按研究问题重写结论 / 增加回扣句 / 对齐题目摘要结论
- **批注**："与前文核心研究问题的对应关系仍可再明确。"

### D2：研究发现提炼度
- **核心问题**：是在提炼发现，还是只把正文压缩复述？
- **风险**：结论像正文缩写版 / 没有层次 / 细节太多反而不成结论
- **动作**：用"第一、第二、第三"分层 / 删除过程性叙述
- **批注**："研究发现仍可进一步提炼和分层。"

### D3：结论边界与强度控制
- **核心问题**：结论与证据强度是否匹配？
- **风险**：从有限材料推过强结论 / 把案例写成普遍规律 / 过于绝对
- **动作**：加限定词 / 收束结论 / 将普遍结论改为条件性结论
- **批注**："建议根据前文材料适当收束边界。"

### D4：建议的可操作性
- **核心问题**：建议是否真正可执行？
- **风险**："加强""完善""提升"式空泛建议 / 没主体没路径没条件
- **动作**：明确"谁做什么" / 明确条件和目标 / 将口号改为可执行动作
- **批注**："建议方向明确，但仍可进一步写实到具体主体和行动路径。"

### D5：与研究发现的对应关系
- **核心问题**：建议是否都来自前文发现，而非凭空提出？
- **风险**：建议正确但不是本文研究出来的 / 结尾突然引入新议题
- **动作**：为每条建议添加发现依据 / 在建议前加桥梁句
- **批注**："建议与前文发现的对应关系还可进一步显性化。"

### D6：局限意识
- **核心问题**：是否清楚研究的边界和不足？
- **风险**：没有局限 / 局限只是敷衍 / 与本文实际问题无关
- **动作**：增加局限段 / 将局限具体化到对象/方法 / 与展望衔接
- **批注**："若能补充对象/方法上的局限，本节会更完整。"

### D7：整体收束能力
- **核心问题**：是否让整篇论文形成闭环？
- **风险**：结尾松散 / 突然结束 / 过于拔高 / 口号化
- **动作**：增加首尾呼应句 / 用"问题—发现—建议—边界"完成闭环
- **批注**："若能进一步回扣开篇问题，会更完整。"

## 专项风险库

| 风险 | 表现 | 动作 |
|---|---|---|
| 重复正文 | 结论像正文压缩版 | 提炼发现，删过程性内容 |
| 结论过大 | 证据有限结论很满 | 收束结论，加限定词 |
| 建议口号化 | "加强""完善"等抽象动词 | 明确主体/动作/条件/目标 |
| 建议与发现脱节 | 建议正确但非本文研究所得 | 建立"发现—建议"映射 |
| 局限失语 | 没有或敷衍局限 | 增加对象/方法/材料边界说明 |
| 结尾松散 | 无首尾呼应，未完成闭环 | 增加收束句，回扣题目 |

---

## 问题分级

- **S级**：结论与研究问题严重脱节 / 核心结论无正文支撑 / 建议完全脱离发现
- **A级**：发现提炼失败 / 建议大面积口号化 / 局限意识严重不足
- **B级**：结论层次不清 / 建议可操作性不足 / 收束不够有力
- **C级**：个别句子重复 / 局限表述略泛 / 首尾呼应不足

---

## 复审清单

1. 是否更直接回应研究问题
2. 是否真正提炼出研究发现
3. 结论是否与证据强度匹配
4. 建议是否更具体可操作
5. 建议是否与发现形成对应
6. 是否增加了局限与边界说明
7. 是否形成更强的全文闭环
