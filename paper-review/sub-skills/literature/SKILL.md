# 综述性描述专项审阅规则

## 适用范围

文献综述、国内外研究现状、理论研究回顾、相关研究梳理、学术争议总结、研究脉络归纳、方法路径比较、概念演进回顾。

## 综述的五项核心任务

1. **信息整理**：把零散文献整理成可理解的研究版图
2. **比较分类**：识别不同观点、方法、路径与分歧
3. **评价判断**：指出已有研究的优点、不足和边界
4. **研究定位**：说明本文在已有研究中的位置
5. **过渡铺垫**：引向本文研究问题、理论或方法

## 识别系统

### 语言线索
"已有研究主要集中于……""学者普遍认为……""相关文献大致可分为……""然而现有研究仍存在……"

### 子类型
R1 研究脉络型 / R2 主题分类型 / R3 观点比较型 / R4 方法比较型 / R5 理论回顾型 / R6 争议梳理型 / R7 研究空白型

---

## 六大核心审查维度

### D1：覆盖面
- **核心问题**：是否覆盖代表性研究？
- **检查**：核心研究 / 经典+近期 / 是否只选支持预设观点的文献
- **风险**：文献太少 / 只看单一来源 / 代表性缺失
- **动作**：补代表性文献 / 加入经典研究或关键近作
- **批注**："代表性研究的覆盖还可进一步加强。"

### D2：组织方式
- **核心问题**：是按问题组织还是按作者机械堆放？
- **检查**：是否按主题/方法/观点/争议组织 / 是否有分类逻辑
- **风险**：一人一句 / 时间顺序流水账 / 段内跳跃
- **动作**：按主题归类 / 在段首补分类句
- **批注**："目前仍以作者顺序展开，建议改按主题或观点组织。"

### D3：比较能力
- **核心问题**：是否在比较不同研究，而非只是依次转述？
- **检查**：是否比较异同 / 是否指出分歧 / 是否建立文献关系
- **风险**：文献彼此孤立 / 像资料摘录合集
- **动作**：增加比较句 / 总结共同点与差异点
- **批注**："不同研究之间的异同仍可进一步比较。"

### D4：评述能力
- **核心问题**：是否对已有研究做出判断，而非只转述？
- **检查**：是否指出优势与局限 / 是否有归纳性判断 / 是否"综而不述"
- **风险**：只有介绍没有评价 / 作者声音弱
- **动作**：增加归纳句 / 增加局限判断 / 增加作者立场句
- **批注**："评述力度偏弱，建议补充对已有研究局限的判断。"

### D5：研究位置建构
- **核心问题**：综述是否成功引出"本文为什么还要写"？
- **检查**：是否指出研究空白 / 是否明确切入点 / 是否与题目一致
- **风险**：介绍完就结束 / 空白说太空 / 切入点不清
- **动作**：补写研究空白 / 补写本文切口 / 将末句改为"前人—本文"结构
- **批注**："本文的研究位置仍不够清晰，建议在段末明确切入点。"

### D6：引文质量
- **核心问题**：引用本身是否准确、规范、恰当？
- **检查**：是否准确表达原意 / 是否断章取义 / 格式是否规范
- **风险**：误引 / 拼接式改写 / 堆引无主线
- **动作**：核对引文 / 精简引用 / 改为概括式引用后评述
- **批注**："引文较密集，建议突出真正关键的研究。"

## 专项风险库

| 风险 | 表现 | 动作 |
|---|---|---|
| 作者流水账 | 一人一观点，无分类 | 按主题/观点/方法重组 |
| 只综不述 | 只有"谁说了什么" | 补总结句、评价句、边界句 |
| 研究位置缺失 | 综述后仍不知本文为何要写 | 补研究空白和切入点 |
| 综述与本文脱节 | 文献多但与研究问题关系弱 | 删无关文献，收束范围 |
| 引文堆砌 | 一句堆多个引用 | 精简引用，增加作者归纳 |
| 空白套话化 | "有待进一步研究"但不说具体 | 具体化不足的对象/方法/视角 |

---

## 问题分级

- **S级**：核心综述无法引出研究位置 / 关键文献严重缺失 / 综述与题目严重脱节
- **A级**：大面积流水账 / 只综不述 / 覆盖狭窄 / 空白识别失效
- **B级**：比较不够 / 分类不够清楚 / 引文略堆砌
- **C级**：引用格式不统一 / 少量重复介绍 / 措辞不够凝练

---

## 复审清单

1. 是否从"作者流水账"转向"分类综述"
2. 是否增加了比较句和评述句
3. 是否更明确地指出研究不足
4. 是否形成本文研究位置
5. 是否减少了无关文献与堆砌引用
6. 综述与题目、研究问题、后文分析是否更一致
