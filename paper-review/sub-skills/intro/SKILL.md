# 问题提出 / 引言专项审阅规则

## 适用范围

引言、绪论、研究背景、问题提出、研究意义、研究目标、研究问题、研究思路、论文结构安排、研究范围界定。

## 引言的七项核心任务

高质量引言应完成：
1. 交代研究背景（现实/学术/制度）
2. 提出核心问题
3. 限定研究对象与边界
4. 说明研究必要性
5. 建立研究位置（相对已有研究的切入点）
6. 概述研究路径
7. 导入全文结构

## 七大核心审查维度

### D1：问题意识强度
- **核心问题**：是否真正提出了可研究的问题，而非停留在背景陈述？
- **检查**：是否明确问题 / 是否具体 / 是否从背景自然过渡到问题
- **风险**：问题淹没在背景里 / 问题过大 / 问题口号化 / 引言看完仍不知研究什么
- **动作**：补问题句 / 将现象改写为问题 / 收缩问题范围
- **批注**："本段背景已有铺垫，但核心研究问题仍可再明确一些。"

### D2：研究对象与边界聚焦度
- **核心问题**：研究对象是谁、什么范围、什么条件？
- **检查**：对象是否明确 / 边界是否清楚 / 是否与题目一致
- **风险**：对象过大 / 没有边界 / 标题具体但引言泛化
- **动作**：明确对象 / 补时间空间场景条件 / 对齐题目
- **批注**："研究对象已出现，但边界仍可进一步明确。"

### D3：背景与问题的连接度
- **核心问题**：背景是否真正服务于问题提出？
- **检查**：是否与问题直接相关 / 是否过多宏观背景 / 能否自然过渡
- **风险**：背景过长 / 背景与问题脱节 / 套话过多
- **动作**：压缩背景 / 删除常识叙述 / 增加桥梁句
- **批注**："背景铺陈较充分，但与核心问题的连接仍可更紧。"

### D4：研究必要性与意义的真实性
- **核心问题**：是否真正说明了为什么值得研究？
- **检查**：现实重要性 / 学术空缺 / 是否避免空洞套话
- **风险**：意义全是套话 / 必要性与内容不匹配 / 没有独特性
- **动作**：将意义具体化 / 区分现实与研究必要性 / 删模板表述
- **批注**："研究意义已有表达，但仍可更具体地落到本文对象与问题上。"

### D5：研究问题与目标的可回答性
- **核心问题**：问题在本科论文能力范围内是否可回答？
- **检查**：问题是否过大 / 目标是否过多 / 材料方法篇幅能否承载
- **风险**：同时想解决太多问题 / 引言承诺太多正文接不住
- **动作**：缩减问题数量 / 将宏大目标改为可回答任务
- **批注**："研究目标方向明确，但仍可进一步收束。"

### D6：研究位置与切入点清晰度
- **核心问题**：读者是否知道本文从哪里切入？
- **检查**：是否说明切入点 / 是否区别于已有研究 / 是否引出后文
- **风险**：只说话题重要 / 切入点过空 / 引言与文献综述衔接差
- **动作**：补切入点句 / 明确视角方法特色 / 与后文结构挂钩
- **批注**："本文的具体切入点仍可进一步明确。"

### D7：结构导入与全文一致性
- **核心问题**：引言是否帮助读者理解后文结构？
- **检查**：是否说明各章功能 / 结构与问题是否对应 / 是否与实际一致
- **风险**：机械报章名 / 章结构与研究任务不对应
- **动作**：用"本章解决什么问题"方式重写 / 对齐目录与引言
- **批注**："结构安排已有说明，但各章任务间逻辑关系仍可再显性化。"

---

## 专项风险库

| 风险 | 表现 | 动作 |
|---|---|---|
| 背景淹没问题 | 背景长，问题晚出现 | 压缩背景，前置问题句 |
| 问题口号化 | 问题像政策宣传 | 具体化，限定对象和场景 |
| 对象失焦 | 对象过大，题目与引言不一致 | 收缩范围，补边界 |
| 意义套话化 | "理论和实践意义"模板 | 删模板句，改为具体必要性 |
| 承诺过量 | 引言目标太多 | 压缩目标，聚焦主问题 |
| 结构报目录化 | 只列章名，无逻辑推进 | 改写为"问题—任务—结构"导入 |

---

## 问题分级

- **S级**：研究问题不清 / 对象严重失焦 / 引言与正文任务不一致
- **A级**：背景淹没问题 / 意义套话化严重 / 目标过量 / 切入点不清
- **B级**：对象边界不够清晰 / 背景略长 / 结构说明略泛
- **C级**：个别表述冗长 / 过渡不顺 / 目录导入不够凝练

---

## 复审清单

1. 核心研究问题是否更明确
2. 研究对象与边界是否更清楚
3. 背景是否更服务于问题
4. 研究意义是否减少套话
5. 研究目标是否更可回答
6. 切入点是否更清晰
7. 结构导入是否更具逻辑
8. 引言与题目、摘要、正文是否更一致
