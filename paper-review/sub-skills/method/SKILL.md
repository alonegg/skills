# 方法说明专项审阅规则

## 适用范围

研究方法说明、研究设计、研究对象与样本说明、数据来源、材料筛选标准、调查/访谈/问卷设计、案例选择依据、分析流程、可信度与局限性说明。

## 方法说明的六项核心任务

1. 回答"为什么用这个方法"（方法与问题匹配）
2. 回答"研究对象是谁"（对象与样本）
3. 回答"材料从哪里来"（来源说明）
4. 回答"过程怎么走"（流程透明）
5. 回答"标准是什么"（工具与判断依据）
6. 回答"方法的边界在哪里"（局限意识）

方法部分的关键功能是让读者确信：从问题到结论的路径是可理解、可追溯、可信赖的。

## 七大核心审查维度

### D1：方法与问题的适配性
- **核心问题**：方法能否回答本文的研究问题？
- **风险**：问题需解释因果却只做描述 / 同时列多种方法但都没展开
- **动作**：解释方法选择理由 / 删除装饰性方法 / 增加"问题—方法"连接句
- **批注**："方法已有说明，但与核心研究问题的对应关系仍可更明确。"

**好的连接句示例**：
> "本文研究问题侧重于X的形成机制，因此选择案例研究法，通过对典型案例的深度分析来揭示机制运作过程。"

**需改进的示例**：
> "本文采用文献分析法、比较分析法、案例分析法和系统分析法。"
> → 问题：方法堆砌，每种方法都没展开，也没说明为什么选这些方法

### D2：对象与样本清晰度
- **核心问题**：读者能否清楚知道研究对象及其边界？
- **风险**：对象模糊 / 只说"若干案例" / 与标题不一致
- **动作**：补对象界定 / 补时间空间数量范围 / 对齐标题
- **批注**："研究对象已提到，但范围与边界仍可进一步明确。"

### D3：来源可说明性
- **核心问题**：材料和数据从哪里来？
- **风险**：只说"来自网络" / 来源与研究问题不匹配
- **动作**：补来源说明 / 说明获取方式 / 区分材料类型
- **批注**："材料来源仍可更具体，以增强方法透明度。"

### D4：流程透明度
- **核心问题**：读者能否看懂从材料到结论的过程？
- **风险**：只说"进行分析"不说怎么分析 / 流程过粗 / 缺关键环节
- **动作**：分步骤写 / 补关键环节 / 用序列结构重写
- **批注**："研究流程仍可进一步展开。"

**好的流程描述示例**：
> "数据分析分三步进行：第一步，对访谈录音进行逐字转录并编码；第二步，采用主题分析法归纳核心主题；第三步，将主题与理论框架进行对照分析。"

### D5：工具与标准明确度
- **核心问题**：依据什么标准筛选、分类、比较或判断？
- **风险**：说做了分类但不说怎么分 / 说选了案例但不说为何
- **动作**：补筛选标准 / 补分类维度 / 补工具说明
- **批注**："若能补充筛选/分类标准，方法逻辑会更完整。"

### D6：可信度与有效性意识
- **核心问题**：是否意识到方法的偏差和局限？
- **风险**：完全不谈局限 / 方法像万能钥匙
- **动作**：增加局限段 / 提示偏差来源 / 收束结论边界
- **批注**："若能补充方法局限，本节会更完整也更稳妥。"

### D7：结果可理解性
- **核心问题**：方法是否足以帮读者理解后文分析和结论是如何得出的？
- **风险**：方法和结果像两篇文章 / 后文用到的维度方法部分没交代
- **动作**：补方法与结果连接句 / 在方法部分预告分析维度
- **批注**："若能说明本方法如何导向后文结果，连接会更清楚。"

## 专项风险库

| 风险 | 表现 | 动作 |
|---|---|---|
| 方法堆砌 | 列出很多方法但都没展开 | 保留主方法，删装饰性方法 |
| 对象模糊 | 边界不清，总体与样本混淆 | 界定对象，补范围 |
| 流程空洞 | "进行分析"类表达多，步骤缺失 | 拆分步骤，标出关键操作 |
| 标准缺失 | 做了筛选分类但无判断依据 | 写明标准/维度/工具 |
| 方法与问题脱节 | 方法不能支持问题 | 改写问题或调整方法 |
| 局限失语 | 方法写得过满，无边界意识 | 增加局限和偏差提示 |

---

## 问题分级

- **S级**：方法无法支撑研究问题 / 对象不清 / 核心数据来源不明
- **A级**：流程长期不透明 / 标准长期缺失 / 方法堆砌无主方法
- **B级**：样本边界不够清楚 / 来源略笼统 / 方法与结果连接弱
- **C级**：个别术语不统一 / 表述不够精炼

---

## 叠加规则提示

- 若方法部分实质在说明案例事实 → 叠加 `fact` 规则，检查事实准确性和来源
- 若方法部分包含文献方法比较 → 叠加 `literature` 规则，检查比较是否充分
- 方法部分中的理论框架介绍 → 注意检查是否与后文分析的理论使用一致

---

## 复审清单

1. 方法是否更贴合研究问题
2. 对象与样本是否更明确
3. 来源是否更清楚
4. 流程是否更透明
5. 标准是否更显性
6. 方法与结果是否更连通
7. 是否增加了局限说明
