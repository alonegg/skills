# 分析论证专项审阅规则

## 适用范围

结果分析、讨论部分、案例分析、机制解释、对比分析、原因分析、影响分析、路径分析、结论前的综合判断段、数据或案例后的解释性段落。

## 分析论证的六项核心任务

1. **提出主张**：说清楚要判断什么
2. **连接证据**：主张必须有事实、数据、案例支撑
3. **展开解释**：说明"为什么会这样"
4. **建立逻辑**：有内在推理链
5. **回应研究问题**：服务于论文主线
6. **形成推进**：段落间有递进、转折、补充关系

## 七大核心审查维度

### D1：主张清晰度
- **核心问题**：这一段到底在判断什么？
- **风险**：中心句缺失 / 多个主张混在一起 / 标题像分析但正文只描述
- **动作**：补写中心句 / 收束到一个主张 / 拆分段落
- **批注**："本段材料较充分，但中心判断仍可再明确一点。"

### D2：证据匹配度
- **核心问题**：主张有没有被证据真正支撑？
- **风险**：只一个例子推一般判断 / 数据与主张不匹配 / 以常识替代证据
- **动作**：补证据 / 更换更直接证据 / 降低结论强度
- **批注**："支撑判断的证据仍可更直接。"

### D3：解释深度
- **核心问题**：有没有真正解释"为什么"，还是只重复现象？
- **风险**：只有现象没有原因 / 机械复述图表 / "说明了""反映出"但没展开
- **动作**：增加原因解释 / 从"是什么"推进到"为什么" / 增加条件分析
- **批注**："本段已有现象描述，建议继续追问背后的原因或机制。"

### D4：理论关联度
- **核心问题**：是否与全文理论框架形成稳定联系？
- **风险**：理论只在综述出现 / 分析脱离概念系统 / 理论贴标签式使用
- **动作**：补理论连接句 / 用理论维度重组分析 / 让段末回扣概念
- **批注**："若能补一句与理论框架的对应关系，会更完整。"

### D5：逻辑推进性
- **核心问题**：段内与段间是否形成清楚的逻辑推进？
- **风险**：句子都对但拼不成论证链 / 段首突然跳转 / 前后段说同一件事
- **动作**：重组段内顺序 / 补过渡句 / 合并重复段
- **批注**："本段论述顺序略显跳跃，建议按'主张—证据—解释'重排。"

### D6：边界与反驳意识
- **核心问题**：是否知道判断有边界？
- **风险**：结论写得过满 / 从有限材料推出唯一解释 / 以论断替代论证
- **动作**：加限定语 / 加边界说明 / 增加"另一种可能解释"
- **批注**："此处判断较有力度，若能补充适用边界会更稳妥。"

### D7：回应研究问题的程度
- **核心问题**：本段是否真正服务于全文研究问题？
- **风险**：分析本身不差但偏离主线 / 一段段在分析但全篇拼不成回答
- **动作**：增加回扣研究问题的句子 / 删除偏题分析
- **批注**："本段分析已有展开，但与全文核心问题的对应关系还可再明确。"

## 专项风险库

| 风险 | 表现 | 动作 |
|---|---|---|
| 描述冒充分析 | 换方式复述事实 | 补中心判断，补解释句 |
| 结论悬空 | 判断强但证据弱 | 补证据，降低强度 |
| 理论失联 | 前文有理论但分析不用 | 补理论对应，用维度重组 |
| 分析扁平化 | 停留在"有影响""比较重要" | 细化因素/机制/条件 |
| 逻辑断裂 | 段内跳跃，段间衔接弱 | 重排顺序，补过渡 |
| 边界缺失 | 直接下大结论 | 加限定词，交代条件 |

---

## 问题分级

- **S级**：核心分析无主张 / 关键结论无证据 / 核心推理断裂 / 与研究问题严重脱节
- **A级**：大面积停留描述层 / 理论与分析长期失联 / 结论系统性过满
- **B级**：中心句不够明确 / 解释深度不足 / 过渡略弱
- **C级**：个别措辞不精准 / 段末小结缺失

---

## 复审清单

1. 中心判断是否更明确
2. 证据是否更贴近主张
3. 是否从描述推进到解释
4. 是否补上了理论连接
5. 段内顺序是否更顺、段间承接是否更清楚
6. 是否降低了过度结论
7. 是否更直接回应研究问题
