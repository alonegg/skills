# 医学护理类 学科配置

> 本文件是 paper-review 技能的学科叠加层。它不替换通用规则，而是在通用规则基础上增加医学护理类学科特有的审查维度、调整权重、补充常见问题模式。医学护理类是证据驱动型学科，方法章节的规范性和伦理合规性是审查的核心关注点。

## 适用范围

- **覆盖学科**：临床医学、预防医学、中医学、药学、护理学、康复治疗学、医学检验技术、公共卫生
- **典型论文类型**：
  - 临床研究型（Clinical Research）——前瞻性或回顾性队列研究、临床试验
  - 回顾性分析型（Retrospective Analysis）——病历资料回顾分析、数据库挖掘
  - 调查研究型（Survey Research）——横断面调查、知信行调查、现状调查
  - 实验研究型（Experimental Study）——基础医学实验、动物实验、体外实验
  - 病例报告型（Case Report/Series）——典型或罕见病例报告、系列病例分析
  - 护理干预型（Nursing Intervention）——护理方案设计与效果评价
- **识别线索**：出现"患者""病例""临床""诊断""治疗""护理""康复""药物""量表""问卷调查""知信行""伦理审批""知情同意""纳入标准""排除标准""随机对照""回顾性""前瞻性""队列""病例对照""横断面""IRB""Helsinki""CONSORT""STROBE""PRISMA""P值""OR""RR""HR""95%CI""Kaplan-Meier""Cox回归""logistic回归""中医辨证""方剂""穴位""护理干预""健康教育"等关键词；引用医学期刊（《中华医学杂志》《中华护理杂志》等）；涉及疾病、药物、护理措施等主题

## 结构期望

### 标准章节结构

医学护理类本科论文按论文类型有六种典型结构：

**临床研究型**
```
第1章 引言（研究背景、文献综述、研究目的与意义）
第2章 资料与方法（研究设计、研究对象、纳入排除标准、伦理审批、干预措施/暴露定义、结局指标、统计方法）
第3章 结果（基线资料比较、主要结局、次要结局、亚组分析）
第4章 讨论（结果解释、与已有研究比较、机制探讨、局限性）
第5章 结论
```

**回顾性分析型**
```
第1章 引言（研究背景、临床问题、研究目的）
第2章 资料与方法（资料来源、纳入排除标准、数据提取方法、变量定义、统计方法）
第3章 结果（一般资料描述、单因素分析、多因素分析）
第4章 讨论（结果解释、临床意义、与已有研究比较、回顾性设计的局限）
第5章 结论
```

**调查研究型**
```
第1章 引言（研究背景、研究现状、研究目的）
第2章 对象与方法（调查对象、抽样方法、调查工具、质量控制、统计方法）
第3章 结果（调查对象一般资料、各维度得分、影响因素分析）
第4章 讨论（现状分析、影响因素讨论、对策建议）
第5章 结论
```

**护理干预型**
```
第1章 引言（研究背景、护理现状、研究目的与意义）
第2章 文献综述（相关概念界定、国内外护理干预研究进展）
第3章 研究方法（研究设计、研究对象、干预方案详述、评价指标与工具、伦理审批、统计方法）
第4章 结果（基线可比性、干预效果评价、患者满意度/依从性）
第5章 讨论（干预效果分析、护理实践启示、局限与展望）
第6章 结论
```

**病例报告型**
```
第1章 引言（疾病概述、报告目的）
第2章 病例资料（主诉、现病史、既往史、体格检查、辅助检查、诊断过程、治疗经过、转归）
第3章 文献复习（类似病例的文献回顾、诊治要点总结）
第4章 讨论（本例特殊性分析、诊治经验与教训）
第5章 结论
```

**中医研究型**
```
第1章 引言（病证概述、中医认识沿革、研究目的）
第2章 文献综述（古籍论述、现代研究进展）
第3章 资料与方法（辨证标准、纳入排除标准、治疗方案/方剂组成、疗效评价标准、统计方法）
第4章 结果（一般资料、疗效评价、中医证候积分变化、安全性评价）
第5章 讨论（方义分析、现代药理学佐证、与已有研究比较）
第6章 结论
```

### 必须有的章节

- **引言**：必须包含研究背景、临床问题的明确界定、研究目的；临床研究必须说明研究的临床意义和公共卫生价值
- **资料与方法/对象与方法**：医学论文的核心章节，必须详尽到其他研究者可据此复现研究。最低要求：研究设计类型明确声明、研究对象的纳入和排除标准、伦理审批信息（涉及人的研究）、干预/暴露的详细描述、结局指标的定义与测量方法、统计分析方法
- **结果**：必须与讨论分开，纯粹报告数据分析结果；基线资料的组间可比性必须先行报告
- **讨论**：必须将结果与已有文献进行比较，探讨临床意义和可能的机制
- **结论**：必须直接回应研究目的，结论的强度须与研究设计的证据等级相匹配

### 可选/特殊章节

- **文献综述**：调查研究型和护理干预型通常需要独立成章；临床研究型可在引言中整合
- **伦理声明**：涉及人体研究时必须有明确的伦理审批段落（可在方法章节中独立小节呈现）
- **安全性评价**：药物研究和中医干预研究需报告不良反应/不良事件

### 结构红线

- 涉及人体研究但全文无伦理审批信息——这是医学论文的底线要求
- 方法章节无纳入排除标准——研究对象的选择缺乏依据，结果不可复现
- 临床干预研究没有对照组——无法判断干预效果是否优于自然转归
- 回顾性研究未说明数据来源的时间范围和医院/科室
- 护理干预研究未详细描述干预方案的具体内容（谁来实施、做什么、何时做、频率、持续时间）

## 专项规则权重调整

基于通用6大专项规则，调整医学护理类的审查权重：

| 专项规则 | 权重调整 | 理由 |
|---|---|---|
| intro | 0 | 保持通用权重；医学论文引言通常简洁，重点在临床问题界定 |
| literature | +1 | 需评估文献综述是否覆盖最新临床指南和高质量证据（系统综述、RCT） |
| method | +3 | 医学研究的方法规范性直接决定证据等级和结论可信度——这是最核心的审查对象 |
| analysis | +2 | 统计方法的正确选用、混杂因素的控制、效应量的合理报告在医学研究中至关重要 |
| conclusion | +1 | 结论的强度必须与研究设计的证据等级匹配，过度推断在医学领域可能导致不当临床决策 |
| fact | +1 | 医学论文涉及大量事实性描述（疾病流行病学、药理学机制等），准确性要求极高 |

## 方法审查补充维度

通用 method 规则之外，医学护理类需额外检查以下维度。**方法审查是医学护理类论文审查的重中之重，审查者应在此投入最多精力。**

### 1. 研究设计分级与匹配

医学研究的证据等级遵循如下层级（由高到低）：

| 等级 | 研究设计 | 适用场景 |
|---|---|---|
| I | 系统综述/Meta分析 | 综合已有证据（本科少见） |
| II | 随机对照试验（RCT） | 干预效果评价的金标准 |
| III | 队列研究 | 暴露与结局的因果关联 |
| IV | 病例对照研究 | 罕见疾病的危险因素探索 |
| V | 横断面研究 | 现状描述、关联分析 |
| VI | 病例报告/系列 | 罕见病例的经验分享 |

**审查要点**：
- 研究设计是否明确声明？不能含糊地写"采用统计学方法分析"而不说明设计类型
- 研究设计是否与研究问题匹配？如要评价某干预的疗效，横断面设计不够，至少需要前后对照或RCT
- 结论的因果性语言是否与设计等级匹配？横断面研究不能使用因果性结论

### 2. 伦理审查

**涉及人体研究时，以下为强制性审查项目：**

- **IRB/伦理委员会审批**：是否获得所在医院或院校伦理委员会的审批？审批编号是否注明？审批日期是否在研究启动之前？
- **知情同意**：是否获取了受试者（或其法定代理人）的书面知情同意？未成年受试者是否同时取得法定监护人同意和本人知情同意（适龄者）？
- **赫尔辛基宣言合规**：是否声明研究符合《赫尔辛基宣言》原则？
- **数据匿名化**：患者个人信息是否去标识化？病例报告是否去除可识别信息（姓名、住院号、影像中的个人信息等）？
- **动物实验伦理**：动物实验是否获得实验动物伦理委员会批准？是否遵循3R原则（Replacement, Reduction, Refinement）？动物品系、来源、许可证号是否注明？

### 3. 样本量与抽样

- **样本量计算**：是否进行了先验样本量计算（power analysis）？是否报告了检验效能所用参数（α水平、检验效能1-β、效应量或预期差值、检验类型）？使用的计算软件或公式是否说明？
- **纳入标准**：是否具体、可操作？如诊断标准应引用最新版指南或共识（如"符合《慢性阻塞性肺疾病诊治指南（2021年修订版）》诊断标准"），不能只写"确诊为XX病的患者"
- **排除标准**：是否合理且不引入选择偏倚？排除标准是否过多导致研究结果的外推性受限？
- **抽样方法**：随机抽样、便利抽样、整群抽样？抽样方法是否明确说明？便利抽样时是否讨论了选择偏倚？
- **招募时间段**：研究起止时间是否明确？（如"2023年1月至2024年6月于XX医院XX科收治的患者"）
- **失访与脱落**：是否报告了脱落人数、脱落原因？脱落率是否在可接受范围内（通常<20%）？是否进行了脱落分析（脱落者与完成者的基线比较）？

### 4. 统计方法审查

**审查原则：研究设计→数据类型→统计方法三者必须逻辑一致。**

| 分析方法 | 审查要点 |
|---|---|
| **t检验** | 数据是否满足正态性？方差齐性检验（Levene检验）是否报告？不齐时是否使用校正方法？ |
| **卡方检验** | 理论频数是否满足要求（≥5）？不满足时是否使用Fisher精确检验或校正卡方？ |
| **方差分析（ANOVA）** | 正态性和方差齐性是否检验？事后多重比较方法是否说明？ |
| **非参数检验** | 数据不满足参数检验条件时是否改用非参数方法（Mann-Whitney U、Kruskal-Wallis H等）？ |
| **Logistic回归** | 自变量的筛选方式是否说明（单因素→多因素）？多重共线性是否检查？模型拟合优度（Hosmer-Lemeshow检验）是否报告？ |
| **Cox回归/生存分析** | 比例风险假设（PH假设）是否检验？Kaplan-Meier曲线是否呈现？Log-rank检验是否报告？ |
| **诊断试验** | 灵敏度、特异度、ROC曲线下面积（AUC）是否报告？最佳截断值的确定方法（如约登指数）是否说明？ |

- **混杂因素控制**：是否识别了潜在的混杂因素？是否通过匹配、分层、多变量分析等方法加以控制？控制变量的选择依据是否说明？
- **意向性治疗分析（ITT） vs 方案分析（PP）**：临床干预研究是否区分了ITT分析和PP分析？RCT应以ITT分析为主要分析方式
- **效应量报告**：是否报告了效应量（OR、RR、HR、NNT等）及其95%置信区间？仅报告P值不够

### 5. 量表与工具审查

- **经过验证的量表**：所用量表是否为经过信效度检验的标准化工具？量表来源和编制者/修订者是否注明？
- **中文版量表**：国外量表是否使用了经过中国人群验证的中文版？是否注明中文版修订者和验证研究？
- **自编问卷**：如果使用自编问卷，是否经过专家审查？是否进行了预调查？是否报告了本研究样本中的Cronbach's α系数？α ≥ 0.70为可接受
- **计分方法**：量表的计分方式是否明确说明？总分/维度分的计算方式是否清晰？
- **量表版本**：是否使用了最新版本？（如PHQ-9、GAD-7、SAS/SDS、SF-36等常用量表的版本是否正确）

### 6. 护理研究特殊维度

- **干预方案的详细程度**：干预方案是否详细到其他护理人员可以据此复现？最低要求包括：
  - **谁**（Who）：干预实施者的资质和培训情况
  - **什么**（What）：干预的具体内容和步骤
  - **何时**（When）：干预的起始时间和时间节点
  - **多长时间**（How long）：每次干预的持续时间
  - **多频繁**（How often）：干预的频率
  - **对照组**：对照组接受了什么护理（常规护理的内容也必须描述）
- **结局指标的临床意义**：结局指标是否具有临床意义而非仅有统计学意义？是否选择了患者相关结局（Patient-Reported Outcomes, PROs）？
- **PRO工具的验证**：患者报告结局工具是否经过验证？是否适合目标人群（年龄、文化水平、疾病状态）？
- **盲法的可行性讨论**：护理干预研究通常难以实施双盲，是否讨论了盲法的局限及其对结果的潜在影响？
- **依从性报告**：干预组的依从性是否报告？依从性的评估方法是否说明？

### 7. 中医研究特殊维度

- **辨证标准**：中医辨证分型标准是否明确、可操作？是否引用了权威标准（如《中医诊断学》教材标准、中华中医药学会发布的辨证标准）？
- **中西医双重评价**：是否同时使用了中医证候疗效标准和西医客观指标进行疗效评价？仅用中医主观证候评分不够充分
- **方剂/药物组成**：处方组成是否完整列出？每味药的剂量是否标注？如有加减法，加减原则是否说明？
- **中药质量控制**：药材来源、炮制方法、制剂工艺是否说明？（本科论文至少需注明药材来源）
- **安全性评价**：是否报告了不良反应？肝肾功能等安全性指标是否监测？

## 证据与论证标准补充

通用 analysis/fact 规则之外，医学护理类的证据标准：

### 可接受的证据类型

- **临床试验数据**：在严格控制条件下收集的临床结局数据（症状改善率、治愈率、实验室指标变化等）。使用条件：必须有伦理审批、知情同意、明确的结局指标定义和测量方法
- **流行病学数据**：队列研究、病例对照研究、横断面调查获取的数据。使用条件：研究设计类型明确声明，混杂因素的控制方法说明
- **问卷调查数据**：经信效度检验的标准化量表得分（如生活质量、焦虑抑郁评分、护理满意度等）。使用条件：量表来源和本研究信度必须报告
- **实验室数据**：血液学、生化学、微生物学、影像学等客观检测指标。使用条件：检测方法、仪器型号、试剂来源应说明；检测的质控措施应描述
- **病历数据**：电子病历系统或纸质病历中提取的回顾性数据。使用条件：注明数据来源医院、科室、时间段、提取方式；数据质量控制措施说明
- **定性数据**：半结构化访谈、焦点小组等获取的质性数据。使用条件：必须有系统的主题分析过程，直接引用参与者原话

### 证据红线

- **涉及人体研究但无伦理审批**：医学研究中使用人体数据、组织、血液等必须有伦理审批，无审批的研究结论在学术上不被接受
- **临床声明无统计证据支撑**：如声称"某药物对XX病有效"但无统计检验或仅以"有效率较高"等模糊表述代替
- **诊断结论无金标准对照**：评价某诊断方法的准确性时，未以公认的金标准（如病理诊断、标准化检测）作为参照
- **混淆统计学显著性与临床意义**：P < 0.05即声称"具有临床意义"，未讨论效应量和临床最小重要差异（MCID）
- **选择性报告阳性结果**：只报告有统计学差异的指标，隐匿无差异的结局指标
- **伪造或篡改数据迹象**：如数据分布过于完美、样本量极小但P值极显著、描述统计与推断统计在数学上不一致

### 论证模式期望

- **临床研究型**：假设-验证模式。临床问题→文献依据→提出假设→研究设计→数据收集→统计检验→结论。论证链须闭合
- **回顾性分析型**：探索-验证模式。临床观察→回顾数据→单因素筛选→多因素验证→提出风险/保护因素。需承认回顾性设计的因果推断局限
- **调查研究型**：描述-分析模式。明确调查目的→标准化工具→数据收集→现状描述→影响因素分析→对策建议。建议须从数据中逻辑推出
- **护理干预型**：问题-干预-评价模式。识别护理问题→设计干预方案→实施干预→效果评价→护理实践建议。干预方案须有理论或证据支撑
- **病例报告型**：个案-文献对话模式。病例呈现→诊治过程分析→文献复习→经验总结。需明确指出本例的教学价值或临床启示

## 常见问题模式

### S级（论文是否成立）

1. **涉及人体研究但无伦理审批**：表现为全文无伦理委员会审批编号、无知情同意描述，或仅写"患者知情同意"但无具体程序说明 → 违反医学研究的基本伦理要求，论文在学术上不被接受，涉嫌伦理违规 → 修改方向：补充伦理审批信息（审批机构、审批编号、审批日期），描述知情同意的获取过程，声明符合《赫尔辛基宣言》原则

2. **知情同意缺失或不规范**：表现为涉及患者数据或临床干预的研究未说明知情同意获取过程，或涉及特殊人群（未成年人、认知障碍患者）未说明法定代理人同意 → 研究的伦理合规性无法确认 → 修改方向：在方法章节明确描述知情同意的获取方式（书面/口头）、特殊人群的代理同意程序，附伦理审批批件编号

3. **临床干预研究无对照组**：表现为仅设一组患者接受干预，观察前后变化，将改善归因于干预效果 → 无法排除自然病程转归、安慰剂效应、霍桑效应等替代解释，因果推断无效 → 修改方向：增设对照组（常规护理对照、等待名单对照、或阳性药物对照），或明确承认前后对比设计的局限并大幅弱化因果性结论

4. **诊断研究无金标准对照**：表现为评价某检测方法或诊断指标的准确性时，未使用公认的金标准进行对照，仅以"临床诊断"作为参照 → 无法计算真正的灵敏度和特异度，诊断价值评价无效 → 修改方向：明确金标准的定义和来源（如病理诊断、标准化实验室检测），计算灵敏度、特异度、阳性/阴性预测值和ROC曲线下面积

5. **样本量根本不足以支撑研究设计**：表现为参数检验样本量 n < 30且无正态性论证、多因素Logistic回归中每个自变量对应的事件数（EPV）< 10、生存分析中事件数 < 10×自变量数 → 参数估计不稳定，统计检验效力严重不足，结论不可信 → 修改方向：补充样本量计算过程（说明α、β、效应量/预期差值），增加样本量或简化统计模型

### A级（严重影响质量）

1. **缺少样本量计算/论证**：表现为方法章节无样本量计算过程，未说明α水平、检验效能和效应量，样本量的确定缺乏依据 → 读者无法判断研究是否有足够的统计效力来检测临床有意义的差异 → 修改方向：使用PASS、G*Power等软件进行先验样本量计算，报告所用参数和计算结果

2. **纳入/排除标准模糊或缺失**：表现为仅写"选取XX病患者"而无具体诊断标准来源、无明确的排除条件，或排除标准过于宽泛无法操作 → 研究对象的同质性无法保证，结果的可复现性存疑 → 修改方向：纳入标准引用最新版临床指南或诊断共识中的明确标准，排除标准逐条列出并说明理由

3. **统计方法与数据类型不匹配**：表现为对非正态分布数据使用参数检验而不做说明、对等级资料使用连续变量的统计方法、对小样本数据使用大样本近似方法 → 统计结果的有效性受损 → 修改方向：根据数据类型（连续/分类/有序）、分布特征（正态/非正态）和样本量选择正确的统计方法

4. **未控制混杂因素**：表现为比较两组差异时未考虑基线不均衡的混杂因素，或多因素分析中未纳入已知的重要混杂变量（如年龄、性别、疾病严重程度等） → 组间差异可能被混杂因素解释而非干预效果 → 修改方向：识别潜在混杂因素，通过随机化、匹配、分层或多变量回归加以控制

5. **护理干预方案描述不充分**：表现为干预内容仅写"实施XX护理干预"或"给予健康教育"，无具体的实施步骤、频率、时长和实施者资质说明 → 研究不可复现，其他护理人员无法据此开展类似干预 → 修改方向：按TIDieR清单（Template for Intervention Description and Replication）描述干预方案的各个要素

6. **自编问卷未经信效度检验**：表现为自行设计调查问卷，未经专家审查、未做预调查、未报告信度效度指标即用于正式调查 → 测量工具的质量未知，调查结果的可靠性无法保证 → 修改方向：进行专家内容效度评审（CVI ≥ 0.80）、预调查（n ≥ 30）、报告Cronbach's α，或改用已有的成熟量表

7. **回顾性研究未讨论偏倚**：表现为回顾性分析未讨论选择偏倚、信息偏倚和混杂偏倚的可能影响 → 回顾性设计固有的偏倚未被识别和评估，结论的可靠性被高估 → 修改方向：在讨论中逐一分析选择偏倚（纳入排除过程中的系统偏差）、信息偏倚（数据记录的完整性和准确性）、混杂偏倚（未测量的混杂因素）

8. **效应量仅报告P值而无置信区间**：表现为所有统计结果仅以"P < 0.05"或"P > 0.05"呈现，无OR、RR、HR等效应量及95%CI → 读者无法判断效应的大小和精度，P值受样本量影响，仅P值无法评估临床意义 → 修改方向：所有效应量均伴随报告95%置信区间，如"OR = 2.35 (95%CI: 1.42-3.89, P = 0.001)"

### B级（影响说服力）

1. **P值报告为"P = 0.000"**：表现为统计软件输出的"0.000"直接照搬至论文 → 不规范，P值在理论上不可能为0 → 修改方向：改为"P < 0.001"

2. **表格不符合医学论文规范**：表现为表格使用过多网格线和底纹、缺少标题或标题不规范、单位标注不清、小数位数不统一 → 影响论文的专业性和可读性 → 修改方向：使用三线表格式，标题置于表格上方，数值保留统一的小数位数，单位在表头或表注中标明

3. **医学缩写未首次定义**：表现为直接使用BMI、HbA1c、CRP、COPD、CHF等缩写而不在首次出现时给出全称 → 医学缩写数量庞大，不定义影响可读性 → 修改方向：每个缩写在正文中首次出现时给出中文全称和英文缩写，如"体质量指数（Body Mass Index, BMI）"

4. **未遵循报告规范清单**：表现为临床试验未参照CONSORT声明、观察性研究未参照STROBE声明、系统综述未参照PRISMA声明、诊断研究未参照STARD声明 → 缺少标准化报告框架，关键信息可能遗漏 → 修改方向：根据研究类型选择对应的报告规范清单，逐条核对并补充缺失信息

5. **药物名称使用商品名而非通用名**：表现为使用"拜糖平""立普妥""甲钴胺片"等商品名而非"阿卡波糖""阿托伐他汀""甲钴胺" → 不符合学术论文的药物命名规范 → 修改方向：一律使用国际非专利名称（INN）或中国药品通用名称，商品名仅在必要时用括号标注

6. **结果章节掺入讨论性内容**：表现为在结果部分出现"这可能是因为……""提示我们……""说明了……"等解释性语言 → 结果与讨论混淆，影响论文的逻辑清晰性 → 修改方向：结果章节严格只呈现数据分析结果，所有解释和推断放到讨论章节

7. **统计结果报告不完整**：表现为只报告P值不报告检验统计量（如t值、χ²值、F值等），或报告回归结果不给出回归系数和置信区间 → 读者无法核实和评估结果 → 修改方向：完整报告统计结果，如"t = 2.34, P = 0.021""χ² = 8.56, P = 0.003""OR = 2.15 (95%CI: 1.28-3.61)"

8. **参考文献时效性不足**：表现为核心参考文献多为10年以上的旧文献，未引用近5年的最新研究进展和最新版临床指南 → 研究背景和讨论部分缺乏时效性 → 修改方向：补充近5年相关领域的重要文献，确保近5年文献占比不低于40%

## 文献综述特殊要求

### 文献来源优先级

1. **第一梯队（必须引用）**：与研究主题直接相关的高质量证据和权威来源
   - 国际顶级期刊：*New England Journal of Medicine (NEJM)*, *The Lancet*, *BMJ*, *JAMA*, *Annals of Internal Medicine*
   - 中文核心期刊：《中华医学杂志》《中华护理杂志》《中国公共卫生》《中华预防医学杂志》《中华流行病学杂志》《药学学报》
   - 系统综述与Meta分析：Cochrane系统综述、BMJ/Lancet/JAMA发表的系统综述
   - 临床实践指南：WHO指南、各学会/协会发布的临床指南、中国版诊疗指南/专家共识
   - 循证医学资源：UpToDate、BMJ Best Practice

2. **第二梯队（推荐引用）**：SCI/CSCD来源期刊论文、专科权威期刊（如*Nursing Research*, *Journal of Clinical Nursing*, *International Journal of Nursing Studies*, *Cancer*, *Diabetes Care*等）、高质量队列研究和RCT

3. **第三梯队（可以引用）**：其他学术期刊论文、博士论文、学术专著、卫生统计年鉴

4. **不可接受**：百度百科、知乎回答、微信公众号、未经同行评审的网络内容、过期的教科书作为唯一证据来源

### 文献数量参考标准

- 本科论文参考文献一般不少于30篇，其中外文文献不少于8篇
- 近5年文献应占总量的40%以上，核心问题的引用应包含近3年的最新进展
- 至少引用1-2篇最新版临床指南或专家共识
- 如有系统综述/Meta分析可直接引用，优先于单个研究

### 综述组织方式

- **推荐**：按研究主题的逻辑线索组织。典型结构为：疾病/问题概述→流行病学现状→已有干预/治疗研究进展→现有研究的不足→本研究的切入点
- **可接受**：按研究方法分类组织（如将RCT、队列研究、调查研究分别综述）
- **不推荐**：按"国外研究""国内研究"简单二分而不整合发现；按发表年份逐篇罗列而无归纳

## 术语与规范

### 术语使用规范

- **疾病名称**：使用ICD-10/ICD-11标准编码和名称，首次出现时注明英文名称和ICD编码（如"2型糖尿病（Type 2 Diabetes Mellitus, T2DM, ICD-10: E11）"）
- **药物名称**：一律使用国际非专利名称（INN）或中国药品通用名称，不使用商品名。商品名仅在必要时用括号标注，如"阿托伐他汀（商品名：立普妥）"
- **解剖学术语**：遵循人体解剖学标准名称，中文使用《人体解剖学名词》（全国科学技术名词审定委员会）
- **护理诊断**：使用NANDA-I（北美护理诊断协会国际）标准术语，如"知识缺乏""自我照顾能力缺陷""焦虑"等
- **统计术语规范**：P值大写斜体（*P*），95%CI格式统一（如"95%CI: 1.25-3.42"或"95%CI [1.25, 3.42]"，全文须统一一种格式），α和β水平使用希腊字母
- **中医术语**：使用中医药学名词审定委员会公布的标准名称；方剂名称使用通行名称；穴位名称附国际标准代码（如"足三里（ST36）"）
- **核心概念首次出现时须给出定义**，且全文术语使用保持一致，不能同一概念交替使用不同名称

### 引用格式

- **推荐温哥华格式/顺序编码制**（Vancouver Style）——这是医学论文的通行标准
- 正文中引用以方括号内的阿拉伯数字顺序编号标注，如"研究表明[1]""多项研究[2-5]证实"
- 参考文献列表按引用顺序排列
- 部分院校也接受作者-年份制（APA/Harvard格式），需遵从学校要求
- 期刊名称使用标准缩写（如*N Engl J Med*, *Lancet*）

### 图表规范

- **表格**：使用三线表格式（无竖线、无多余横线），标题置于表上方
- **统计表**：保留统一的小数位数，P值保留三位小数，百分比保留一位小数，效应量保留两位小数
- **生存曲线**：Kaplan-Meier曲线应标注数轴标签、图例和Log-rank检验P值
- **ROC曲线**：标注AUC值及95%CI，标注最佳截断值
- **流程图**：临床研究建议附CONSORT流程图，展示筛选、随机化、脱落和分析的全过程
- **所有图表编号连续**，图标题在图下方，表标题在表上方

## 结论与建议特殊要求

### 结论的要求

- **直接回应研究目的**：结论应逐条回应引言中提出的研究问题/假设，明确说明主要发现
- **结论强度与证据等级匹配**：
  - RCT可使用因果性语言（"XX干预可有效改善……"）
  - 队列/病例对照研究使用关联性语言（"XX是XX的独立危险/保护因素"）
  - 横断面研究仅使用描述性和关联性语言（"XX与XX相关""XX人群的XX水平为……"），**绝不能**写"XX导致/引起了XX"
  - 护理干预研究如无严格的RCT设计，结论应使用审慎的语言（"XX护理干预可能有助于……"）
- **阴性结果同样报告**：不能在结论中只列出有统计学差异的发现而回避无差异的结果
- **不过度推断**：本科论文不宜声称"首次发现""填补空白"等措辞，除非有确凿证据

### 局限性讨论的要求

医学论文必须讨论以下层面的具体局限：
- **研究设计层面**：是横断面还是纵向研究？是回顾性还是前瞻性？是否为单中心研究？
- **样本层面**：样本来源是否有代表性？是否为便利抽样？样本量是否足够？
- **测量层面**：量表是否均为自我报告？是否存在回忆偏倚？实验室检测是否有方法学局限？
- **分析层面**：是否存在未控制的混杂因素？脱落率是否影响了结果？

### 临床实践建议的合理边界

- **建议必须从研究发现中逻辑推出**，不能泛泛地写"加强健康教育""提高护理质量"
  - 合格示例："本研究发现社会支持是老年COPD患者自我管理行为的独立保护因素（OR = 1.85, 95%CI: 1.23-2.78），建议护理人员在制定出院计划时评估患者的社会支持水平，对社会支持不足的患者建立家属参与的照护支持网络"
  - 不合格示例："建议加强对患者的健康教育"——与研究发现无具体关联
- **注明证据等级**：建议应说明是基于本研究的何种证据（RCT证据/观察性研究证据/调查结果），避免将低等级证据的发现包装为强烈推荐
- **不能从观察性研究直接推出干预建议**：横断面研究发现的关联不能直接转化为干预方案，需指出"有待干预性研究验证"

## 各子方向特殊审查要点

### 临床医学

- 诊断标准是否引用最新版指南或共识？诊断标准的版本是否过时？
- 临床分期/分型是否使用标准分类系统（如肿瘤的TNM分期、心功能的NYHA分级）？
- 手术相关研究是否描述了手术方式、术者资质和经验水平？
- 药物研究是否报告了药物剂量、给药途径、疗程和不良反应？

### 预防医学/公共卫生

- 流行病学调查的抽样设计是否清晰（简单随机、分层、整群、多阶段）？
- 调查问卷的信效度是否报告？大规模调查的质量控制措施（培训调查员、逻辑核查、重复调查等）是否描述？
- 暴露和结局的测量是否标准化？是否有测量偏倚的讨论？
- 多因素分析中混杂因素的选择是否基于文献和DAG（有向无环图）？

### 护理学

- 护理理论框架的使用是否恰当？常用理论包括：Orem自理理论、Roy适应模式、Pender健康促进模型、Watson关怀理论、知信行（KAP）理论、跨理论模型（TTM）
- 护理干预是否有理论支撑？不能仅凭经验设计干预方案
- 护理结局指标是否涵盖多维度（生理指标、心理指标、行为指标、患者满意度）？
- 质性护理研究是否遵循了COREQ（Consolidated Criteria for Reporting Qualitative Research）报告规范？

### 中医学

- 辨证分型是否有明确的纳入标准和依据？是否引用了权威的辨证标准文献？
- 是否做到了"病证结合"——既有西医疾病诊断，又有中医证型诊断？
- 方剂的组方原理是否在讨论中进行分析？君臣佐使的配伍分析是否到位？
- 中药药理学研究是否有现代药理学证据的佐证？
- 中药安全性是否关注？特别是含有潜在肝肾毒性成分的药物

### 药学

- 药物实验是否描述了完整的实验条件（温度、pH值、时间、浓度梯度等）？
- 分析方法是否经过方法学验证（专属性、线性、精密度、准确度、检测限、定量限）？
- 药代动力学参数是否完整报告（Cmax、Tmax、AUC、t1/2等）？
- 制剂研究是否进行了处方优化和稳定性考察？

### 康复治疗学

- 康复评定工具是否为标准化的功能评定量表（如Barthel指数、FMA、Berg平衡量表等）？
- 康复干预方案是否详细描述了治疗手法、治疗参数（频率、强度、时长）？
- 治疗前后评定是否由同一评定者（或盲法评定者）完成？
- 功能结局是否包含活动和参与层面的评价（ICF框架）？

### 医学检验技术

- 实验方法是否详细描述了试剂、仪器、操作步骤和质控方法？
- 方法学评价是否包含分析灵敏度、分析特异性、精密度和准确度？
- 参考区间的建立是否符合CLSI（临床和实验室标准协会）指南？
- 诊断效能评价是否使用了ROC分析并报告AUC？
