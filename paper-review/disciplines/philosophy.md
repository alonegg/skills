# 哲学伦理类 学科配置

> 本文件是 paper-review 技能的学科叠加层。它不替换通用规则，而是在通用规则基础上增加哲学伦理类学科特有的审查维度、调整权重、补充常见问题模式。

## 适用范围

- **覆盖学科**：哲学、伦理学、逻辑学、马克思主义哲学、中国哲学、外国哲学、科学技术哲学、宗教学、美学
- **典型论文类型**：
  - 文本解读型（对经典哲学文本的深入诠释与批判性分析）
  - 概念分析型（对哲学概念的界定、辨析、谱系追溯与逻辑澄清）
  - 比较哲学型（跨传统、跨哲学家、跨学派的哲学观点比较与对话）
  - 应用伦理型（将伦理学理论应用于具体现实问题的规范性分析）
  - 逻辑论证型（围绕特定哲学命题展开的形式化或半形式化论证）
- **识别线索**：出现以下关键词/特征时判定属于本学科——哲学概念辨析（"存在""本质""现象""先验""经验""辩证法"）、原典引用（柏拉图、亚里士多德、康德、黑格尔、胡塞尔、海德格尔、维特根斯坦、孔子、老子、庄子、王阳明、马克思）、论证结构分析（"前提""结论""有效性""可靠性"）、伦理学术语（"义务论""后果主义""德性伦理""道德直觉""反思平衡"）、哲学方法论术语（"现象学还原""解释学循环""概念分析""思想实验""辩证法"）

## 结构期望

### 标准章节结构

哲学本科论文存在五种主要结构范式，审查时应先识别论文属于哪种范式，再以对应结构为标杆：

**范式一：文本解读型**

```
第1章 引言（研究问题、文本选择理由、研究方法、文献回顾）
第2章 文本背景与哲学语境（哲学家生平思想背景、文本在其体系中的位置、写作语境）
第3章 文本核心论证重构（逐步重构文本的核心论证结构、关键概念解析）
第4章 批判性分析（论证评价、内在一致性检验、与其他文本/思想家的对话）
第5章 当代意义与回应（文本论证对当代哲学讨论的贡献、局限与可能的发展方向）
第6章 结论
```

**范式二：概念分析型**

```
第1章 引言（为什么需要澄清该概念、研究问题、方法说明）
第2章 概念的历史谱系（概念的起源与演变、不同哲学传统中的使用差异）
第3章 现有定义与分歧（梳理主要定义方案、各方案的优缺点）
第4章 概念辨析与重构（与相近概念的区分、提出本文的定义方案或修正方案）
第5章 应用检验（将定义方案应用于具体案例或思想实验，检验其解释力与适用边界）
第6章 结论
```

**范式三：比较哲学型**

```
第1章 引言（比较动机、比较框架说明、可比性论证）
第2章 思想家A/传统A的立场（核心主张、论证、概念框架）
第3章 思想家B/传统B的立场（核心主张、论证、概念框架）
第4章 比较分析（共同问题域识别、异同分析、差异的深层原因探讨）
第5章 哲学对话与启示（比较的理论产出：互补、互鉴或揭示新问题）
第6章 结论
```

**范式四：应用伦理型**

```
第1章 引言（伦理问题的提出、现实背景、研究意义）
第2章 伦理理论框架（选择并阐述所依据的伦理学理论、论证框架选择的理由）
第3章 问题分析（对伦理问题的多维分析：利益相关方识别、价值冲突结构、经验事实基础）
第4章 伦理论证（运用所选理论框架进行规范性论证、回应可能的反驳）
第5章 实践建议与伦理反思（具体的行动建议、理论框架本身的局限反思）
第6章 结论
```

**范式五：逻辑论证型**

```
第1章 引言（命题的提出、论证策略概述）
第2章 预备工作（核心概念定义、必要前提的确立、相关背景理论）
第3章 正面论证（核心论证的展开、前提到结论的推导）
第4章 反驳与回应（对可能反驳的系统回应、替代方案的评估）
第5章 推论与延伸（论证结论的推论、对相关问题的启示）
第6章 结论
```

### 必须有的章节

- **核心论证/核心主张章节**：哲学论文必须有明确的核心论点（thesis），并为之提供系统性论证。仅仅描述、综述他人观点而不提出并论证自己的主张，不构成哲学论文
- **概念界定章节**：核心哲学概念必须有专门的界定与辨析，不能含糊使用日常语义。概念界定的深度直接决定后续论证的精度
- **批判性分析/反驳回应章节**：必须正面处理可能的反对意见（objections）。哲学论文的质量很大程度上体现在对反驳的预见和回应能力上
- **结论**：必须凝练核心论点，回应引言中的问题，指出论证的局限与开放问题

### 可选/特殊章节

- **思想史背景章节**：研究涉及特定历史时期的哲学思想时需要，但篇幅须控制，不能喧宾夺主
- **方法论反思章节**：采用非标准哲学方法（如现象学方法、实验哲学方法）时，需专门论证方法选择的合理性
- **案例/思想实验章节**：应用伦理型和概念分析型论文中，用具体案例或思想实验检验论证时可独立设章

### 结构红线

- 无核心论点的"综述论文"：全文仅罗列各家观点，最后以"可见该问题十分复杂"收尾——这不是哲学论文，而是读书笔记
- 核心论证不可重构：读者无法从论文中提取出清晰的"前提→结论"结构——论证混乱或实际上不存在论证
- 批判性分析完全缺失：对所讨论的哲学家或理论无任何批判性审视，全文为"介绍+赞同"——缺乏哲学研究的基本要求
- "感想式写作"替代论证：大量使用"笔者深深感到""这一思想给我们很大启示"等感性表达，缺乏逻辑论证——这是读后感而非学术论文
- 概念核心论文却无概念界定：围绕某一概念展开讨论，但从未正式定义该概念的内涵与外延

## 专项规则权重调整

基于通用6大专项规则，调整本学科的审查权重：

| 专项规则 | 权重调整 | 理由 |
|---|---|---|
| intro | +1 | 哲学论文的问题提出须具有哲学维度，必须指向一个可论证的哲学命题而非社会现象描述；需审查问题是否具有哲学意义 |
| literature | +2 | 哲学论文的"文献综述"同时包含原典研读和二手研究梳理，须审查是否直接面对原典、学说梳理是否准确呈现争论格局 |
| method | +1 | 哲学方法多为隐性的（不像实证研究有显式方法论章节），需审查是否自觉运用了恰当的哲学方法（文本诠释、概念分析、逻辑论证等） |
| analysis | +3 | **最关键**。哲学论文的核心价值在于论证的质量——论证是否有效、前提是否可靠、反驳是否被充分回应、概念分析是否精确 |
| conclusion | +1 | 哲学论文的结论须明确回应核心问题，同时坦诚指出论证的局限和开放问题；不能空泛地讨论"重大意义" |
| fact | 0 | 哲学论文中的"事实"主要是文本事实（哲学家确实说了什么）和逻辑事实（论证确实具有某种结构），与经验科学的事实概念不同，通用规则基本适用 |

## 方法审查补充维度

通用 method 规则之外，本学科需额外检查：

### 一、文本解读方法审查

1. **原典参与深度**：论文是否直接引用并分析原典文本？是否仅依赖教材或二手文献对原典的转述？哲学研究必须直接面对原始文本——仅引用"某学者认为康德主张……"而从未引用康德自己的文字，标记 A 级问题。表现→全文关于某哲学家思想的论述均来自教材或综述文章，无一处直接引用原典。影响→无法确认作者真正阅读过原典，论文的学术基础不可靠。修改方向→回到原典，找到支撑论点的关键段落，直接引用并进行逐步分析

2. **诠释学循环意识**：对文本的局部解读是否放入整体语境中理解？是否注意到部分与整体的互相制约关系？孤立抽取一句话作为哲学家"核心观点"而不顾上下文的，标记 B 级问题。表现→从长篇著作中抽取一两句话作为论据，但该引文在原文语境中的含义与论文中的使用明显不同。影响→曲解原典，论证基础不可靠。修改方向→扩大引文的上下文范围，说明该段落在原著中的位置和功能，确保解读与整体一致

3. **历史语境关注**：解读哲学文本时是否考虑了文本产生的历史、思想背景？是否存在明显的时代错置（anachronism），如用当代概念框架强行解读古代文本？表现→用20世纪分析哲学的概念框架"翻译"柏拉图对话录，完全不顾古希腊哲学的概念语境。影响→解读失真，遮蔽了原典的真实思想内涵。修改方向→交代文本的历史思想背景，使用与文本时代相适应的概念框架，如需借用当代概念须明确说明是"重构"而非"还原"

4. **外国哲学翻译质量审查**：研究外国哲学时，关键术语是否对照了原文？是否注意到中文翻译可能存在的歧义？涉及翻译争议的术语是否比较了不同译本？表现→全文使用单一中译本，关键术语未附原文，遇到翻译歧义处（如海德格尔的Dasein译为"此在""亲在""定在"）未加辨析。影响→翻译选择本身蕴含解读立场，不加反思地使用某一翻译可能无意识地预设了特定解读。修改方向→关键术语首次出现时附原文，涉及翻译争议时简要说明不同译法及本文选择的理由

### 二、概念分析严谨性审查

5. **概念定义的明确性**：核心概念是否有明确、一致的定义？是否仅凭日常语感使用哲学术语而未加界定？表现→全文围绕"自由""正义""美"等概念展开讨论，但从未正式界定这些概念在本文中的具体含义。影响→概念模糊导致论证不可控——当概念含义随语境漂移时，论证的有效性无法评估。修改方向→在概念首次用于论证时给出工作定义（working definition），说明本文采用哪种定义、为何如此选择

6. **概念边界的划定**：是否明确了概念"是什么"与"不是什么"的边界？是否与相近概念做了区分？表现→讨论"道德责任"但未区分道德责任与法律责任、因果责任、角色责任的不同。影响→概念外延不清，论证中可能发生概念偷换。修改方向→列出与核心概念相近但不同的概念，逐一说明区别标准

7. **概念谱系的追溯**：是否追溯了概念在哲学史中的演变？是否注意到同一术语在不同哲学家/传统中含义的重大差异？表现→笼统地说"亚里士多德和康德都讨论了德性概念"而不说明两者的德性概念在内涵上根本不同。影响→遮蔽概念的历史复杂性，导致虚假的"共识"。修改方向→追溯概念在相关哲学家那里的具体含义，明确标注含义差异

8. **概念使用一致性**：同一概念在全文中的使用是否保持一致？是否出现了在不同段落中以不同含义使用同一术语的情况？表现→文章前半部分将"理性"理解为工具理性（计算手段与目的的能力），后半部分滑动为实质理性（对目的本身的价值判断能力），但未标注这一转换。影响→论证中出现隐性的四项错误（quaternio terminorum），形式上有效的论证实质上无效。修改方向→全文检查核心术语的使用，确保含义一致；如需在不同意义上使用同一术语，须明确标注并说明理由

### 三、论证结构审查

9. **论证可重构性**：论文的核心论证是否能被重构为清晰的"前提→结论"形式？读者能否识别哪些是前提、哪些是结论、推理步骤是什么？表现→论文有大量"论述"但缺乏明确的论证结构，读者无法判断作者到底在主张什么、依据什么。影响→论文不满足哲学论文的基本要求——提出并论证一个哲学主张。修改方向→明确陈述核心论点（thesis statement），列出支撑论点的关键前提，展示从前提到结论的推理步骤

10. **逻辑有效性**：论证是否包含形式逻辑谬误？常见谬误包括：循环论证（begging the question）、肯定后件（affirming the consequent）、否定前件（denying the antecedent）、合成谬误、分割谬误、偷换概念（equivocation）。表现→论证中出现"如果A则B；B为真；所以A为真"式的肯定后件谬误，或前提中隐含了结论的循环论证。影响→论证形式上无效，结论不能从前提中逻辑地得出。修改方向→重新检查论证的逻辑形式，消除形式谬误；如论证结构无法修复，须重构论证

11. **前提可靠性**：论证的前提是否得到了充分的支持？是否存在未经论证的隐含前提（hidden premise）？关键前提是否有争议而未被讨论？表现→论证依赖某个未经论证的前提，该前提本身就是哲学上有争议的主张。影响→论证的"可靠性"（soundness）受损——即使形式有效，前提为假则结论不必然为真。修改方向→识别所有关键前提，为有争议的前提提供独立论证或明确标注为预设

12. **反驳的充分回应**：是否识别并正面回应了对核心论点最强有力的反驳？回应是否针对反驳的实质而非稻草人（strawman）？表现→对反对观点的呈现明显弱化或歪曲（例如只回应最弱的反驳版本而忽视最强版本），或完全不提及已知的重要反驳。影响→论证的说服力大打折扣，学术诚信受到质疑。修改方向→按"最善意解读原则"（principle of charity）呈现最强版本的反驳，然后给出实质性回应

13. **辩证推进**：论证是否呈现了辩证的推进过程？即：正题→反题→合题，或：初始立场→反驳→修正后的立场。表现→论证是线性的——提出主张、堆砌支持理由，但无任何自我质疑和立场调整。影响→论证显得教条，缺乏哲学思考应有的反思深度。修改方向→在论证中主动引入挑战自己立场的考虑，展示这些挑战如何促使立场得到修正或更精确的表述

### 四、比较方法审查

14. **比较框架的明确性**：比较是否有明确的比较框架（围绕什么问题比较、按什么维度比较），还是仅仅是两个思想家观点的并置（juxtaposition）？表现→前半篇写A哲学家的思想，后半篇写B哲学家的思想，最后一段说"两者有异有同"。影响→不构成真正的比较研究，只是两篇独立介绍的拼合。修改方向→在引言中明确比较框架——围绕什么共同的哲学问题进行比较、设定哪些具体的比较维度；比较分析应贯穿全文而非仅在结尾

15. **可比性论证**：为什么这两个思想家/传统可以放在一起比较？是否讨论了可比性的前提条件？特别是跨文化比较（如中西哲学比较），是否意识到概念框架的不可通约性问题？表现→将儒家"仁"直接等同于亚里士多德的"德性"进行比较，未讨论两个概念在各自传统中的不同功能和预设。影响→强行类比导致两个传统都被扭曲。修改方向→正面讨论比较的前提条件和限制，承认概念差异，在差异中寻找可对话的空间

16. **避免强行趋同**：比较分析是否尊重了被比较对象的差异性？是否存在为了得出"殊途同归"的结论而忽视实质差异的倾向？表现→比较庄子与海德格尔，结论为"两者思想本质上一致"，但对两者在存在论预设、方法论、文化语境上的根本差异轻描淡写。影响→比较研究丧失学术价值——如果结论只是"其实都一样"，比较的意义何在？修改方向→同等重视异同，差异分析往往比相似性分析更有哲学价值；探讨差异的深层原因

### 五、应用伦理分析审查

17. **伦理框架的明确选择与论证**：应用伦理分析是否明确选择了伦理学理论框架（义务论、后果主义、德性伦理、关怀伦理等），并论证了为何选择该框架？表现→论文讨论某一伦理问题时，时而引用康德的义务概念，时而计算后果的利弊，时而诉诸德性，但从未明确自己的理论立场。影响→论证缺乏一致性，不同理论框架的结论可能相互矛盾，读者无法评估论证的融贯性。修改方向→明确选择一个或两个理论框架作为分析工具，论证选择理由；如采用多元框架须说明整合方式

18. **利益相关方分析的完整性**：是否识别了所有相关的利益相关方？是否存在遗漏关键群体的偏差？表现→讨论人工智能伦理时只考虑开发者和用户，忽略受算法影响的非用户群体、未来世代等。影响→伦理分析不完整，遗漏的利益相关方的正当权益未被考虑。修改方向→系统列出所有利益相关方，分析各方的利益诉求和可能的价值冲突

19. **规范性与描述性的区分**：论文是否清晰区分了"事实上人们怎么做/怎么想"（描述性伦理）和"应该怎么做"（规范性伦理）？表现→以"大多数人认为X是可接受的"作为"X是道德上正确的"的论据——从"是"推出"应该"。影响→犯自然主义谬误或诉诸民意谬误。修改方向→明确区分描述性前提和规范性前提，从描述到规范的过渡需要额外的桥接论证

20. **现实复杂性的回应**：应用伦理分析是否回应了现实情境的复杂性，还是仅在理想化条件下讨论？表现→讨论安乐死伦理时完全不考虑知情同意的实际困难、医患权力不对等、制度滥用风险等现实因素。影响→伦理分析脱离实际，建议缺乏可行性。修改方向→在理论分析基础上增加对现实约束条件的讨论，使用真实案例或贴近现实的思想实验

### 六、原典引用规范审查

21. **引用精度**：哲学文本引用是否标注到具体的页码、节号、段落号？不能仅写"康德认为……"而不注明出处位置。表现→论文中出现大量"亚里士多德认为""黑格尔指出"等表述，但均无具体出处标注或仅注书名。影响→引用无法追溯核实，也表明作者可能并未直接阅读原典。修改方向→所有原典引用必须标注到可追溯的定位信息

22. **标准版本引用**：是否引用了公认的标准版本？西方哲学经典有固定的学术引用规范——柏拉图用Stephanus页码，亚里士多德用Bekker编码，康德用Akademie版页码，维特根斯坦《逻辑哲学论》用命题编号，海德格尔《存在与时间》用德文原版页码（Max Niemeyer版）。中国哲学经典应引用中华书局点校本或其他学术公认的标准版本。表现→引用柏拉图仅注中译本页码而非Stephanus页码，引用《论语》使用的是某普及读物版本而非学术标准版本。影响→不符合哲学学术规范，影响学术可信度。修改方向→查找并标注标准版本信息，中译本页码可同时保留但须附标准编码

## 证据与论证标准补充

通用 analysis/fact 规则之外，本学科的证据标准：

### 可接受的证据类型

- **原典文本**：哲学论文中最核心的"证据"形式——直接引用哲学家的原始文本，用以支撑对其思想的解读。引用时须标注标准版本信息，并提供对引文的分析（而非仅仅引用）
- **逻辑论证**：形式论证（演绎推理、归谬法、构造性两难等）本身构成哲学中最有力的"证据"。论证必须具有可重构性
- **思想实验**：哲学中广泛使用的论证工具（如电车问题、中文房间、缸中之脑），用于检验直觉、测试理论。须说明思想实验的设计意图及其限制
- **概念分析**：通过分析概念的逻辑结构、必要和充分条件、反例等来论证哲学主张
- **历史/文本事实**：关于哲学文本的成书背景、版本流传、哲学家的思想发展等事实性材料，用于支撑文本解读
- **经验事实**（限应用伦理和科学技术哲学）：来自自然科学、社会科学的经验研究结果，作为应用伦理分析的事实前提。须注明数据来源和研究质量

### 证据红线

- **以权威代替论证**：仅因"某大哲学家这么说"就将其观点视为真理而不加论证——哲学不承认权威论证（argument from authority）作为有效论证形式。标记 A 级
- **以感受代替论证**：以"笔者深深感受到""这一思想令人深思"等感性表达代替理性论证。标记 A 级
- **原典引用全部来自二手文献**：论文声称研究某哲学家思想，但所有"原典引用"实际上来自教材或其他学者的转引，作者并未直接阅读原典。标记 S 级
- **捏造或严重歪曲哲学家观点**：将哲学家从未表达过的观点归于其名下，或对原典进行明显违背文本本意的解读。标记 S 级
- **以百度百科、知乎、微信公众号等非学术来源作为哲学论据**。标记 S 级
- **将科普读物或通俗介绍等同于学术文献**：引用"XX分钟读懂康德"类通俗读物作为学术论据。标记 A 级

### 论证模式期望

- **文本解读型**：诠释学方法——"文本定位→语境还原→关键段落精读→论证重构→批判性评价→当代对话"。每一步解读必须以文本为锚点，不能脱离文本进行自由联想
- **概念分析型**：分析哲学方法——"概念界定→必要/充分条件分析→反例检验→概念修正→应用验证"。概念分析的目标是达到逻辑上的清晰和精确
- **比较哲学型**：比较方法——"共同问题设定→各自立场重构→维度比较→差异归因→对话可能性探讨"。比较必须在尊重各传统自身逻辑的前提下进行
- **应用伦理型**：规范性论证——"问题界定→伦理框架选择→利益相关方分析→规范性论证→反驳回应→实践建议"。伦理分析必须在明确的理论框架内进行
- **逻辑论证型**：形式/半形式论证——"命题陈述→前提确立→推理展开→反驳预应→结论确立"。论证的每一步须对读者透明

## 常见问题模式

### S级（论文是否成立）

1. **无原创论点——论文实质为读书笔记**：表现→全文仅综述各家观点或概述某哲学家思想，通篇没有明确的核心论点（thesis），以"综上所述，该问题十分复杂"或"各家观点各有优劣"收尾。作者始终处于"介绍者"而非"论证者"的角色。影响→不具备哲学论文的基本属性——哲学论文必须提出并论证一个自己的哲学主张，无论该主张是对既有观点的辩护、修正还是反驳。修改方向→必须在引言中明确陈述核心论点（thesis statement），全文围绕论证该论点展开，综述他人观点的目的是为自己的论证服务

2. **核心论证包含形式逻辑谬误**：表现→论文的核心论证存在不可修补的形式逻辑错误——循环论证（结论隐含在前提中）、肯定后件、否定前件、偷换概念（同一术语在前提和结论中含义不同）等。影响→论证形式无效，结论不能从前提中逻辑地得出，论文的核心主张缺乏论证支撑。修改方向→重新审查论证的逻辑结构，消除形式谬误；如果论证结构根本无法修复，须重构整个论证路径

3. **原典未实际阅读——所有引用均来自二手来源**：表现→论文声称研究柏拉图/康德/庄子等哲学家的思想，但所有"原典引用"均来自教材、综述文章或其他学者著作的转引；论文中看不到任何直接面对原典文本的分析；对原典的理解停留在教科书概括的水平。影响→研究的学术基础不存在——未读过原典而讨论哲学家思想，如同未看过判决书而分析判例。修改方向→必须回到原典，直接阅读关键文本段落（必要时对照原文），在论文中展示对原典的一手分析

4. **基础性概念混淆——误解核心哲学家的基本立场**：表现→对所研究的哲学家的核心主张存在根本性误解。例如：将康德的"自律"（Autonomie）理解为日常意义上的"自我管理"而非立法性的自我立法；将海德格尔的"此在"理解为"这个存在物"而非人的存在方式；将马克思的"异化"理解为心理学意义上的疏离感而非社会经济结构概念。影响→整篇论文建立在对核心概念的误解之上，后续所有分析和论证均不成立。修改方向→回到原典和权威二手文献，纠正对核心概念/立场的理解，在此基础上重建论证

### A级（严重影响质量）

1. **论点有而论证无——主张未被系统性辩护**：表现→论文明确提出了一个核心论点，但后续论证仅为反复重申该论点或堆砌与论点方向一致的材料，未提供结构化的论证（即缺乏明确的前提和从前提到结论的推理步骤）。影响→论点沦为未经论证的断言（mere assertion），缺乏学术说服力。修改方向→为核心论点构建完整的论证结构：明确列出支撑前提，展示推理过程，解释每个前提的合理性

2. **反驳未回应或回应为稻草人**：表现→论文对核心论点可能面对的反驳完全不提及，或仅呈现并回应了一个弱化版的反驳（strawman），而忽视了文献中已知的最强反驳。影响→论证的可信度大打折扣——哲学论文的质量在很大程度上体现在对反驳的回应能力上。修改方向→识别核心论点面对的最强反驳（按最善意解读原则重构），正面回应；如果反驳确实有力，须修正自己的论点而非回避

3. **时代错置——以今律古或以古律今**：表现→用当代伦理观念评价古代哲学家的道德立场（如批评亚里士多德"不尊重女性权利"），或不加反思地将古代哲学概念直接应用于当代问题而忽略语境差异。影响→扭曲了被研究对象的思想面貌，论证缺乏历史意识。修改方向→将哲学家的观点放入其历史语境中理解；如需将古代思想应用于当代，须明确说明是"创造性转化"并讨论其限度

4. **比较论文沦为并置——缺乏真正的哲学对话**：表现→前半篇介绍哲学家A的思想，后半篇介绍哲学家B的思想，最后一段以"两者有异有同"收尾，但全文无真正的比较分析——未围绕共同问题设定比较维度，未分析异同的深层原因，未探讨对话的可能性。影响→不构成比较哲学研究，仅为两篇独立介绍的机械拼合。修改方向→围绕一个共同的哲学问题设定比较框架，在每个比较维度上展开真正的对话式分析

5. **应用伦理论文缺乏明确伦理框架**：表现→讨论伦理问题时不在任何明确的伦理学理论框架内展开论证，而是诉诸模糊的"道德感""社会公德""人性"等概念；不同段落隐含地采用不同理论框架但未自觉。影响→伦理论证无标准可循，论文的规范性结论缺乏理论基础。修改方向→在论文前部明确选择并阐述所依据的伦理理论（义务论/后果主义/德性伦理/关怀伦理等），说明选择理由，后续分析在该框架内一致地展开

6. **核心术语使用不一致或含糊**：表现→同一关键术语（如"理性""自由""主体性""存在"）在论文不同部分以不同含义使用，但未标注含义的转换；或关键术语从未被定义，全文凭日常语感使用。影响→论证中出现隐性的偷换概念谬误；概念含糊导致论证精度不可评估。修改方向→核心术语首次出现时给出明确定义；如需在不同意义上使用同一术语，须加限定词区分（如"工具理性"与"实质理性"）

### B级（影响说服力）

1. **过度依赖二手文献而原典参与不足**：表现→论文引用了大量关于某哲学家的研究论文（二手文献），但对原典的直接引用和分析相对薄弱；论点主要建立在其他学者的解读之上。影响→论文变成"关于研究的研究"而非"对哲学问题的直接思考"，独立思考的深度不足。修改方向→增加对原典关键段落的直接引用和分析，使二手文献发挥辅助而非主导作用

2. **引用作装饰而非分析素材**：表现→大量引用哲学家的经典语句，但引用后未进行分析——引文仅起到"装饰"或"镇场"的作用。典型模式："正如康德所言，'……'（引用一大段）。由此可见……"，"由此可见"后面的内容与引文无实质分析关系。影响→原典引用未真正转化为论证资源。修改方向→每一处引用后必须跟随对引文的分析——这段话是什么意思？它如何支撑本文的论点？其中哪些概念需要进一步辨析？

3. **缺少标准版本引用信息**：表现→引用哲学经典仅注中译本页码，未附标准学术编码（Stephanus页码、Bekker编码、Akademie版页码等）；引用中国哲学经典未注明版本（如引《论语》未标明用哪个版本的注疏）。影响→引用不符合哲学学术规范，降低学术可信度。修改方向→补充标准版本信息，至少在首次引用时完整标注

4. **论证的逻辑结构未被明确呈现**：表现→论文有论证意图，但论证散布在叙述性段落中，读者需自行从大量文字中提取论证结构；前提和结论的关系不明确。影响→论证的有效性难以评估，读者可能误解论证意图。修改方向→在关键论证处使用明确的论证标识词（"因此""由此推出""这一前提的依据是"等），或以列点方式呈现论证结构

5. **过度修辞替代严谨论证**：表现→大量使用排比、感叹、隐喻等修辞手法表达哲学观点，如"思想的光辉照耀着人类精神的殿堂""在历史的长河中闪烁着永恒的智慧"。影响→修辞无法替代论证——华丽的语言不能弥补逻辑的缺失。修改方向→将修辞性段落改写为论证性段落：明确主张是什么，论据是什么，推理步骤是什么

6. **思想实验使用不当**：表现→引入思想实验但未说明其设计意图和论证功能，或思想实验的设计不够精确（包含与论证无关的干扰因素），或对思想实验的直觉反应未经反思就被视为结论。影响→思想实验的论证功能未能实现。修改方向→明确每个思想实验的论证目的，精简设计以突出关键变量，对直觉反应进行批判性检验

## 文献综述特殊要求

### 组织方式

- **优先采用问题导向式组织**：围绕核心哲学问题的不同立场组织综述，呈现各立场的核心论证、代表人物和相互之间的批评与回应关系。避免按时间线或按人物逐一介绍
- **原典与二手文献分层处理**：综述中必须区分两个层次——(a) 对原典/经典文本的梳理：哲学家自己怎么说、论证什么；(b) 对研究文献的梳理：学者们如何解读这些原典、有哪些争议。两个层次不可混淆
- **呈现学术争论的结构**：不能只列出"学者A认为……学者B认为……"的意见清单，须呈现争论的逻辑结构——各方的核心分歧点是什么、各自的论证优势和困难是什么、争论至今未解的关键问题是什么
- **必须包含批判性评述**：指出现有研究的空白、方法论局限、未被充分讨论的反驳，从中自然引出本文的研究空间。纯罗列式综述不合格

### 文献来源优先级

| 优先级 | 来源类型 | 说明 |
|---|---|---|
| 第一层 | 原典/经典文本（标准版本） | 所研究哲学家的原始著作——柏拉图对话录、亚里士多德著作集、康德三大批判、黑格尔著作、马克思恩格斯全集、《论语》《道德经》《庄子》等标准学术版本 |
| 第一层 | Stanford Encyclopedia of Philosophy (SEP) | 国际公认最权威的哲学百科全书，由专家撰写并同行评审，可作为可靠的二手参考 |
| 第二层 | 权威学术专著 | 学科领域公认的研究性专著（如 Korsgaard 论康德伦理学、陈来论宋明理学） |
| 第二层 | 核心期刊论文 | 中文：哲学研究、中国哲学史、伦理学研究、自然辩证法研究、哲学动态、世界哲学、道德与文明、逻辑学研究；外文：Philosophy and Phenomenological Research, Mind, The Journal of Philosophy, Ethics, Nous, Philosophical Review |
| 第三层 | 外文原版著作 | 研究外国哲学时，重要著作应参考原语言版本或权威英译本（如 Cambridge/Oxford 出版的译本） |
| 第三层 | 学术会议论文集 | 知名学术出版社出版的专题论文集（如 Cambridge Companion to... 系列） |
| 慎用 | 教材/通史类著作 | 可用于背景了解但不宜作为论证的核心依据（如罗素《西方哲学史》观点偏颇，冯友兰《中国哲学史》有特定立场） |
| 不可接受 | 百度百科、知乎、微信公众号、通俗哲学读物 | 非学术来源，绝对禁止作为学术论据 |

### 文献数量与时效

- 本科哲学论文参考文献总量建议 20-40 篇（篇数较其他学科可略少，但原典参与深度要求更高）
- 原典/经典文本引用：不设数量限制，但必须有直接引用和分析（不能仅列入参考文献而未在正文中使用）
- 近 5 年文献不低于总量的 30%（哲学研究时效性压力低于实证学科，但须了解最新学术进展）
- 外文文献：研究外国哲学方向不低于参考文献总量的 40%（应包含原典原语言版本或权威译本）；其他方向不低于 15%
- 核心期刊论文和学术专著不少于参考文献总量的 50%

## 术语与规范

### 术语使用

- **哲学术语首次出现时必须定义**：不能假设读者理解"先验""超越论""解构""扬弃""格物致知"等术语的含义。每个术语在首次用于论证时须给出工作定义
- **外国哲学术语须附原文**：关键术语首次出现时在中文翻译后括注原文。例如："此在（Dasein）""扬弃（Aufhebung）""存在者（Seiendes）与存在（Sein）""范畴命令（kategorischer Imperativ）""对价（consideration）"
- **翻译一致性**：对同一外文术语的翻译在全文中须保持一致。如果选择将 Dasein 译为"此在"，则全文不得交替使用"亲在""定在"等其他译法。首次出现时注明："本文将 Dasein 译为'此在'，下同"
- **区分日常用语与哲学术语**：许多哲学术语在日常语言中也有使用但含义不同（如"理性""自由""本质""存在""意识"）。论文中使用这些词语时须明确是在哲学意义上还是日常意义上使用
- **逻辑符号的正确使用**：如论文使用形式逻辑符号（→、∧、∨、¬、∀、∃等），须确保使用正确且前后一致；非逻辑学方向论文不强制要求使用逻辑符号，但如使用则必须精确
- **中国哲学术语的现代阐释**：使用"仁""义""礼""道""气""理"等中国哲学概念时，须说明在本文语境中的具体含义，不能仅凭日常中文理解

### 引用格式

- 哲学论文可采用脚注制或文末参考文献制，具体遵循各校指定规范
- 原典引用格式须包含标准学术编码：
  - 柏拉图：Stephanus 页码（如 Republic 514a-520a）
  - 亚里士多德：Bekker 编码（如 Nicomachean Ethics 1094a1-1094b10）
  - 康德：Akademie 版页码（如 KrV A51/B75）
  - 维特根斯坦《逻辑哲学论》：命题编号（如 TLP 5.6）
  - 中国经典：篇章名+标准版本（如《论语·学而》，中华书局2006年版）
  - 马克思恩格斯：《马克思恩格斯全集》卷数+页码
- 外文文献引用遵循 Chicago Manual of Style 或各校指定格式
- 译著引用须标注原著信息和译者

### 图表规范

- 哲学论文一般较少使用图表，但以下情况可使用：
  - 论证结构图：用以清晰呈现复杂论证的逻辑结构
  - 概念关系图：用以展示多个概念之间的逻辑关系
  - 比较分析表：比较哲学论文中可使用表格对比不同立场的核心主张
  - 逻辑推导表/真值表：逻辑学方向论文中使用
- 所有图表须有编号和标题，图表内容须在正文中加以分析说明

## 结论与建议特殊要求

### 结论部分

- **必须明确回答核心哲学问题**：结论第一段须直接回应引言中提出的哲学问题，给出本文论证所支撑的核心回答。如引言问"康德的道德哲学能否回应当代德性伦理学的挑战"，结论须正面回答并概括论证要点
- **展示论证的整体逻辑**：结论须简明扼要地重述从前提到结论的论证路径，使读者能快速把握论文的论证全貌
- **坦诚指出论证的局限与开放问题**：哲学论文的结论应体现谦逊的智识态度——指出论证所依赖的预设、未能充分回应的反驳、论证结论的适用边界。这不是削弱论文，而是增强其学术诚信
- **禁止在结论中引入新论据或新论证**：结论是对前文论证的凝练，不是开始新一轮论证
- **避免空泛的"意义"声明**：不能以"本文对促进人类思想发展具有重大意义""对构建和谐社会有重要启示"等套话结尾。结论须聚焦于论证本身的哲学贡献

### 研究局限与展望

- **局限性须与哲学方法论相关**：不能只说"时间和精力有限"。应指出具体的方法论局限：如本文仅从义务论角度分析未考虑后果主义视角；仅依据某一译本而非原文；比较仅涉及两个传统而未纳入第三种立场等
- **未来展望须具有哲学研究的方向性**：指出哪些哲学问题值得进一步探讨、可从哪些新文本或新角度切入、当代哲学的哪些新发展可能改变本文的讨论

### 特别说明

哲学论文的评价标准具有独特性：**核心质量标准是论文是否提出并有效论证了一个原创的哲学主张**。不同于经验研究论文以方法论严谨性和数据质量为核心，哲学论文以论证的逻辑质量、概念的清晰度、对反驳的回应能力为核心。审查时须始终以论证质量为第一标准。

## 各子学科特别审查要点

### 马克思主义哲学

- 是否真正基于马克思恩格斯的原典文本展开分析，而非仅引用教科书对马克思主义的概括？原典引用应指向《马克思恩格斯全集》或《马克思恩格斯选集》的具体卷页
- 是否注意到马克思思想发展的阶段性（如早期手稿与成熟时期著作的差异），避免将不同时期的观点不加区分地混用
- 是否在学术讨论的框架内展开分析，而非变成政治宣传或教条化的阐释
- 是否关注了当代马克思主义研究的学术前沿（如分析马克思主义、生态马克思主义、西方马克思主义各流派）

### 中国哲学

- 原典引用是否使用了标准学术版本（中华书局点校本、上海古籍出版社等）？是否标注了篇章出处？
- 对古文的理解是否准确？是否参考了主要的注疏传统（如《论语》的何晏注、朱熹注等）？
- 是否避免了以西方哲学概念框架机械地"翻译"中国哲学——如将"道"直接等同于黑格尔的"绝对精神"、将"仁"等同于"博爱"
- 是否注意了中国哲学概念的语境敏感性——同一术语（如"道""理""气"）在不同哲学家和不同时期含义可能有重大差异

### 外国哲学

- 关键术语是否对照了原文？重要引文是否核对了原文或权威译本？不同中译本之间存在分歧时是否有所辨析？
- 是否注意到翻译选择本身蕴含解读立场？如将 Sein 译为"存在"还是"是"会导致对海德格尔的不同理解
- 是否使用了该哲学家研究领域公认的标准版本和引用规范？
- 研究分析哲学传统时，是否达到了分析哲学要求的概念精确度和论证严格度？

### 伦理学

- 元伦理学、规范伦理学、应用伦理学三个层次是否区分清楚？论文在哪个层次上展开？
- 规范伦理学论文是否明确了理论立场（义务论、后果主义、德性伦理等）并在该框架内一致地展开论证？
- 应用伦理学论文是否充分回应了现实的复杂性，而非仅在理想化条件下讨论？
- 道德直觉作为论据使用时，是否进行了反思性检验（如 Rawls 的反思平衡方法）？

### 逻辑学

- 形式系统的定义是否严格？公理和推理规则是否完整列出？
- 证明过程是否完整、无跳步？每一步推导是否标注了所依据的规则？
- 元定理的证明（如完全性、可靠性、不可判定性）是否区分了对象语言和元语言？
- 非形式逻辑/论证理论方向：论证分析是否使用了标准的论证图式（如 Toulmin 模型）？谬误识别是否准确？

### 科学技术哲学

- 对所讨论的科学理论/技术是否有基本准确的理解？哲学分析是否建立在对科学内容的正确把握之上？
- 科学哲学论文是否运用了科学哲学的核心概念（如可证伪性、范式、研究纲领、实在论/反实在论等）并对其有准确理解？
- 技术哲学论文是否超越了对技术的日常抱怨或赞美，深入到技术的本体论、认识论或伦理学维度？
- 是否注意到科学哲学与科学史的密切关系，在必要时引入科学史案例支撑论证？

### 宗教学

- 是否在学术研究的框架内讨论宗教问题，而非进行宗教宣传或无依据的反宗教批判？
- 宗教文本的引用是否使用了学术标准版本？是否注意到不同译本/校本之间的差异？
- 宗教哲学论文是否与分析宗教哲学的当代讨论（如恶的问题、宗教经验的认识论地位、神圣命令理论等）进行了对话？
- 比较宗教研究是否尊重了各宗教传统的内在逻辑，避免以某一传统的标准评判其他传统？

### 美学

- 是否区分了美学理论（关于美和艺术的哲学反思）与艺术批评（对具体作品的评价）？本科美学论文应以前者为主
- 美学概念（如"美""崇高""审美经验""艺术""表现"）是否有明确的理论界定，而非凭日常感受使用？
- 涉及具体艺术作品分析时，是否在美学理论框架内进行，而非仅做印象式鉴赏？
- 中国美学研究是否注意到中国美学范畴（如"气韵""意境""味"）与西方美学概念的差异性？
