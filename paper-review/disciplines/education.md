# 教育学 学科配置

> 本文件是 paper-review 技能的学科叠加层。它不替换通用规则，而是在通用规则基础上增加教育学科特有的审查维度、调整权重、补充常见问题模式。

## 适用范围

- **覆盖学科**：教育学、课程与教学论、学前教育、小学教育、教育技术学、特殊教育、体育教育
- **典型论文类型**：教学实验型、调查研究型、课程/教学设计型、理论分析型
- **识别线索**：出现"教学""课程""学生""教师""学习""课堂""教育""核心素养""学科教学""教学策略""学情分析""教学目标""布卢姆""建构主义""最近发展区""教学评价""课程标准"等关键词；引用教育类期刊（《教育研究》《课程·教材·教法》等）；研究对象为学生、教师、学校

## 结构期望

### 标准章节结构

教育学本科论文按论文类型有四种典型结构：

**教学实验型**
```
第1章 引言（研究背景、研究目的、研究意义、核心概念界定）
第2章 文献综述（国内外研究现状、述评）
第3章 理论基础（所依据的教育理论/学习理论）
第4章 研究设计（研究假设、实验方案、被试选择、变量控制、测量工具）
第5章 实验实施（教学过程描述、数据采集过程）
第6章 数据分析（前后测对比、统计检验、结果呈现）
第7章 结论与反思（研究结论、教育启示、研究局限、展望）
```

**调查研究型**
```
第1章 引言（研究背景、研究目的与意义、概念界定）
第2章 文献综述（相关研究梳理与述评）
第3章 研究设计（研究问题、研究方法、样本选择、工具设计/选用）
第4章 数据收集与整理（问卷发放/访谈实施、数据清洗）
第5章 结果分析（描述统计、推断统计或质性编码分析）
第6章 对策建议（基于发现提出可操作的建议）
第7章 结论（研究总结、局限与展望）
```

**课程/教学设计型**
```
第1章 引言（研究背景、设计目的与意义）
第2章 文献综述（相关教学模式/课程理论研究）
第3章 理论基础与设计理念
第4章 教学/课程设计方案（目标、内容、活动、评价的完整设计）
第5章 设计实施与反馈（实际教学过程、学生反馈、课堂观察数据）
第6章 反思与改进（对设计的评价、迭代修改）
第7章 结论（设计原则总结、适用范围、局限）
```

**理论分析型**
```
第1章 引言（问题的提出、研究意义）
第2章 核心概念界定与辨析
第3章 历史演变/国际比较/政策梳理
第4章 现状描述与问题分析（基于文献或二手数据）
第5章 归因分析/深层机制探讨
第6章 对策建议
第7章 结论
```

### 必须有的章节

- **引言/绪论**：必须包含研究背景、研究目的、研究问题（明确的问句形式）、核心概念界定；教育学论文的"概念界定"极为重要，因为教育术语往往含义模糊（如"核心素养""深度学习""有效教学"），不界定则后续论述缺乏锚点
- **文献综述**：必须有，且必须有"述评"小节——不能只罗列文献，必须指出已有研究的不足，为本研究提供逻辑入口
- **研究设计/方法**：实证类（实验型、调查型）论文必须有独立的研究设计章节，说明方法选择的理由、样本、工具、数据分析策略
- **结论**：必须回应引言中提出的研究问题，逐条作答

### 可选/特殊章节

- **理论基础**：教学实验型和教学设计型通常需要；调查研究型可以并入文献综述；理论分析型视论题而定
- **伦理声明**：涉及未成年人研究时应有（详见"特殊审查维度"）
- **附录**：问卷原件、访谈提纲、编码表、教学设计详案应放入附录，正文仅呈现核心内容

### 结构红线

- 教学实验型论文没有独立的研究设计章节，直接跳到"教学过程"——缺少可复现性
- 调查研究型论文"对策建议"占全文三分之一以上篇幅，而数据分析只有两三页——头重脚轻，本末倒置
- 理论分析型论文没有明确的分析框架，全文是"想到哪写到哪"的随感式论述
- 教学设计型论文只有设计方案，没有任何形式的实施或反馈——这是"教案"不是"论文"
- 引言中没有以明确问句形式提出研究问题——整篇论文缺少聚焦

## 专项规则权重调整

基于通用6大专项规则，调整教育学科的审查权重：

| 专项规则 | 权重调整 | 理由 |
|---|---|---|
| intro | +1 | 教育学论文的"概念界定"质量直接决定后续论述是否有效；研究问题的聚焦程度是教育学本科论文最常见的区分度来源 |
| literature | +2 | 教育学文献综述质量普遍偏低（罗列式综述、只引中文文献、缺乏批判性评价），需重点审查 |
| method | +2 | 教育学研究方法是最大薄弱环节——实验设计不严谨、问卷未经检验、质性分析缺乏规范，直接决定论文是否有学术价值 |
| analysis | +1 | 数据分析常见统计误用（如小样本用参数检验、混淆相关与因果），需加强审查 |
| conclusion | +1 | "对策建议"是教育学论文的高频雷区——空泛的政策口号式建议是通病 |
| fact | 0 | 教育学论文中事实性描述段落相对较少，保持通用权重 |

## 方法审查补充维度

通用 method 规则之外，教育学科需额外检查以下维度：

### 1. 教学实验设计审查

- **控制组设置**：是否有对照班/控制组？如果没有，论文是否明确承认这是"准实验"或"前实验"设计，并讨论了内部效度威胁？单组前后测设计必须解释为何不设控制组
- **前后测设计**：是否有前测数据证明实验组与控制组基线可比？仅有后测的实验几乎无法得出有效结论
- **干预时长与强度**：教学实验持续多长时间？少于4周的教学干预很难排除新奇效应（novelty effect）。审查者应质疑干预时长是否足以产生真实的学习效果
- **额外变量控制**：是否由同一教师执教实验组和控制组？不同教师执教则教师变量成为混淆因素。是否控制了教学时长、教材版本、班级原有成绩差异？
- **霍桑效应**：是否讨论了被试知道自己参与实验可能带来的行为改变？教育实验中学生知道"被关注"本身就可能提升表现
- **测量工具**：学习效果用什么测量？如果只用"学生满意度问卷"或"学生感受"作为教学效果指标，则测量效度存疑。应有学业成绩测试、标准化量表或经过信效度检验的自编测试

### 2. 问卷调查审查

- **总体界定与抽样**：研究总体（population）是否明确？"中小学教师"是一个极大的群体，必须说明地区、学段、学科范围。抽样方式是否说明？便利抽样必须承认局限性，不能在结论中做超越样本范围的推论
- **样本量**：因子分析至少需要题目数的5-10倍样本；回归分析每个自变量至少需要15-20个样本。样本量低于100份的问卷调查，其统计分析的稳定性需要被质疑
- **问卷来源**：自编问卷必须报告信度（Cronbach's α ≥ 0.70）和效度（内容效度的专家评审、结构效度的因子分析）。引用成熟量表必须注明出处，并说明是否经过本土化修订
- **预测试（pilot study）**：自编问卷是否经过预测试？预测试样本量、修改情况应说明
- **回收率**：问卷回收率必须报告。回收率低于70%时需讨论无应答偏差
- **数据清洗**：是否说明了无效问卷的剔除标准（如全选同一选项、缺失值过多）

### 3. 访谈/质性研究审查

- **访谈提纲**：是否附有访谈提纲？提纲是否与研究问题对应？是否有追问策略？
- **编码方法**：是否说明了编码过程（开放编码→轴心编码→选择性编码，或其他编码框架）？是否给出编码示例（原始语句→初始编码→范畴）？
- **编码者间信度**：是否有第二编码者独立编码并计算一致性（Cohen's Kappa ≥ 0.60）？单人编码需说明如何确保编码的可靠性（如反复阅读、同伴审核）
- **饱和度**：是否说明了理论饱和的判断标准？访谈人数少于6人时需特别关注
- **研究者立场**：质性研究中研究者本身是研究工具，是否反思了自己的立场、预设和可能的偏见？尤其当研究者同时是被研究班级的教师时，"教师-研究者"双重身份需要被明确讨论

### 4. 行动研究审查

- **循环描述**：行动研究必须呈现至少一个完整的"计划—行动—观察—反思"循环。单一循环的行动研究可以接受，但必须说明后续改进方向
- **研究者身份**：行动研究者通常就是实践者（教师），需明确说明"我"在研究中的角色，以及如何处理实践者与研究者的张力
- **证据收集**：行动研究应有多元证据（课堂观察、学生作品、教师反思日志、同伴观察记录等），不能仅凭主观感受判定效果
- **与"教学心得"的区别**：审查者需判断论文是真正的行动研究还是伪装成研究的教学心得。判断标准：是否有系统的证据收集？是否有基于证据的反思而非感悟？是否有理论框架指导？

### 5. 混合方法审查

- **混合理由**：为什么需要同时使用定量和定性方法？不能仅说"为了更全面"，需具体说明两种方法分别回答什么问题
- **整合点**：量化和质性数据在哪个环节整合（数据收集阶段、分析阶段、还是解释阶段）？整合方式是什么（互补、拓展、还是验证）？
- **优先级**：是以量化为主、质性为辅（QUAN→qual），还是反过来？需明确说明

### 6. 涉及未成年人的伦理审查

- **知情同意**：研究对象为中小学生时，是否获得了学生本人的知情同意和家长/监护人的书面同意？
- **匿名保护**：是否使用了化名/编号？是否在论文中泄露了可识别信息（如具体学校名+班级+学生特征的组合）？
- **最小伤害**：教学实验是否确保控制组学生的学习权益不受损害？实验结束后控制组是否也获得了实验组的教学资源？
- **数据保管**：问卷、访谈录音等原始数据的保管与销毁计划是否说明？

## 证据与论证标准补充

通用 analysis/fact 规则之外，教育学科的证据标准：

### 可接受的证据类型

- **量化证据**：标准化测试成绩、问卷量表得分、课堂观察编码频次、学习行为日志数据。使用条件：必须报告描述统计（M, SD）和推断统计（t检验、ANOVA、卡方检验等），并报告效应量（Cohen's d, η²等）。p值报告到小数点后三位，p < .001时报告为 p < .001
- **质性证据**：访谈转录文本、课堂观察记录、学生作品/反思日记、教师教学日志。使用条件：必须有"厚描述"（thick description），直接引用受访者原话并标注编号（如"S3-20231015"表示第3位受访者2023年10月15日的访谈），不能仅用研究者的概括代替原始证据
- **文本证据**：政策文件、课程标准、教材内容。使用条件：需注明文件全称、发布机构、发布年份
- **二手数据**：教育统计年鉴、PISA/TIMSS数据、学校档案数据。使用条件：需注明数据来源、年份、口径，不能将不同口径的数据直接比较

### 证据红线

- **用学生感受代替学习效果**：教学实验中仅用"学生觉得这种教学方式好"作为效果证据，而无客观学业表现数据——这不能证明教学有效
- **用频率代替推断**：问卷数据仅做频率/百分比描述，从不进行差异检验或相关分析，却在结论中做因果推断
- **无来源的事实断言**：声称"研究表明……""大量学者认为……"但不给出任何引用
- **选择性报告**：只报告支持假设的结果，回避不支持的发现（如实验组在某些维度上反而低于控制组的数据被省略）
- **过度推论**：从一个学校的30名教师的问卷数据推论"我国中小学教师普遍……"
- **混淆相关与因果**：问卷调查发现A和B相关，就写成"A导致了B"

### 论证模式期望

- **教学实验型**：假设-验证模式。明确提出假设→设计实验→数据检验假设→接受/拒绝假设。论证链条必须完整，不能假设提了但结论不回应
- **调查研究型**：描述-解释-建议模式。现状描述（what）→ 原因分析（why）→ 对策建议（how）。每一层都需要证据支撑，不能在"why"层级凭空推测
- **质性研究型**：归纳-建构模式。从原始数据中提炼主题/范畴→建构理论框架或命题。需展示从数据到结论的推理路径
- **理论分析型**：概念辨析-逻辑演绎模式。界定概念→梳理逻辑关系→分析问题→提出主张。每一步论证需有文献支撑或逻辑推演，不能仅凭常识

## 常见问题模式

### S级（论文是否成立）

1. **教学实验无控制组且无前测**：表现为"在某班实施了XX教学法，学生成绩提高了"，但既没有对照班，也没有前测数据 → 完全无法排除成熟效应、历史效应等威胁，实验结论不成立 → 修改方向：补充控制组设计，或至少采用单组前后测设计并充分讨论效度威胁
2. **样本与研究总体严重错配**：声称研究"中小学教师"但仅在一所学校发放30份问卷，且未做任何抽样说明 → 样本完全不能代表声称的总体，所有推论均无效 → 修改方向：缩小研究总体的表述范围（如改为"X市Y区小学语文教师"），或扩大样本并说明抽样策略
3. **对策建议与研究发现脱节**：数据分析发现的是问题A、B、C，但对策建议写的是"加强师资培训""完善评价体系""营造良好氛围"等放之四海而皆准的政策口号，与前文发现无逻辑对应关系 → 论文"研究"与"建议"两张皮，研究过程失去意义 → 修改方向：每条建议必须标注"针对第X章发现的Y问题"，建议的具体性要与发现的具体性匹配
4. **研究问题过于宽泛无法操作**：如"小学语文教学存在的问题及对策"——这个题目可以写一本书，本科论文无法驾驭 → 论文必然陷入面面俱到又蜻蜓点水的困境 → 修改方向：聚焦到具体学段+具体教学环节+具体能力维度（如"小学三年级叙事性写作中的支架式教学策略研究"）
5. **伪研究：教学心得冒充教育研究**：全文没有研究问题、没有系统方法、没有数据支撑，实质是"我教了几年书的感想" → 不符合学术论文的基本要素 → 修改方向：重新确定研究问题，选择恰当的研究方法，系统收集和分析数据

### A级（严重影响质量）

1. **文献综述只引中文文献，缺失国际核心框架**：表现为综述只引用中国知网文献，完全不了解该主题的国际研究现状。如研究"深度学习"（教育学意义上的）却不引用Marton & Säljö的经典研究、SOLO分类理论等 → 文献基础薄弱，理论视野受限 → 修改方向：补充该主题领域的国际经典文献和近5年重要外文文献，至少覆盖该方向的奠基性研究
2. **理论框架徒有其名**：声称"以建构主义理论为指导"，但全文的教学设计/研究设计中看不出建构主义的任何具体影响——既没有知识建构的环节设计，也没有社会协商的活动安排 → 理论只是"帽子"，实际研究与理论无关 → 修改方向：删除空挂的理论标签，或者真正依据理论框架的核心主张来设计研究（如建构主义→设计协作探究活动→观察知识建构过程）
3. **混淆"现状描述"与"问题评价"**：在"现状"章节中写"教师对信息技术的运用不足"——这已经是评价判断，不是现状描述。现状应是"X%的教师每周使用信息技术不超过1次"，"不足"是相对于某个标准的评价 → 缺乏评价标准导致论述主观 → 修改方向：先呈现客观数据（现状），再明确评价标准（如课程标准的要求、国际平均水平等），然后做出评价判断
4. **问卷分析中混淆相关与因果**：问卷发现"教师职业倦怠与教学效能感呈负相关"，论文写成"职业倦怠导致教学效能感下降" → 相关关系不能证明因果关系，方向也可能相反 → 修改方向：将因果表述改为相关表述，或引用纵向/实验研究支持因果推断
5. **对策建议超越可操作范围**：建议"国家应改革教师评价制度""提高教师工资待遇" → 本科生的对策应聚焦于教师/学校层面可操作的改进，而非国家政策层面 → 修改方向：将建议分层（教师个人层面、学校管理层面、政策建议层面），并标注各层级的可行性
6. **教学设计缺乏评价对齐**：教学目标写的是"培养学生的批判性思维"，但评价方式只有"期末考试选择题" → 教学目标-教学活动-教学评价不一致（违背"教学评一致性"原则） → 修改方向：依据教学目标设计匹配的评价工具（如批判性思维需要用开放性任务、量规评价）

### B级（影响说服力）

1. **文献综述结构为"张三说……李四说……王五说……"的点名式综述**：每段以一个作者开头，只是转述观点，没有分类、比较、批判 → 综述缺乏整合性，读者无法把握研究脉络 → 修改方向：按主题/争议点/时间线组织文献，对不同观点进行比较和评价
2. **统计方法选择不当但不影响基本结论**：如对Likert 5点量表数据直接计算均值并做t检验（严格来说序数数据不宜用参数检验）→ 如果数据分布接近正态且样本量足够，结果可能仍可接受，但需要说明 → 修改方向：补充数据正态性检验结果，或改用非参数检验作为稳健性检查
3. **引言中缺乏对"研究意义"的分层论述**：只笼统说"具有重要的理论意义和实践意义"但不具体说明 → 读者无法判断研究的价值 → 修改方向：理论意义说明填补了什么研究空白，实践意义说明对谁（教师/学校/教育管理者）有什么具体帮助
4. **访谈引用过长或过少**：引用受访者发言超过半页，或者整个质性分析部分只引用了两三句话 → 前者喧宾夺主，后者证据不足 → 修改方向：每个主题引用2-3段关键语句（3-5行/段），配合研究者的分析阐释
5. **图表呈现不规范**：表格没有三线表格式，图表没有标题和数据来源，图表编号不连续 → 影响论文的专业性 → 修改方向：统一使用三线表，所有图表标注标题、数据来源、样本量
6. **概念界定仅引用词典/百度百科**：核心概念界定引用《辞海》或百度百科，而非学术文献中的定义 → 表明对学术话语体系不熟悉 → 修改方向：引用该概念的原始提出者或权威学术文献的定义，并说明本文采用哪个定义及理由

## 文献综述特殊要求

### 文献来源优先级

1. **第一梯队（必须引用）**：与论文主题直接相关的经典文献和近5年高质量文献
   - 中文核心期刊：《教育研究》《课程·教材·教法》《比较教育研究》《教育学报》《华东师范大学学报（教育科学版）》《教师教育研究》《教育发展研究》《全球教育展望》
   - 外文期刊：*Educational Researcher*, *Review of Educational Research*, *Teaching and Teacher Education*, *Journal of Curriculum Studies*, *Educational Psychology Review*, *Computers & Education*（教育技术方向）
   - 经典著作：与论题相关的教育学/心理学经典著作

2. **第二梯队（可以引用）**：CSSCI来源期刊的其他教育类期刊、学位论文（博士论文优先）、教育类专著

3. **第三梯队（慎重引用）**：普通期刊论文、硕士论文、报纸评论

4. **不可接受**：百度百科、微信公众号文章、没有作者的网页、教辅材料

### 文献数量参考标准

- 本科论文参考文献一般不少于30篇，其中外文文献不少于5篇
- 近5年文献应占总量的三分之一以上
- 文献综述章节引用的文献应占总参考文献的60%以上

### 综述组织方式

- **推荐**：按研究主题/争议焦点组织（如"关于XX的概念界定""关于XX的影响因素研究""关于XX的干预/改进研究"）
- **可接受**：按时间线组织（适用于梳理某一概念/政策的演变）
- **不推荐**：按作者逐一罗列、按"国外研究""国内研究"简单二分（除非确有必要做国际比较）

### 述评要求

综述必须以"述评"结尾，包含：
- 已有研究取得了哪些共识
- 存在哪些分歧或空白
- 本研究如何填补这些空白（与引言中的"研究问题"呼应）

### 各方向关键理论家/框架

审查者可据此判断文献综述的覆盖度：

| 研究方向 | 应知应引的关键人物/框架 |
|---|---|
| 学习理论 | 皮亚杰（认知发展）、维果茨基（最近发展区/支架理论）、布鲁纳（发现学习）、奥苏伯尔（有意义接受学习）、班杜拉（社会学习理论） |
| 教学设计 | 布卢姆（教育目标分类学/掌握学习）、加涅（学习条件/教学事件）、梅里尔（首要教学原理）、乔纳森（建构主义学习环境设计） |
| 课程理论 | 泰勒（课程基本原理）、施瓦布（实践课程观）、派纳（课程理解范式）、古德莱德（课程层次理论） |
| 学前教育 | 蒙台梭利、杜威（经验与教育）、瑞吉欧教育方法、高瞻课程（High/Scope）、《3-6岁儿童学习与发展指南》 |
| 教育评价 | 斯克里文（形成性评价vs总结性评价）、布卢姆（掌握学习评价）、Wiggins & McTighe（逆向设计/UbD）、SOLO分类理论 |

| 体育教育 | 运动教育模型（Siedentop）、体育素养（Physical Literacy）、TGFU（Teaching Games for Understanding）、体育课程标准 |
| 特殊教育 | 融合教育/全纳教育（Inclusive Education）、个别化教育计划（IEP）、通用学习设计（UDL）、萨拉曼卡宣言 |

## 术语与规范

### 术语使用规范

- **核心概念必须在引言/绪论中界定**，且全文使用统一术语。如确定用"核心素养"则不能时而写"关键能力"时而写"核心素养"
- 区分日常用语与学术术语：不能用"上课效果好"代替"教学有效性"；不能用"差生"代替"学业困难学生"或"学习落后学生"
- 外来术语首次出现时标注英文原文，如"最近发展区（Zone of Proximal Development, ZPD）"
- **政策文件术语需使用官方准确表述**：如"立德树人"不能写成"立德育人"，"核心素养"不能写成"核心素质"，引用课程标准需注明版本年份

### 引用格式

- 一般遵循GB/T 7714-2015《信息与文献 参考文献著录规则》，或学校指定的APA格式
- 正文中引用建议使用"作者-年份"制（如"李克强，2020"或"Vygotsky, 1978"），或按学校要求使用顺序编码制
- 引用教育政策文件需注明发文机关和文号，如"教育部（2022）《义务教育课程方案和课程标准（2022年版）》"

### 图表规范

- 表格一律使用三线表（顶线、栏目线、底线）
- 问卷调查的描述统计表需包含：变量名、M（均值）、SD（标准差）、N（样本量）
- 差异检验结果表需包含：t值/F值/χ²值、df（自由度）、p值、效应量
- 质性研究的编码表需包含：一级编码（范畴）、二级编码（类属）、典型引用语句、频次
- 所有图表均需编号并有标题（表标题在表上方，图标题在图下方）

## 结论与建议特殊要求

### 结论的要求

- **逐条回应研究问题**：结论数量应与研究问题数量对应，不能多出来也不能遗漏
- **结论应基于本研究的数据**：不能在结论中突然引入新的数据或论点
- **结论的语气应与证据强度匹配**：小样本便利抽样的研究，结论不能用"证明了""确定了"这样的强因果语气，应使用"发现""表明""提示"等
- **明确说明研究局限**：至少包括样本代表性、方法局限、时间局限

### 对策建议的合理边界

教育学论文的"对策建议"是最容易出问题的章节。审查标准：

1. **建议必须从发现中逻辑推出**：每条建议应标注或可追溯到具体的研究发现。审查者可做"删除测试"——如果把数据分析章节删掉，这些建议是否还能写出来？如果能，说明建议与研究脱节
2. **建议必须具体可操作**：不接受"加强XX""提高XX""完善XX"这类无操作定义的建议。合格的建议应说明：谁来做、做什么、怎么做。例如，不合格："加强教师信息技术培训"；合格："学校可每学期开展2次以信息化教学工具操作为主题的校本研修，由信息技术教师担任主讲，重点训练交互式白板和在线评测工具的课堂应用"
3. **建议应分层且标注可行性**：
   - 教师个人层面（本人可直接行动）
   - 学校/教研组层面（需要学校支持）
   - 区域/政策层面（仅作为参考建议，明确标注为"政策建议"且论文作者无法直接推动）
4. **建议数量适当**：一般3-5条为宜，每条有充分论证。"建议"超过10条通常意味着缺乏聚焦
5. **不能建议已有的做法**：如果建议内容是该学校/地区已经在做的事情，则该建议无意义

### 教育学论文结论的常见病

- **"研究展望"变成新的研究计划**：展望应简要指出方向，而非写出完整的研究设计
- **自我拔高**：声称"本研究丰富了XX理论"——本科论文一般不具备理论创新能力，应表述为"本研究在XX方面提供了实证案例"
- **回避消极结果**：教学实验中实验组没有显著优于控制组的结果被轻描淡写带过——消极结果同样有价值，应如实报告并分析原因

## 各子方向特殊审查要点

### 学前教育

- 研究对象为3-6岁幼儿时，数据收集方式受限（不能用问卷），通常采用观察法、作品分析法、访谈法（访谈家长或教师）
- 游戏活动观察需有明确的观察编码表和时间取样/事件取样策略
- 引用《3-6岁儿童学习与发展指南》《幼儿园教育指导纲要（试行）》时需注明发布年份
- 注意区分"幼儿教育"的学术研究与"幼儿活动设计方案"——后者不是论文

### 小学教育

- 涉及具体学科教学时，需引用该学科的课程标准（如《义务教育语文课程标准（2022年版）》）
- 教学实验的被试为小学生时，尤其注意伦理审查要求
- 小学教育论文容易陷入"经验总结"模式，审查者需判断是否有系统的研究方法

### 教育技术学

- 需审查技术工具的选用是否有理论依据（不能仅因为"新"就用）
- 技术整合教学的研究需明确技术的角色（替代、增强、修改、还是重新定义——SAMR模型）
- 数字化学习平台的数据分析需注意隐私保护
- 区分"技术介绍"与"技术整合研究"——前者不是教育学论文

### 体育教育

- 体能测试数据需说明测试标准、测试环境、测试者资质
- 运动技能评价需有明确的评分标准（如量规）
- 引用《体育与健康课程标准》时注明版本
- 注意区分"运动训练"（偏竞技）和"体育教育"（偏教学）的研究范畴

### 特殊教育

- 涉及残障学生的研究需特别关注伦理保护和去标识化
- 个案研究在特殊教育中是合法且重要的方法，但需说明个案选择的理由和典型性
- 融合教育研究需同时关注特殊学生和普通学生的体验
- 使用标准化评估工具时需说明工具的适用对象和本土化情况
