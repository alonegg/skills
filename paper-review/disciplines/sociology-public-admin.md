# 社会学与公共管理 学科配置

> 本文件是 paper-review 技能的学科叠加层。它不替换通用规则，而是在通用规则基础上增加社会学与公共管理学科特有的审查维度、调整权重、补充常见问题模式。

## 适用范围

- **覆盖学科**：社会学、社会工作、公共管理（行政管理）、政治学、公共政策
- **典型论文类型**：社会调查型、政策分析型、质性研究型、社区/个案研究型
- **识别线索**：出现"社会结构""阶层""社会资本""社区治理""公共服务""行政体制""基层治理""政策执行""社会工作介入""福利""田野调查""深度访谈""编码""扎根理论""制度分析""利益相关者""政策工具""公共利益""治理现代化""社会组织""社会保障"等关键词；引用社会学/公共管理类期刊（《社会学研究》《中国行政管理》等）；研究对象为社会群体、社区、政府部门、政策过程

## 结构期望

### 标准章节结构

社会学与公共管理本科论文按论文类型有四种典型结构：

**社会调查型**
```
第1章 引言（研究背景、研究问题、研究意义、核心概念界定）
第2章 文献综述（相关理论与既有研究梳理、述评）
第3章 研究设计（研究方法、总体与抽样、问卷/量表设计、数据收集过程、分析策略）
第4章 数据收集与分析（描述统计、推断统计、模型结果）
第5章 发现与讨论（主要发现归纳、与既有文献对话、理论解释）
第6章 结论（研究总结、政策含义/实践启示、局限与展望）
```

**政策分析型**
```
第1章 引言（问题的提出、研究意义、核心概念界定）
第2章 理论框架（政策分析的理论工具选择与论证）
第3章 政策背景与现状（政策演变历程、现行政策体系梳理）
第4章 政策问题分析（政策目标与效果差距、执行偏差、利益相关者分析）
第5章 政策建议（基于分析的改进方案、可行性与风险评估）
第6章 结论（研究总结、理论贡献、局限与展望）
```

**质性研究型**
```
第1章 引言（研究背景、研究问题、研究意义）
第2章 文献综述（理论脉络与经验研究回顾）
第3章 研究方法论（方法论立场、研究策略选择、进入田野过程、资料收集方法、编码与分析过程、研究伦理、研究者反身性）
第4章 田野/访谈资料呈现（按主题组织的资料展示，含参与者引述）
第5章 主题分析（从资料中提炼的核心范畴/主题及其关系）
第6章 讨论（发现与既有理论的对话、本土情境的解释）
第7章 结论（研究总结、理论与实践启示、局限与展望）
```

**社区/个案研究型**
```
第1章 引言（研究背景、研究问题、个案选择理由）
第2章 文献综述（相关理论与个案研究方法论回顾）
第3章 个案介绍（个案基本信息、时间线、关键行动者、数据来源说明）
第4章 个案分析（运用分析框架对个案进行系统解读）
第5章 理论对话（个案发现对既有理论的验证、修正或拓展）
第6章 结论（研究总结、分析性推广、局限与展望）
```

### 必须有的章节

- **引言/绪论**：必须包含明确的研究问题（以问句形式提出）、核心概念的学术界定（不可用日常含义替代学术定义）、研究意义的分层论述（理论意义与实践意义分开阐述）
- **文献综述**：必须有，且必须有"述评"小节指出既有研究的不足和本研究的切入点；社会学/公共管理论文尤其强调理论脉络的梳理——不是简单罗列"谁说了什么"，而是展示学术争论的演进线索
- **研究设计/方法**：实证类论文必须有独立的方法章节，详细说明方法选择依据、样本/个案选取策略、资料收集过程、分析方法、伦理措施；质性研究需额外说明研究者立场与反身性
- **结论**：必须回应引言中提出的研究问题，且必须包含研究局限的坦诚讨论

### 可选/特殊章节

- **理论框架**（独立章节）：政策分析型和质性研究型通常需要独立的理论框架章节；社会调查型可并入文献综述
- **伦理声明**：涉及弱势群体（流动人口、残障人士、低保户等）的研究应有独立伦理说明
- **附录**：问卷原件、访谈提纲、编码本、政策文件清单应放入附录

### 结构红线

- 社会调查型论文没有独立的研究设计章节，直接报告"调查结果"——读者无法评估研究的科学性和可信度
- 政策分析型论文没有明确的分析框架，全文是政策内容的罗列和复述，实质是"政策介绍"而非"政策分析"
- 质性研究型论文没有方法论章节，不说明编码过程和饱和度判断——无法区分于"随感式写作"
- 社区/个案研究型论文只有个案描述没有理论对话——这是"新闻报道"不是"学术研究"
- 全文没有理论视角，从头到尾是"现状描述+问题罗列+对策建议"的三段式——缺乏分析深度，不符合社会科学研究的基本要求

## 专项规则权重调整

基于通用6大专项规则，调整本学科的审查权重：

| 专项规则 | 权重调整 | 理由 |
|---|---|---|
| intro | +1 | 社会学/公管论文的研究问题聚焦程度是区分"学术研究"与"工作报告"的关键；概念界定的学术性直接决定后续分析是否有理论深度 |
| literature | +2 | 本学科高度依赖理论框架指导经验研究；文献综述需要展示学术争论脉络而非简单罗列；缺乏理论视角的论文是本领域最普遍的问题 |
| method | +2 | 方法论是本学科本科论文的最大薄弱环节——调查设计不规范、质性研究缺乏系统编码、政策分析没有分析框架，直接决定论文是否具有学术价值 |
| analysis | +1 | 社会学/公管论文需要在经验证据与理论之间建立对话；常见问题是纯描述无分析、数据与理论"两张皮" |
| conclusion | +1 | "对策建议"是公管类论文的高频雷区——空泛口号式建议是通病；社会学论文的结论需要回归理论层面而非停留在现象描述 |
| fact | +1 | 社会学/公管论文涉及大量社会事实的陈述（政策法规、统计数据、制度背景），事实准确性和来源可靠性需重点审查 |

## 方法审查补充维度

通用 method 规则之外，本学科需额外检查以下维度：

### 1. 问卷调查设计审查

- **总体界定与抽样框**：研究总体（population）是否清晰界定？例如"城市社区居民"过于宽泛，需明确城市层级、社区类型、年龄范围等边界。抽样框（sampling frame）是否说明——从什么名单/数据库中抽取样本？便利抽样（如在一个社区门口拦截路人）必须明确承认局限性，不得在结论中做超越样本的推论
- **样本量与代表性**：声称"调查研究"的论文，样本量低于50人时需严格审查其统计分析的合理性。回归分析中每个自变量至少需要15-20个观测值。多元统计分析（因子分析、结构方程等）通常需要200+样本。审查者应检查样本的人口学特征分布是否合理披露
- **问卷回收率**：必须报告发放量、回收量、有效问卷量。回收率低于60%需讨论无应答偏差（non-response bias）。网络问卷需说明发放渠道和筛选机制
- **量表来源与信效度**：自编量表必须报告信度（Cronbach's α >= 0.70）和效度（专家评审记录、探索性/验证性因子分析）。使用成熟量表必须注明原始出处、翻译/修订过程、在本研究中的信度值。审查者应警惕"自编问卷"仅有几个项目且未经任何检验的情况
- **代表性讨论**：样本特征与总体特征的比较是否呈现？例如调查某市公务员，是否说明了样本在性别、年龄、职级、部门等方面的分布与总体的差异？

### 2. 质性研究方法审查

- **编码程序**：是否说明了编码的具体步骤？扎根理论应有开放编码（open coding）→轴心编码（axial coding）→选择性编码（selective coding）的清晰过程。主题分析应说明主题的生成过程（Braun & Clarke六步法或其他明确框架）。审查者应检查论文是否给出了从原始语句到初始编码到范畴的示例
- **备忘录写作**：质性研究过程中是否有备忘录（memo）记录研究者的分析思路演变？缺乏备忘录说明分析过程不够系统
- **理论饱和**：是否说明了判断理论饱和（theoretical saturation）的标准——即新增访谈不再产生新的编码/范畴？访谈人数少于8人的扎根理论研究需特别关注饱和度问题。但需注意：并非所有质性研究都追求饱和，现象学研究的样本标准不同
- **资料管理与审计线索**：访谈录音是否转录？转录是否经受访者确认（member checking）？编码是否使用了NVivo、ATLAS.ti等软件辅助？是否保留了可供审计的分析过程记录（audit trail）？
- **信实度策略**：是否采用了三角验证（triangulation）、同伴审核（peer debriefing）、负面案例分析（negative case analysis）、长期浸入（prolonged engagement）等策略中的至少两种？

### 3. 访谈方法论审查

- **访谈类型**：是否明确说明采用半结构化访谈（semi-structured）还是非结构化访谈（unstructured）？访谈提纲是否附于附录？提纲问题是否与研究问题对应？
- **知情同意**：是否获得受访者的知情同意？是否说明了知情同意的具体形式（书面/口头）和内容（研究目的、录音许可、匿名保障、退出权利）？
- **录音与转录**：是否录音？是否逐字转录？转录文本的准确性如何保障？若因条件限制未能录音，是否说明了替代的记录方式（如当场笔记+事后扩充）？
- **成员检验（member checking）**：是否将分析结果反馈给部分受访者进行确认？这是提升质性研究可信度的重要手段
- **受访者选取**：是否说明了受访者的选取标准（目的性抽样的具体策略）？受访者的基本信息是否以匿名方式呈现（如列表显示编号、性别、年龄、职业等）？

### 4. 政策分析框架审查

- **分析框架选择**：政策分析型论文必须有明确的理论/分析框架。常用框架包括：
  - Kingdon多源流模型（Multiple Streams Framework）：适用于分析政策议程设置过程
  - 倡导联盟框架（Advocacy Coalition Framework, ACF）：适用于分析政策变迁中的利益博弈
  - 制度分析与发展框架（IAD Framework, Ostrom）：适用于分析制度安排与集体行动
  - 政策过程理论（议程设置→政策制定→执行→评估）：适用于分析政策全周期
  - Smith政策执行过程模型：适用于分析政策执行偏差
  - 渐进主义（Lindblom）vs理性模型：适用于分析决策模式
- **框架适配性**：选择的分析框架是否与研究问题匹配？审查者应检查：框架的核心变量是否在分析中被真正操作化？还是只在文献综述中出现一次，后文再未提及？
- **中国情境适用性**：西方政策分析框架移植到中国语境时，是否讨论了适用性和需要调整之处？例如Kingdon的多源流模型假定的政治体制与中国不同，直接照搬需要说明理由
- **政策文本分析**：如果涉及政策文本分析，是否说明了文本选取标准、分析维度、分析方法（如内容分析法的编码类目）？

### 5. 混合方法审查

- **混合理由**：为什么需要同时使用定量和定性方法？不能仅说"为了更全面"。需具体说明：定量部分回答什么问题？定性部分回答什么问题？两者如何互补？
- **设计类型**：是顺序设计（sequential）还是并行设计（concurrent）？如果是顺序设计，哪个阶段在前、后一阶段如何利用前一阶段的结果？如果是并行设计，两套数据在什么环节汇合？
- **整合策略**：定量和定性结果在什么环节整合？整合方式是什么？（互补解释、交叉验证、还是拓展理解？）仅在结论中并列报告两套结果不算"整合"
- **合法化策略（legitimation）**：是否讨论了混合方法设计的效度问题？如样本整合合法性（quantitative和qualitative样本是否来自同一总体）、范式整合合法性（不同认识论立场如何共存）

### 6. 个案研究方法审查

- **个案选取理由**：为什么选择这个个案而非其他？必须说明个案的典型性（typical case）、极端性（extreme case）、关键性（critical case）或揭示性（revelatory case），不能仅因为"方便获取"就选择
- **多元证据来源（三角验证）**：个案研究的核心要求是多元数据来源。审查者应检查：是否使用了至少两种数据来源（如访谈+文档、观察+访谈、档案+问卷）？不同来源的数据是否相互印证？
- **分析性推广**：个案研究的结论不能做统计推广（"因此全国的社区都……"），只能做分析性推广（analytic generalization）——即讨论本个案的发现在什么条件下可能适用于其他情境
- **证据链**：从原始数据到结论之间是否有清晰的证据链（chain of evidence）？读者能否从结论追溯到具体的数据来源？

### 7. 社会工作研究特殊审查

- **研究与实务的区分**：社会工作论文容易混淆"服务记录/工作报告"与"学术研究"。审查标准：是否有明确的研究问题（而非服务目标）？是否有系统的方法论（而非工作日志）？是否有基于证据的分析（而非经验感悟）？
- **介入研究**：如果论文涉及社会工作介入（intervention），是否说明了介入的理论依据、介入方案设计、效果评估指标和评估方法？仅描述"我们做了什么"而不评估效果，不是研究
- **伦理特殊性**：社会工作研究涉及的服务对象通常是弱势群体（低收入家庭、残障人士、儿童、老年人等），伦理审查需格外严格——知情同意是否获得？"研究者-社工"双重角色如何处理？是否存在权力不对等影响自愿参与？

### 8. 伦理审查维度

- **知情同意**：涉及人类被试的研究是否获得知情同意？弱势群体（流动人口、低保户、残障人士、未成年人）是否有额外的知情同意保障？
- **匿名化**：受访者是否使用编号/化名？社区/组织是否匿名化处理？个人信息（姓名+单位+职务的组合）是否足以识别具体个人？
- **敏感话题处理**：涉及收入、政治态度、违法行为、心理健康等敏感话题时，是否说明了保密措施和风险控制？
- **研究者与被研究者的权力关系**：研究者是否是被访者的上级/管理者/社工？如果是，如何确保受访者真正自愿参与且回答不受影响？

## 证据与论证标准补充

通用 analysis/fact 规则之外，本学科的证据标准：

### 可接受的证据类型

- **量化证据**：问卷调查数据（需完整报告抽样、信效度、统计分析过程）、官方统计数据（统计年鉴、人口普查、政府公报等）。使用条件：描述统计（频率、均值、标准差）和推断统计（卡方检验、t检验、回归分析等）都需报告，并报告效应量。统计检验方法需与数据类型匹配（如分类变量用卡方、连续变量用t/F、有序变量慎用参数检验）
- **质性证据**：访谈转录文本（需标注受访者编号，如"受访者A3-20250615"）、田野观察笔记、档案文本。使用条件：必须有"厚描述"（thick description），直接引用受访者原话，不能仅用研究者的概括代替原始证据；每个分析主题需有至少2-3段不同受访者的引述支撑
- **政策文本证据**：政府文件（需注明全称、发文机关、文号、发布日期）、法律法规、领导讲话/政府工作报告（需注明具体出处和年份）、政策实施细则。使用条件：需区分"政策文本"和"政策实践"——文件规定不等于实际执行情况
- **个案证据**：社区/组织/事件的多源数据（访谈+文档+观察+档案）。使用条件：需构建时间线、识别关键行动者、展示证据之间的相互印证
- **二手数据**：中国统计年鉴、中国社会状况综合调查（CSS）、中国综合社会调查（CGSS）、中国家庭追踪调查（CFPS）、世界银行/联合国数据。使用条件：需注明数据来源、年份、口径，不同口径数据不可直接比较

### 证据红线

- **"调查研究"实为便利样本小调查**：声称"调查研究"但实际只在身边发了不到50份问卷、没有抽样框、没有回收率报告、没有信效度检验——这不是社会调查，是"问卷作业"。论文基于此数据得出的一切推论均不成立
- **政策建议无证据基础**：对策建议部分出现"政府应加大投入""社会应提高重视""完善法律法规"等口号式建议，与前文分析发现无逻辑对应关系——这是政治口号不是学术建议
- **质性研究无系统编码**：声称"质性研究"但没有编码过程，只是从访谈中挑选了几句支持作者预设观点的引语——这是选择性引用不是质性分析
- **没有理论框架的纯描述**：全文只有"现状"描述（X是什么情况、Y存在什么问题），没有任何理论视角解释"为什么"——这是调查报告不是学术论文
- **混淆事实与评价**：在"现状"章节中直接写"政策执行不力""治理效果差"等评价判断，但不给出评价标准和支撑数据——缺乏标准的评价是主观臆断
- **选择性引用官方数据**：只引用支持自己论点的数据，刻意回避不利数据。例如论证某政策效果好时只引用改善指标，不提恶化指标
- **将相关当因果**：问卷数据发现两个变量相关，就写成"A导致了B"。横截面调查数据原则上不能做因果推断

### 论证模式期望

- **社会调查型**：假设检验模式或描述-解释模式。提出研究假设（或描述性问题）→收集数据→统计检验→接受/拒绝假设（或描述现状→寻求解释）。论证链条完整，每个结论都有对应的数据支撑
- **政策分析型**：框架-诊断-处方模式。建立分析框架→诊断政策问题（目标偏离、执行缺陷、效果不足）→提出改进方案。分析框架必须贯穿始终，不能前面提了一个框架后面分析时不用
- **质性研究型**：归纳-建构模式。从原始资料中提炼编码/范畴→构建范畴之间的关系→生成理论命题或深层解释。需展示从数据到结论的完整推理路径，让读者能"看到"研究者如何从原始材料中得出结论
- **社区/个案研究型**：描述-分析-推广模式。深描个案→运用理论框架分析个案→讨论分析性推广的条件和边界。个案描述需"厚"（thick），分析需有理论穿透力，推广需审慎

## 常见问题模式

### S级（论文是否成立）

1. **便利样本小调查冒充"调查研究"**：表现为在一个社区/单位发放不到50份问卷，无抽样框、无回收率报告，却声称"调查了XX群体的XX情况"并做统计推断 → 样本完全不具备代表性，所有基于此数据的统计推断和结论均无效 → 修改方向：要么扩大样本并建立规范的抽样程序，要么缩小研究总体表述范围并采用个案/质性方法替代
2. **政策建议纯属口号**：表现为"对策建议"章节充斥"政府应加大投入""社会应提高重视""完善相关法律法规""加强宣传教育"等放之四海而皆准的空话，与前文数据分析发现没有逻辑对应关系 → 论文"研究"与"建议"两张皮，研究过程失去意义 → 修改方向：每条建议必须标注"基于第X章发现的Y问题"，说明具体做什么、谁来做、如何做、预期效果，建议的具体性必须与发现的具体性匹配
3. **质性研究无系统编码**：表现为声称采用"深度访谈法""扎根理论"，但全文不见编码过程，只在论述中穿插几句受访者引语作为"点缀"，实质是研究者先有预设结论再找引语支撑 → 这不是质性研究而是选择性引证，研究的可信度和可靠性无法保证 → 修改方向：重新进行系统编码（展示开放编码→轴心编码→选择性编码的完整过程），给出编码示例表，说明理论饱和判断
4. **无理论框架的纯描述论文**：表现为全文结构是"XX的现状→XX存在的问题→XX的对策建议"三段式，没有任何理论视角指导分析，通篇是现象的罗列和常识性评论 → 缺乏分析深度，本质是调查报告或新闻稿而非学术论文 → 修改方向：引入恰当的理论框架（如组织理论、制度主义、社会资本理论等）作为分析透镜，用理论概念解释"为什么"而非仅描述"是什么"

### A级（严重影响质量）

1. **文献综述按作者罗列**：表现为"张三（2018）认为……李四（2019）指出……王五（2020）发现……"的点名式综述，缺乏主题整合、观点比较和批判性评价 → 读者无法把握该领域的研究脉络和学术争论 → 修改方向：按研究主题、理论立场或争议焦点重新组织文献，将不同学者的观点在同一主题下进行比较和评述
2. **研究设计章节过于单薄**：表现为方法章节只有"本研究采用问卷调查法/访谈法"一句话，不讨论信度、效度、局限性、研究者立场 → 读者无法评估研究的科学性 → 修改方向：补充总体界定、抽样策略、工具来源与检验、数据分析方法、效度威胁讨论、研究局限
3. **访谈资料呈现与研究者解释混为一谈**：表现为在同一段落中既引用受访者原话又夹杂研究者的评论和判断，读者无法分辨哪些是数据、哪些是分析 → 降低研究的透明度和可信度 → 修改方向：明确区分"资料呈现"（受访者说了什么，用引号和编号标注）和"研究者分析"（研究者认为这意味着什么），可以先呈现再分析，或分段标注
4. **政策分析只描述不评估**：表现为论文详细梳理了某项政策的内容和演变历程，但不评估政策效果、不分析执行偏差、不讨论目标与结果的差距 → 这是"政策介绍"而非"政策分析"，缺乏学术价值 → 修改方向：建立政策评估的标准（效率、公平、可行性等维度），用数据/证据评估政策在各维度上的表现，识别执行中的问题及原因
5. **社会工作论文混淆服务描述与研究**：表现为论文主体是"我们在XX社区开展了XX服务，服务内容包括……服务过程是……"的实务记录，缺少研究问题、理论框架和系统的效果评估 → 这是工作报告不是学术论文 → 修改方向：明确研究问题（如"XX介入模式对XX群体的XX效果如何？"），建立理论依据，设计效果评估方案（前后测、控制组、或至少系统的质性评估）

### B级（影响说服力）

1. **"背景"章节过长且读如新闻报道**：表现为引言或背景章节花费大量篇幅描述社会现象（"随着城市化进程的加快……""随着老龄化问题的日益严峻……"），内容缺乏学术性，更像报纸社论 → 背景应简洁地建立研究的社会语境，而非替代文献综述 → 修改方向：压缩背景描述至1-2页，聚焦于与研究问题直接相关的社会事实，引用学术文献和官方数据而非笼统描述
2. **问卷项目未公开或缺乏检验**：表现为论文使用问卷调查但正文和附录中均未呈现问卷具体项目，也未报告信度和效度数据 → 读者无法评估测量工具的质量 → 修改方向：在附录中附上完整问卷，正文中报告各量表的Cronbach's α值和效度检验结果
3. **访谈引语装饰性使用**：表现为每个观点后附上一段受访者引语，但引语内容与分析观点的对应关系不清晰，或引语只是重复了研究者已经说过的话 → 引语失去了证据功能 → 修改方向：每段引语前后都应有分析阐释——引语前说明为什么引用这段话，引语后分析这段话如何支撑/丰富/复杂化该主题
4. **对策建议超越证据基础**：表现为基于一个社区的调查就提出全市/全国层面的政策建议，或基于一个部门的个案就建议"全面改革行政体制" → 建议的范围远超研究发现的范围 → 修改方向：建议应分层（微观层面-本研究直接支持的、中观层面-可合理推论的、宏观层面-仅作为参考方向），并明确标注各层建议的证据强度
5. **质性研究缺乏反身性**：表现为研究者不讨论自身的立场、预设、身份对研究过程和结论的可能影响 → 降低研究的可信度，特别是研究者同时是研究场域的"内部人"（如研究自己所在的社区/单位）时 → 修改方向：增加研究者反身性段落，讨论"我是谁""我为什么关注这个问题""我的身份如何影响了田野进入和资料收集""我如何控制自身偏见"
6. **中国行政体制术语不准确**：表现为混淆"街道办事处"与"社区居委会"的行政性质、不清楚"科级""处级"对应的决策权限、将"事业单位"与"政府部门"混为一谈 → 反映出对中国行政体制的理解不足 → 修改方向：核实行政层级和机构性质的准确表述，参照《中华人民共和国地方各级人民代表大会和地方各级人民政府组织法》等法律文件

## 文献综述特殊要求

### 文献来源优先级

1. **第一梯队（必须引用）**：与论文主题直接相关的经典文献和近5年高质量文献
   - 中文核心期刊：《社会学研究》《社会》《公共管理学报》《中国行政管理》《政治学研究》《管理世界》《社会学评论》《中国社会科学》《公共行政评论》《中国社会工作研究》
   - 外文期刊：*American Sociological Review*, *American Journal of Sociology*, *Social Forces*, *Public Administration Review*, *Journal of Public Administration Research and Theory*, *Governance*, *Policy Sciences*, *Social Service Review*, *British Journal of Sociology*, *Annual Review of Sociology*
   - 经典著作：与论题相关的社会学/公管经典著作（如韦伯、涂尔干、布迪厄、福柯、吉登斯、费孝通等的代表作）

2. **第二梯队（可以引用）**：CSSCI来源期刊的其他社会科学类期刊、高质量博士论文、社会科学专著、重要的工作论文（working paper）

3. **第三梯队（慎重引用）**：普通期刊论文、硕士论文、报纸评论、智库报告（需注明机构和可靠性）

4. **不可接受**：百度百科、微信公众号文章、没有作者的网页、无法核实来源的"数据"

### 文献数量参考标准

- 本科论文参考文献一般不少于30篇，其中外文文献不少于5篇
- 近5年文献应占总量的三分之一以上
- 文献综述章节引用的文献应占总参考文献的60%以上
- 社会学理论类论文的文献量通常需要更多（40-60篇），因为理论梳理本身就是核心工作

### 综述组织方式

- **推荐**：按理论立场/研究视角组织（如"制度主义视角的研究""社会资本视角的研究""治理理论视角的研究"），展示不同视角之间的争论与互补
- **推荐**：按研究主题/争议焦点组织（如"关于XX的概念界定""关于XX的影响因素""关于XX的效果评估"）
- **可接受**：按时间线组织（适用于梳理某一概念/政策的学术演变史）
- **不推荐**：按作者逐一罗列、按"国外研究""国内研究"简单二分（除非确有必要做中外比较且两个学术传统确实有本质差异）

### 述评要求

综述必须以"述评"或"研究评价与本文定位"结尾，包含：
- 已有研究取得了哪些共识
- 存在哪些理论争论或经验空白
- 方法论上有何局限（如多数研究为定量缺乏质性、或多为个案缺乏比较等）
- 本研究如何填补这些空白（与引言中的"研究问题"形成呼应）

审查者可据此判断文献综述的覆盖度：

### 特别要求：必须与中国语境对话

- 是否讨论了西方理论在中国语境中的适用性和局限性？
- 是否引用了中国学者对该问题的本土化研究？
- 是否关注了中国特有的制度背景（如党政关系、条块体制、户籍制度、单位制遗产等）对研究问题的影响？
- 纯粹用西方理论解释中国现象而不做任何情境化调适，应视为A级问题

## 术语与规范

### 术语使用规范

- **核心概念必须在引言/绪论中进行学术界定**，且全文统一使用同一术语。如确定用"社会资本"则不能时而写"社会网络资源"时而写"社会关系"代替
- 区分日常用语与学术术语：不能用"关系"代替"社会资本"或"社会网络"（除非讨论的正是"关系"这一本土概念）；不能用"当官的"代替"行政官员"或"基层干部"
- 外来概念首次出现时标注英文原文，如"场域（field）""惯习（habitus）""嵌入性（embeddedness）""治理（governance）""新公共管理（New Public Management, NPM）"
- **中国行政体制术语须准确**：
  - "街道办事处"是政府派出机关，"社区居民委员会"是基层群众自治组织，两者性质不同，不可混淆
  - 行政层级表述准确：国家级-省部级-厅局级-县处级-乡科级
  - "事业单位"≠"政府部门"，"国有企业"≠"政府机构"
  - "基层政府"通常指县级及以下政府，不包括地级市及以上
- **政策文件引用规范**：须注明发文机关全称、文件全称、文号和发布年份。如"国务院办公厅《关于加强基层治理体系和治理能力现代化建设的意见》（国办发〔2021〕XX号）"

### 引用格式

- 一般遵循GB/T 7714-2015《信息与文献 参考文献著录规则》，或学校指定的APA格式
- 正文中引用建议使用"作者-年份"制（如"费孝通，1947"或"Bourdieu, 1986"），或按学校要求使用顺序编码制
- 引用法律法规需注明全称和施行日期
- 引用统计数据需注明来源机构、年份、统计口径

### 图表规范

- 表格一律使用三线表
- 问卷调查的描述统计表需包含：变量名、M（均值）、SD（标准差）、N（样本量）
- 差异检验结果表需包含：统计量（t/F/χ²）、df（自由度）、p值、效应量
- 质性研究的编码表需包含：一级编码（范畴）、二级编码（类属）、三级编码（概念/初始编码）、典型引用语句、频次
- 政策梳理表需包含：政策名称、发文机关、发布时间、核心内容、与本研究的关联
- 所有图表均需编号并有标题（表标题在表上方，图标题在图下方），标注数据来源

## 结论与建议特殊要求

### 结论的要求

- **逐条回应研究问题**：结论数量应与研究问题对应，不能多出未经论证的新结论，也不能遗漏已提出的研究问题
- **结论应基于本研究的数据/证据**：不能在结论中引入新的数据或文献来支撑论点
- **结论语气与证据强度匹配**：
  - 大样本随机抽样调查可以用"研究发现""结果表明"
  - 小样本便利抽样或个案研究应用"本研究提示""在本研究的情境中"
  - 质性研究应避免频率化表述（如"大多数受访者认为……"——质性研究不以频率取胜）
- **明确说明研究局限**：至少包括样本/个案的代表性局限、方法局限、研究者立场可能带来的偏差、时效性局限

### 对策建议/政策启示的合理边界

社会学/公管论文的"对策建议"或"政策启示"是最容易出问题的章节。审查标准：

1. **建议必须从研究发现中逻辑推出**：每条建议应可追溯到具体的研究发现。审查者可做"删除测试"——如果把数据分析章节删掉，这些建议是否还能写出来？如果能，说明建议与研究脱节
2. **建议必须具体可操作**：不接受"加大投入""提高重视""完善制度"等无操作定义的口号。合格的建议应说明：谁（哪个层级/部门）、做什么（具体行动）、怎么做（实施路径）。例如，不合格："完善社区养老服务体系"；合格："街道办事处可依托现有社区日间照料中心，引入专业社会工作机构运营管理，设立老年人需求评估-服务匹配-效果跟踪的闭环机制，重点解决本研究发现的'服务供给与需求错位'问题"
3. **建议应分层且标注证据强度**：
   - 微观层面（社区/组织层面，本研究直接支持）
   - 中观层面（区县/行业层面，可合理推论但需谨慎）
   - 宏观层面（国家政策层面，仅作为方向性参考，明确标注"超出本研究直接证据范围"）
4. **建议数量适当**：一般3-5条为宜。超过8条通常意味着缺乏聚焦
5. **建议必须体现对中国行政体制的理解**：不能提出在现行体制下明显不可行的建议（如建议"取消街道办事处"），也不能忽视政策建议涉及的条块关系和层级管辖问题
6. **社会学论文的结论应回归理论层面**：社会学论文不一定需要"对策建议"，更重要的是理论贡献——本研究的发现对既有理论有何验证、修正或拓展？个案发现在什么条件下可以做分析性推广？

### 社会学/公管论文结论的常见病

- **口号化**：结论和建议读起来像政府文件或领导讲话——"坚持以人民为中心""深入贯彻新发展理念"——这些表述在论文结论中不具有学术信息量
- **超越证据的宏大叙事**：基于一个社区的调查就谈"推进国家治理体系和治理能力现代化"——论文的证据基础完全支撑不起这个层次的讨论
- **回避消极发现**：调查发现某项政策效果不佳，但结论中轻描淡写甚至不提——消极发现同样有学术价值，应如实报告并分析原因
- **理论贡献自我拔高**："本研究丰富了XX理论""本研究构建了XX的分析框架"——本科论文一般不具备理论创新能力，应表述为"本研究为XX理论在中国XX领域的应用提供了经验案例"

## 各子方向特殊审查要点

### 社会学

- **理论自觉**：社会学论文必须有理论视角——不能只是"社会问题+数据描述+对策建议"。审查者应检查：论文是否清楚地说明了使用什么理论视角来分析问题？该理论视角是否确实贯穿了分析过程？
- **经典与前沿的平衡**：社会学强调学术传统的积累，论文应既引用经典理论家的奠基性工作，又关注近5年的前沿研究
- **"社会调查"vs"社会研究"的区分**：审查者需判断论文是真正的社会学研究（有理论问题意识、系统方法论、分析深度），还是只是社会调查（描述现状、罗列问题、提出建议）。区分标准：是否有理论框架指导？是否在经验发现与理论之间建立了对话？

### 社会工作

- **研究与实务的区分**：社会工作论文容易变成"实习报告"或"服务记录"。审查标准：是否有明确的研究问题（不同于服务目标）？是否有系统的研究方法？是否有基于证据的分析而非经验感悟？
- **介入的理论依据**：社会工作介入方案必须有理论依据（如任务中心模式、认知行为理论、生态系统理论、优势视角等），不能仅凭"实践经验"设计介入方案
- **效果评估**：介入研究必须有效果评估——不能只说"服务对象表示满意"就算有效。应有可测量的指标（如量表前后测、行为变化记录、目标达成度评估）
- **伦理敏感性**：社会工作研究对象通常是弱势群体，伦理要求高于一般社会研究。知情同意、匿名保护、"研究者-社工"双重身份的处理都需要明确讨论

### 公共管理（行政管理）

- **中国行政体制的理解**：公管论文必须体现对中国行政体制特殊性的理解——党政关系、条块体制、压力型体制、"运动式治理"与"常规治理"的交替、"政策试点"机制等。简单套用西方公共管理理论而不讨论中国体制的特殊性是A级问题
- **"管理学论文"vs"行政学论文"的区分**：公共管理不等于企业管理。公共部门的特殊性（非营利性、政治约束、多重委托代理、公共利益导向）应体现在分析中
- **数据来源的特殊性**：中国政府数据的可获得性和可靠性需要被讨论。使用政府公报数据时需注意统计口径变化、数据可能的"水分"等问题，但讨论应基于学术文献而非主观猜测

### 政治学

- **规范性与经验性的区分**：政治学论文需区分规范性研究（应然层面）和经验性研究（实然层面），不能在两者之间随意跳跃。审查者应检查：论文的研究问题是规范性的还是经验性的？方法和论证是否与问题类型匹配？
- **意识形态敏感性**：涉及政治制度、民主、权力等议题时，论述应基于学术文献和经验证据，避免纯粹意识形态化的表述（无论是哪个方向的）
- **比较视野**：政治学研究鼓励比较视野。如果论文涉及制度比较，需说明比较对象的选取理由和比较框架

### 公共政策

- **政策全周期意识**：政策分析不等于"政策内容介绍"。论文应关注政策的某个或多个环节：议程设置（为什么这个问题进入了政策议程？）、政策制定（政策方案是如何形成的？）、政策执行（执行中出现了什么偏差？为什么？）、政策评估（政策效果如何？用什么标准衡量？）
- **利益相关者分析**：政策分析应识别相关利益群体及其立场，不能假设政策只有一个"正确"方向
- **"政策建议"的专业性**：公共政策论文的政策建议需要比一般论文更具政策可行性分析——考虑政治可行性（决策者是否有动机采纳？）、行政可行性（执行能力是否具备？）、经济可行性（财政是否可承受？）、法律可行性（是否需要修法？）

## 区分"社会调查"（新闻/行政）与"社会研究"（学术）

这是社会学/公管本科论文中最核心的质量分水岭。审查者应使用以下检查清单判断论文是否达到了"学术研究"而非仅仅是"社会调查"的标准：

| 检查维度 | "社会调查"（不达标） | "社会研究"（达标） |
|---|---|---|
| 问题意识 | "了解XX的现状" | "为什么XX会出现？""XX对YY的影响机制是什么？" |
| 理论使用 | 无理论，或理论只在文献综述中出现 | 理论贯穿全文，指导研究设计和数据分析 |
| 方法论 | "我做了问卷/访谈" | 详细说明抽样策略、信效度、编码过程、分析方法 |
| 数据分析 | 频率表+百分比描述 | 推断统计/系统编码/理论建构 |
| 结论 | "存在XX问题，建议加强XX" | "本研究发现XX，这与YY理论的预期一致/不一致，因为……" |
| 知识贡献 | 提供了一份现状报告 | 对既有理论或经验知识有增量贡献 |
