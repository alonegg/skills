# 机械土木类 学科配置

> 本文件是 paper-review 技能的学科叠加层。它不替换通用规则，而是在通用规则基础上增加机械土木类学科特有的审查维度、调整权重、补充常见问题模式。

## 适用范围

- **覆盖学科**：机械工程、机械设计制造及其自动化、车辆工程、土木工程、建筑环境与能源应用工程、工程管理、工程力学
- **典型论文类型**：
  - **设计计算型**：基于国标/行标完成某机构、零部件或结构的设计计算（如减速器设计、桥梁截面设计）
  - **有限元分析型**：使用 ANSYS、ABAQUS、COMSOL 等软件对结构或零件进行仿真分析
  - **实验研究型**：通过物理实验获取材料性能、结构承载力、振动特性等数据
  - **工艺优化型**：针对加工工艺、施工流程进行参数优化或流程改进
  - **工程管理型**：基于实际工程项目开展进度、成本、质量或风险管理研究
- **识别线索**：论文中出现工程图纸（CAD）、有限元网格/云图、材料力学公式、载荷工况表、安全系数校核、施工组织设计、GB/T 标准引用、应力-应变曲线等

## 结构期望

### 标准章节结构

本学科论文结构因论文类型而异，审查时应先判断论文属于哪种类型，再对照对应结构：

**设计计算型**（最常见）：
```
第1章 绪论（研究背景、国内外现状、设计任务与技术要求）
第2章 总体方案设计（方案比选、总体布局、工作原理）
第3章 主要零部件设计计算（载荷分析、强度校核、刚度校核、寿命计算）
第4章 关键结构有限元验证（建模、加载、结果分析）
第5章 工程图纸与三维建模说明
第6章 总结与展望
```

**有限元分析型**：
```
第1章 绪论（研究背景、国内外研究现状、本文研究内容）
第2章 理论基础（力学模型、本构方程、失效准则）
第3章 有限元建模（几何模型、材料参数、网格划分、边界条件与载荷）
第4章 计算结果与分析（应力/应变/位移分布、参数影响规律）
第5章 模型验证（与解析解/实验数据对比）
第6章 总结与展望
```

**实验研究型**：
```
第1章 绪论
第2章 文献综述与理论基础
第3章 实验方案设计（试件制备、设备与仪器、加载方案、数据采集）
第4章 实验结果与分析
第5章 理论或数值模拟对比验证
第6章 总结与展望
```

**工艺优化型**：
```
第1章 绪论
第2章 工艺现状分析（现有工艺流程、存在的问题）
第3章 优化方案设计（工艺参数选取、正交实验/响应面设计）
第4章 实验/仿真结果与分析
第5章 优化方案验证与经济性评价
第6章 总结与展望
```

**工程管理型**：
```
第1章 绪论
第2章 理论基础与文献综述
第3章 项目概况与数据来源
第4章 管理方案设计（进度计划/成本估算/风险评估/质量控制）
第5章 方案实施分析与效果评价
第6章 总结与建议
```

### 必须有的章节

- **绪论**：必须包含明确的工程问题或设计任务描述，不能只有宽泛的"行业发展背景"；必须指出论文要解决的具体技术问题
- **理论基础/设计依据**：必须给出所引用的标准、规范、力学原理，不能直接跳到计算
- **核心工作章节**：设计计算型必须有完整的校核计算过程；有限元型必须有建模细节和结果分析两个独立部分
- **验证环节**：设计必须有强度/刚度校核，有限元必须有网格收敛性分析或与其他方法的对比验证，实验必须有数据分析
- **总结与展望**：必须基于实际工作成果进行总结，展望必须指向可操作的改进方向

### 可选/特殊章节

- **经济性分析**：当涉及产品设计或工程项目时需要成本估算
- **装配与制造工艺**：当设计成果需投入加工制造时需要说明
- **施工组织设计**：土木工程类论文涉及实际施工时需要单独章节
- **环境与安全评价**：涉及施工安全或环境影响时需要讨论

### 结构红线

- **设计无校核**：给出了设计方案和参数但没有进行强度、刚度、稳定性等校核计算——结构不完整，S级问题
- **有限元无验证**：仿真分析给出了漂亮的云图但无任何验证手段（解析对比、实验对比、网格收敛性分析）
- **理论基础缺失**：没有交代设计所依据的标准和力学原理，直接给出数值结果
- **结论章节照抄摘要**：总结与展望与摘要内容完全重复，没有基于核心工作的增量总结

## 专项规则权重调整

基于通用6大专项规则，调整本学科的审查权重：

| 专项规则 | 权重调整 | 理由 |
|---|---|---|
| intro | 0 | 机械土木类绪论通常结构规范，无特殊调整 |
| literature | +1 | 需检查是否综述了相关国标/行标和工程案例，而非仅引用教材 |
| method | +2 | 设计计算过程和有限元建模方法是本学科论文的核心，必须重点审查完整性与规范性 |
| analysis | +2 | 校核计算、仿真结果分析、实验数据处理的严谨性是质量关键区分度 |
| conclusion | +1 | 工程类论文结论必须有明确的量化成果总结，不能空泛 |
| fact | +1 | 工程参数、材料属性、标准条文等事实性描述的准确性直接影响设计安全 |

## 方法审查补充维度

通用 method 规则之外，本学科需额外检查：

### 设计计算规范

1. **设计依据完整性**：设计是否明确引用了所依据的国家标准（GB/T）、行业标准（JB/T、JG/T、CJJ）或设计手册？设计参数（如许用应力、安全系数、载荷系数）是否有出处？无依据的设计参数等同于无效计算
2. **载荷工况分析**：是否完整列出了所有工况（正常工作、起动、制动、极限载荷、疲劳载荷等）？载荷组合是否符合规范要求？是否有遗漏的关键工况（如地震工况、风载荷、温度应力）？
3. **安全系数校核**：是否对关键零件/构件进行了强度、刚度、稳定性校核？安全系数是否满足标准要求？校核过程是否展示了完整推导？
4. **材料选择论证**：材料牌号选择是否合理？是否给出了材料的力学性能参数（屈服强度、抗拉强度、弹性模量、泊松比等）及其数据来源？是否考虑了工作环境（温度、腐蚀、疲劳）对材料性能的影响？

### 有限元分析规范

5. **网格收敛性研究**：是否进行了网格敏感性分析（至少三组不同网格密度的对比）？是否给出了网格收敛曲线？关键区域（应力集中处）是否进行了局部加密？
6. **边界条件合理性**：约束和载荷施加方式是否有物理依据？是否讨论了简化假设对结果的影响？接触条件、对称条件的使用是否恰当？
7. **材料模型选择**：是否说明了选择线性/非线性材料模型的理由？非线性分析是否给出了本构关系（弹塑性、超弹性、蠕变等）的参数来源？
8. **结果验证**：有限元结果是否与解析解、经验公式或实验数据进行了对比？误差范围是否在可接受区间内（通常 < 10%）？如无法获取对比数据，是否至少做了网格收敛性验证？

### 实验设计规范

9. **试件制备标准**：试件尺寸、形状、加工精度是否符合相关试验标准（如 GB/T 228 拉伸试验标准试件）？取样位置和数量是否说明？
10. **设备与仪器说明**：试验设备型号、量程、精度等级是否完整列出？设备是否在检定有效期内？数据采集系统（传感器类型、采样频率、通道数）是否说明？
11. **加载方案**：加载速率、加载方式（单调/循环）、加载步序是否清楚？是否符合相关标准的要求？
12. **数据处理方法**：原始数据如何处理（滤波、平均、拟合）？数据处理方法是否有依据？异常数据的取舍标准是否说明？

### 工程制图规范

13. **CAD图纸合规性**：工程图纸是否符合 GB/T 标准（GB/T 4457 机械制图、GB/T 50001 建筑制图标准）？图幅、比例、线型是否规范？
14. **尺寸标注完整性**：关键尺寸是否标注？公差等级是否标注（IT等级）？配合关系是否标注（如 H7/g6）？表面粗糙度是否标注？
15. **装配图与零件图配套**：装配图是否包含零件序号、明细表？零件图是否有足够的视图表达结构？

### 施工/制造工艺审查

16. **工艺流程完整性**：工艺路线是否完整覆盖从毛坯到成品的各工序？关键工序（如热处理、焊接、混凝土浇筑）是否有详细参数？
17. **质量控制点**：是否识别了关键质量控制点？是否给出了检验方法和合格标准？
18. **安全措施**：施工安全方案或操作安全要求是否有说明？是否符合相关安全规程？

### 结构/力学分析审查

19. **受力分析规范性**：是否绘制了完整的受力图（自由体图）？力的方向、作用点、大小是否标注清楚？支座反力是否通过平衡方程求解并给出过程？
20. **失效准则选择**：所选失效准则（von Mises、Tresca、Mohr-Coulomb、Drucker-Prager等）是否适用于对应材料和受力状态？选择依据是否说明？

## 证据与论证标准补充

通用 analysis/fact 规则之外，本学科的证据标准：

### 可接受的证据类型

- **设计计算书**：完整的计算过程，包括公式来源、参数代入、单位换算、结果判断，每一步都需清晰可查
- **有限元分析结果**：应力/应变/位移云图必须配合量化数据（最大值、最小值、分布规律），不能只给彩色图片不给数值
- **实验数据**：需注明试验标准、设备型号和精度、试件数量和编号，单组试件不少于3个（保证统计意义）
- **标准/规范**：作为设计依据的 GB/T、JB/T、CJJ 等标准条文可直接作为论证依据，但须标注标准编号和版本年份
- **工程实测数据**：来自实际工程的监测数据，需注明监测方案、传感器布设方案和数据采集周期
- **经验公式与设计手册数据**：《机械设计手册》《混凝土结构设计原理》等权威手册中的公式和数据可以引用，须注明版本和页码

### 证据红线

- **计算过程缺失**：只给出最终结果不给推导过程——"经计算，轴的直径取 45mm"，没有载荷计算、弯矩图、校核过程，结论不可信
- **有限元云图无量化**：只贴应力云图的截图，没有读取最大应力值并与许用应力对比，仿真分析形同虚设
- **单组实验下结论**：仅做一次实验就给出结论，没有重复性验证，数据不具统计意义
- **标准引用错误**：引用了已废止的标准（如引用 GB 50010-2002 而非现行 GB 50010-2010），或标准适用范围与论文工况不匹配
- **单位不一致**：同一计算过程中混用不同单位制（如力用 kgf、应力用 MPa），导致量纲错误
- **安全系数凭空取值**：安全系数未引用标准或规范就直接赋值——"取安全系数为2"，无来源无依据

### 论证模式期望

- **设计计算型**：设计要求→方案选择→参数确定→计算校核→结果判断的工程设计闭环。每个参数必须有来源，每步校核必须有合格判据
- **有限元分析型**：物理问题→数学模型→离散化→求解→验证的仿真分析闭环。建模假设必须说明，结果必须验证
- **实验研究型**：问题→方案→实施→数据→分析→结论的实证研究范式。实验结果须与理论预测或已有文献数据进行对比讨论
- **工艺优化型**：问题识别→因素分析→方案设计→实验验证→效果评价的优化研究范式。优化前后须有量化对比
- **工程管理型**：项目分析→方案制定→实施模拟→效果评估的管理研究范式。方案必须基于理论方法（如关键路径法、挣值分析法），不能凭经验拍脑袋

## 常见问题模式

### S级（论文是否成立）

1. **设计无安全系数校核/强度校核**：
   - 表现：论文完成了某机构或结构的设计，给出了尺寸参数，但没有进行强度校核或安全系数验证，或仅写了一句"经校核满足要求"而无计算过程
   - 影响：设计成果的安全性无法保证，论文核心结论缺乏可靠性基础；在工程领域，未经校核的设计可能导致安全事故
   - 修改方向：补充完整的强度、刚度、稳定性校核计算过程，明确给出安全系数并与标准规定值对比，给出"满足/不满足"的明确结论

2. **有限元分析无网格收敛性研究**：
   - 表现：直接给出一组网格密度下的仿真结果，没有进行网格敏感性分析，无法判断结果是否已收敛
   - 影响：仿真结果的可信度无法保证，不同网格密度下可能得出完全不同的结论，论文的数值分析结论不成立
   - 修改方向：至少用三组不同网格密度（粗、中、细）进行计算，绘制关键指标（最大应力、最大位移等）随网格数量的变化曲线，证明结果已收敛

3. **结构设计违反强制性国标**：
   - 表现：设计参数或方案明显不符合现行强制性国家标准（如混凝土保护层厚度不满足 GB 50010 要求，钢结构连接不满足 GB 50017 规定）
   - 影响：设计方案违法违规，论文成果不可采纳；在工程实践中会造成安全隐患
   - 修改方向：逐条核对所有设计参数是否满足现行强制性标准的要求，对不满足项进行修正，并在论文中明确标注所依据的标准条文号

4. **载荷工况不完整或不切实际**：
   - 表现：仅考虑了静载荷而忽略了动载荷、交变载荷、温度载荷；或载荷取值远小于实际工况（如桥梁设计未考虑车辆冲击系数，起重机设计未考虑动载系数）
   - 影响：设计的安全裕度被严重高估，实际工况下可能发生失效；或反之载荷取值过于保守导致设计不经济
   - 修改方向：根据相关标准（如 GB 50009 建筑结构荷载规范、GB/T 3811 起重机设计规范）完整列出所有应考虑的载荷工况，给出载荷组合方式和载荷系数

### A级（严重影响质量）

1. **材料性能数据无来源**：
   - 表现：计算中使用的材料参数（屈服强度、弹性模量、许用应力等）没有标注来源，不知道是查手册、查标准还是假设值
   - 影响：计算结果的可靠性存疑，审稿者无法验证参数的正确性
   - 修改方向：所有材料参数必须标注来源（如"Q235B 的屈服强度 $\sigma_s = 235$ MPa，取自 GB/T 700-2006"），非标准材料需给出实验测定的力学性能报告

2. **有限元边界条件未论证**：
   - 表现：约束和载荷的施加方式没有物理解释，如将螺栓连接简化为刚性连接、将分布载荷简化为集中力，但未说明简化的理由和对结果的影响
   - 影响：边界条件不当会导致计算结果与实际严重偏离，仿真结果失去参考价值
   - 修改方向：对每个边界条件说明其物理对应关系和简化假设，讨论简化可能引入的误差方向和量级

3. **数值结果与解析结果未对比**：
   - 表现：有限元分析得到了结果，但没有用理论解析解、经验公式或实验数据进行交叉验证
   - 影响：无法确认有限元模型建立的正确性，仿真结果的可信度大打折扣
   - 修改方向：对于有解析解的简单工况，先用解析方法求解并与有限元结果对比，确认模型的准确性；如无解析解，需与文献中已验证的数值结果或实验数据对比

4. **实验无误差/不确定度分析**：
   - 表现：实验数据直接取平均值作为结果，没有给出标准差、变异系数或测量不确定度；没有讨论系统误差和随机误差的来源
   - 影响：实验结果的精度和可靠性无法评估，读者无法判断数据的置信水平
   - 修改方向：至少给出每组实验数据的均值和标准差，分析主要误差来源（设备精度、试件离散性、环境因素），对关键结果进行不确定度评定

5. **设计参数选取缺乏优化论证**：
   - 表现：关键设计参数（如齿轮模数、梁截面尺寸、钢筋直径）直接给出一个值，没有说明为什么选这个值而不是其他值，没有进行参数比选或优化分析
   - 影响：设计方案可能不是最优的或最经济的，评审会质疑参数选取的合理性
   - 修改方向：对关键参数进行至少两种方案的比选分析，或通过优化方法（如参数化分析、正交实验）给出参数的选取依据

6. **工艺路线缺少关键工序参数**：
   - 表现：工艺路线图列出了工序名称但缺少关键参数（如切削速度、进给量、焊接电流、混凝土浇筑温度），或参数没有计算/查表依据
   - 影响：工艺方案不可操作，无法用于指导实际生产或施工
   - 修改方向：对每道关键工序给出完整的工艺参数，并说明参数的确定方法（计算、查表、实验验证）

### B级（影响说服力）

1. **CAD图纸不符合 GB 制图标准**：
   - 表现：工程图纸存在线型混乱（可见轮廓线与隐藏线粗细不分）、尺寸标注不规范（箭头形式不一致、数字位置不对）、图幅和标题栏缺失或不标准
   - 影响：降低论文的工程专业性，暴露作者对工程制图基本规范的不熟悉
   - 修改方向：严格按照 GB/T 4457 系列标准绘制图纸，使用标准线型（粗实线/细实线/虚线/点划线）、标准标注样式和规范的标题栏

2. **计算过程中单位不一致**：
   - 表现：同一推导过程中混用 N 和 kN、mm 和 m、MPa 和 kgf/cm$^2$ 等不同单位，或公式代入时未统一单位制
   - 影响：容易导致数量级错误，严重时使计算结果差若干个数量级
   - 修改方向：全文统一使用国际单位制（SI），在每次公式代入时明确标注各量的单位，中间计算结果也标注单位

3. **力学分析缺少受力图**：
   - 表现：直接给出弯矩、剪力值或列出平衡方程，但没有绘制对应的自由体图（受力图），读者无法验证力的分析过程
   - 影响：力学分析过程不完整，审阅者无法确认力的施加是否正确，分析结论缺乏直观依据
   - 修改方向：对每个需要进行力学分析的构件，先绘制清晰的受力图，标注所有外力、支座反力和内力方向，然后再列方程求解

4. **BOM 或成本估算不完整**：
   - 表现：设计类论文给出了设计方案但没有物料清单（BOM），或虽有 BOM 但缺少关键零件的规格型号和单价估算
   - 影响：设计方案的可行性和经济性无法评估
   - 修改方向：补充完整的 BOM 表（名称、规格型号、数量、材料、单价、合计），对于非标件给出加工成本估算依据

5. **参数化分析图表不规范**：
   - 表现：参数影响分析的图表缺少坐标轴标签和单位、曲线无图例、数据点过少无法反映趋势、多组数据的曲线难以区分
   - 影响：分析结果的可读性和说服力降低
   - 修改方向：所有图表必须有中文标题，坐标轴标注参数名称和 SI 单位，多条曲线使用不同线型（实线、虚线、点划线）和标记符号，数据点不少于5个以体现变化趋势

6. **公差与配合标注缺失**：
   - 表现：零件图上关键配合面没有标注公差等级和配合类型，或仅标注了基本尺寸而无公差带代号
   - 影响：设计图纸不完整，无法指导加工制造
   - 修改方向：对所有配合面标注公差带代号（如 $\phi 50H7/g6$），对关键形位公差（平行度、垂直度、同轴度等）进行标注

## 文献综述特殊要求

### 文献来源优先级

1. **核心学术期刊**：
   - 机械类：机械工程学报、中国机械工程、摩擦学学报、振动与冲击
   - 土木类：土木工程学报、建筑结构学报、岩土工程学报、工程力学
   - 国际期刊：ASME Journal of Mechanical Design、ASCE Journal of Structural Engineering、International Journal of Mechanical Sciences、Finite Elements in Analysis and Design

2. **国家标准与行业标准**：GB/T、GB（强制性国标）、JB/T（机械行业标准）、JG/T（建筑行业标准）、CJJ（城镇建设行业标准）——这些在本学科中是必须引用的核心文献

3. **权威设计手册**：《机械设计手册》（成大先主编）、《混凝土结构设计原理》、《钢结构设计原理》、《机械设计课程设计》等——可作为设计参数和经验公式的来源

4. **学位论文**：来自相关高校的硕博论文可以引用，但不能作为唯一的理论依据

5. **工程案例与技术报告**：来自设计院、研究院的技术报告可作为工程实践参考

### 文献数量底线

- 设计计算型：至少15篇参考文献，其中学术论文不少于8篇，国标/行标不少于3项
- 有限元分析型：至少20篇参考文献，其中近5年文献占比不低于30%，须包含仿真方法类文献
- 实验研究型：至少20篇参考文献，须包含试验标准和同类实验的文献
- 工程管理型：至少15篇参考文献，须包含管理理论文献和工程案例

### 综述组织方式

- **禁止**：逐篇罗列式综述（"张三[1]设计了…，李四[2]分析了…"），这不是综述
- **要求**：按技术路线（如不同设计方法、不同分析手段）或按发展阶段（理论解析→数值仿真→实验研究）组织，形成对比分析框架
- **必须有**：综述末尾明确说明现有研究的不足（如现有设计偏保守、某工况下的研究较少、某类材料的实验数据缺乏），从而引出本文的研究切入点

## 术语与规范

### 术语使用规范

- **材料牌号**：必须使用国标规定的材料牌号表示法（如 Q235B、HT200、40Cr、C30混凝土），首次出现时说明材料类别和主要性能指标
- **标准编号格式**：严格按照"标准代号 标准编号-年份"格式引用（如 GB/T 700-2006《碳素结构钢》、GB 50010-2010《混凝土结构设计规范》），引用时须核实标准是否为现行有效版本
- **SI单位强制**：全文必须使用国际单位制，力用 N（kN），应力用 MPa（GPa），长度用 mm（m），质量用 kg，角度用 rad（度数须标注 $\degree$ 符号）
- **公差标注**：使用标准公差带代号（如 H7、g6），配合用分数形式表示（如 H7/g6），粗糙度用 Ra 值标注
- **避免口语化**：不使用"拧螺丝"、"开模子"等口语表达，使用"紧固螺栓连接"、"模具开模加工"等规范表述

### 引用格式

- 遵循 GB/T 7714-2015 标准
- 标准类引用格式：标准编号, 标准名称[S]. 发布地: 发布机构, 发布年份
- 设计手册类引用需注明版本（第几版）、出版年份和引用页码
- 软件引用需注明软件名称、版本号和开发机构（如"ANSYS Workbench 2021 R1, ANSYS Inc."）

### 图表规范

- **工程图纸**：必须符合 GB/T 4457 系列标准，包含标题栏、比例、图幅、线型规范
- **有限元图**：网格图须说明单元类型和总数量，应力/位移云图须包含色标（含最大最小值）、坐标系方向、载荷和约束标记
- **实验曲线**：必须有坐标轴标签（物理量名称+SI单位）、图例、数据点标记；多组数据需用不同线型和标记区分
- **受力图**：使用标准力学符号（集中力箭头、分布载荷梯形、固定端/铰支/滚动支座符号），所有力标注大小和方向
- **流程图**：工艺流程图使用标准工序符号，施工网络图使用双代号或单代号表示法

### 公式规范

- 所有公式必须编号，便于后文引用
- 公式中每个符号首次出现时必须定义（名称+单位），建议在公式下方用"式中："逐一解释
- 计算过程须展示完整的数值代入步骤，不能跳过中间结果
- 单位须从公式代入到最终结果全程携带，确保量纲一致

## 结论与建议特殊要求

### 结论章节要求

- **成果量化**：设计类论文须总结关键设计参数和校核结果（如"轴的最小安全系数为1.8，满足 $\geq 1.5$ 的要求"）；有限元类须总结关键分析结论（如"最大应力出现在某位置，数值为某 MPa，低于材料许用应力某 MPa"）；实验类须总结主要实验发现和数据
- **禁止空泛总结**："本文完成了某某机构的设计，具有一定工程应用价值"——这种结论毫无信息量，必须具体到设计的关键成果和关键数据
- **设计合理性声明**：设计类论文结论中必须明确声明所有校核是否通过，有无不满足要求的工况；如有不满足项，须在结论中明确说明并给出处理方案
- **局限性坦诚**：必须讨论设计方案的适用范围、分析方法的假设局限、实验条件的约束等

### 建议/展望部分的边界

- **合理展望**：基于本文工作的不足提出改进方向，如"未来可增加疲劳寿命分析""可通过拓扑优化进一步减重""可增加更多工况的实验验证"
- **过度展望红线**：展望内容与本文工作无关（如论文做的是减速器设计，展望里写"探索人工智能在机械设计中的应用"但全文未涉及）；展望过于宏大（"为我国装备制造业发展做出贡献"）
- **合理粒度**：展望应具体到可操作的下一步研究/设计工作，而非宏观行业展望

## 学科专属审查维度

### 设计安全性审查

本学科论文涉及工程安全，须对设计安全性进行专项审查：

- **强度校核覆盖率**：是否对所有承载零件/构件都进行了强度校核？是否有遗漏的危险截面？
- **载荷组合完整性**：是否考虑了所有可能的载荷组合（包括最不利组合）？载荷分项系数是否正确取值？
- **失效模式覆盖**：是否考虑了所有可能的失效模式（强度失效、刚度失效、稳定性失效、疲劳失效、磨损失效等）？
- **安全系数合理性**：安全系数是否在标准规定范围内？是否存在安全系数取值过高（过于保守、不经济）或过低（安全隐患）的情况？

### 有限元模型质量评估

| 检查项 | 合格标准 | 常见问题 |
|---|---|---|
| 网格质量 | Jacobian > 0.3，纵横比 < 5:1，翘曲度 < 30° | 应力集中处网格过粗、畸变单元 |
| 单元类型选择 | 与结构特征匹配（壳单元/实体单元/梁单元） | 薄壁结构用实体单元导致计算量过大且精度低 |
| 边界条件 | 有物理对应关系，约束数量满足消除刚体位移 | 过约束（约束过多导致假应力）或欠约束（刚体位移） |
| 载荷施加 | 等效载荷与实际载荷等效，施加面/点正确 | 集中力施加在节点上导致应力奇异 |
| 结果合理性 | 位移量级合理，应力分布符合力学直觉 | 应力奇异点的极大值被当作真实最大应力 |

### "设计计算类"论文完整性评估框架

用于判断一篇设计计算类论文是否工作量充分：

**深度不足的信号**（多项命中则工作量存疑）：
- 所有计算仅套用手册公式，无任何推导过程
- 只做了静强度校核，未考虑疲劳、刚度、稳定性
- 全文没有任何有限元分析或实验验证手段
- 工程图纸仅有示意图级别的草图，无完整的零件图和装配图
- 参数选取全部"参考文献[X]取值"，无自主计算或优化
- 设计方案只有一种，未进行方案比选

**深度充足的信号**（至少命中2-3项）：
- 包含完整的载荷分析和多种失效模式的校核
- 关键部位通过有限元分析进行了验证，并与手算结果对比
- 进行了参数化分析或优化设计（如参数对性能的影响规律曲线）
- 工程图纸完整规范，包含装配图和主要零件图
- 有制造工艺分析或施工方案设计
- 讨论了设计的经济性（BOM/成本估算）
