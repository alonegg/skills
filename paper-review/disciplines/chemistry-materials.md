# 化学化工类 学科配置

> 本文件是 paper-review 技能的学科叠加层。它不替换通用规则，而是在通用规则基础上增加化学化工类学科特有的审查维度、调整权重、补充常见问题模式。

## 适用范围

- **覆盖学科**：化学、应用化学、化学工程与工艺、材料化学、高分子材料与工程、新能源材料与器件、制药工程
- **典型论文类型**：
  - **合成表征型**：设计合成路线，制备目标化合物或材料，通过多种表征手段确认结构与性能（如有机合成、无机纳米材料制备、高分子合成）
  - **工艺设计型**：围绕化工生产过程进行工艺流程设计，包含物料衡算、能量衡算、设备选型和工艺优化（如年产万吨某产品的车间工艺设计）
  - **性能测试型**：以材料性能评价为核心，系统测试力学、光学、电化学、催化等性能并与文献值对比（如电池材料的充放电性能、催化剂活性评价）
  - **模拟计算型**：使用分子动力学、密度泛函理论（DFT）、化工流程模拟（Aspen Plus）、有限元等计算手段研究化学体系（如反应机理计算、流场模拟）
  - **综述分析型**：系统梳理某一化学/材料领域的研究进展，提出研究趋势和展望
- **识别线索**：论文中出现化学结构式、反应方程式、表征谱图（XRD/FTIR/NMR/MS/SEM/TEM/TGA/DSC）、合成路线图、化工流程图（PFD/PID）、物料衡算表、电化学曲线（CV/EIS/充放电曲线）等

## 结构期望

### 标准章节结构

本学科论文结构因论文类型而异，审查时应先判断论文属于哪种类型，再对照对应结构：

**合成表征型**（最常见）：
```
第1章 绪论（研究背景、国内外研究现状、研究目的与意义）
第2章 实验部分（试剂与仪器、合成路线与实验步骤、表征方法）
第3章 结果与讨论（结构表征、性能测试、机理分析）
第4章 结论与展望
```

**工艺设计型**：
```
第1章 绪论（产品概述、工艺路线比选、设计任务与依据）
第2章 工艺流程设计（工艺原理、流程描述、PFD）
第3章 物料衡算与能量衡算
第4章 主要设备设计与选型（反应器、换热器、塔设备等）
第5章 车间布置与管道设计（PID、设备布置图）
第6章 安全与环保
第7章 技术经济分析
第8章 总结
```

**性能测试型**：
```
第1章 绪论（研究背景、文献综述、研究目的）
第2章 实验部分（材料制备、测试设备与方法）
第3章 结构表征（物相、形貌、成分分析）
第4章 性能测试与分析（核心性能指标的系统测试与讨论）
第5章 结论与展望
```

**模拟计算型**：
```
第1章 绪论（研究背景与意义、国内外研究现状）
第2章 理论基础与计算方法（理论框架、计算软件与参数设置、模型验证）
第3章 计算结果与分析
第4章 与实验数据的对比讨论（若有）
第5章 结论与展望
```

### 必须有的章节

- **绪论**：必须包含明确的研究问题和研究目的，综述现有研究的不足并引出本文切入点；不能只有宏观的"化学工业重要性"描述
- **实验部分/方法部分**：合成表征型和性能测试型必须有独立的实验章节，详细列出试剂（纯度、厂家）、仪器（型号、测试条件）、实验步骤（可重复的精度）
- **结果与讨论**：必须将数据呈现与分析解释相结合，不能只列数据不讨论，也不能只讨论不给数据
- **结论**：必须基于实际实验/计算结果给出明确结论，不能照搬摘要

### 可选/特殊章节

- **安全与环保**：涉及危险化学品合成或化工工艺设计时必须包含，其他类型论文可作为节或附录
- **技术经济分析**：工艺设计型论文必须包含，其他类型可选
- **补充表征数据**：大量谱图可作为附录呈现，正文中放关键谱图并在附录中放完整数据

### 结构红线

- **无独立实验部分**：合成表征型或性能测试型论文没有独立的实验章节，实验步骤散落在结果讨论中或完全缺失——无法重复的研究不成立，S级问题
- **结果与讨论完全分离且讨论空洞**：结果章节只罗列数据和图表，讨论章节只有空泛的"表明性能良好"——数据不分析等于没有数据
- **工艺设计无衡算**：工艺设计型论文没有物料衡算和能量衡算——设计缺乏定量基础，S级问题
- **结论照搬摘要**：结论与摘要文字重复率超过70%，没有基于具体实验数据的增量总结

## 专项规则权重调整

基于通用6大专项规则，调整本学科的审查权重：

| 专项规则 | 权重调整 | 理由 |
|---|---|---|
| intro | 0 | 化学化工类绪论通常结构明确，无特殊调整 |
| literature | +1 | 须检查是否综述了该体系的合成方法演进、表征手段发展，以及是否追踪到最新研究进展 |
| method | +2 | 实验方法的可重复性是化学研究的核心要求，合成步骤、表征条件、仪器参数必须重点审查 |
| analysis | +2 | 谱图解析的正确性、数据的内部一致性、性能对比的公平性是质量关键区分度 |
| conclusion | +1 | 化学类论文结论必须有明确的实验发现总结，不能空泛 |
| fact | +1 | 化学反应方程式的配平、物性数据的准确性、试剂信息的完整性直接影响实验可重复性 |

## 方法审查补充维度

通用 method 规则之外，本学科需额外检查：

### 实验设计规范

1. **变量控制**：实验设计是否遵循单一变量原则？对照实验（空白对照、阳性/阴性对照）是否设置？平行实验是否设置且数量不少于3组？无对照实验的结论不能归因于所声称的变量
2. **重复性保证**：每组实验是否至少重复3次？关键实验数据是否给出了误差棒（error bar）或标准偏差（SD）？单次实验的数据不能作为确定性结论的依据
3. **随机化与盲法**：涉及性能对比实验时，测试顺序是否随机化？仪器测试是否有操作者偏差的控制措施（如编号盲测）？

### 合成路线合理性

4. **路线选择论证**：为何选择该合成路线而非其他已报道路线？是否讨论了路线的原子经济性、步骤数、总收率等指标？
5. **试剂信息完整性**：所有试剂是否标注了纯度等级（如AR、CP、GR、HPLC级）和生产厂家？溶剂是否注明了是否经过纯化处理（如无水处理、除氧处理）？
6. **反应条件完整性**：每步反应是否给出了温度、时间、气氛（氮气/氩气保护或空气气氛）、溶剂、催化剂用量、反应物摩尔比？缺少关键条件的合成步骤无法重复
7. **产率计算**：产率（yield）是否给出且计算方法是否说明（分离产率 vs. 粗产率）？产率是否合理（过高需解释纯度，过低需讨论原因）？

### 表征手段完整性

8. **表征与声称的匹配性**：所使用的表征手段是否足以支撑论文的结构/性能声称？基本原则——
   - 声称合成了新化合物：必须有结构确认（NMR/MS/元素分析/单晶衍射中至少两种以上）
   - 声称制备了特定晶相材料：必须有 XRD 数据并与标准卡片（PDF卡/JCPDS）对比
   - 声称材料具有特定形貌：必须有 SEM 或 TEM 图像，且分辨率足以支撑结论
   - 声称材料含有特定官能团：必须有 FTIR 或 Raman 光谱并指认特征峰
   - 声称热稳定性：必须有 TGA/DSC 数据
9. **表征条件记录**：每种表征手段是否给出了仪器型号和关键测试参数（如 XRD 的扫描范围、步长、辐射源；NMR 的溶剂和频率；SEM 的加速电压；TGA 的升温速率和气氛）？

### 数据处理规范

10. **误差棒与统计处理**：所有定量数据是否标注了误差棒（error bar）？误差棒代表的含义（SD、SEM、95% CI）是否说明？是否对组间差异进行了统计检验（t检验、ANOVA）？
11. **有效数字**：数据的有效数字位数是否与仪器精度一致？如天平精度为0.0001 g，质量数据不应报告到0.00001 g；XRD 衍射角保留2位小数即可
12. **曲线拟合**：涉及曲线拟合时（如动力学拟合、等温吸附拟合），是否给出了拟合方程和相关系数 R^2？是否讨论了拟合模型选择的依据？

### 安全与环保

13. **危险化学品标识**：使用的危险化学品（易燃、剧毒、强腐蚀性、爆炸性）是否在实验部分有安全提示？涉及HF、叠氮化物、有机过氧化物等高危试剂时是否说明了防护措施？
14. **废液废物处理**：是否说明了实验废液和废物的收集与处置方式？涉及重金属离子（Cr(VI)、Hg、Cd等）或有机溶剂废液时是否注明了分类收集？

### 工艺流程设计（化学工程类）

15. **物料衡算**：是否以反应方程式为基础，逐步计算各物料的质量流量？原料转化率、选择性、收率是否明确？衡算是否自洽（进料=出料+损耗）？
16. **能量衡算**：是否计算了各单元操作（反应、精馏、换热等）的热负荷？公用工程消耗（蒸汽、冷却水、电力）是否列出？
17. **设备选型**：关键设备（反应器、精馏塔、换热器、泵等）是否给出了设计计算过程（如塔径、塔板数、换热面积）？选型是否依据计算结果而非凭空指定？
18. **PFD/PID图**：是否绘制了规范的工艺流程图（PFD）？涉及详细设计时是否有管道仪表流程图（PID）？图中的物料走向、控制点、仪表是否与正文描述一致？

### 模拟计算规范

19. **计算方法与软件**：是否明确说明了使用的理论方法/力场/基组（如 DFT 的泛函和基组选择：B3LYP/6-311G(d,p)）和计算软件版本？方法选择是否有文献依据？
20. **模型验证**：计算模型是否通过与已知实验数据的对比进行了验证（如计算的键长/键角与晶体学数据对比、计算的光谱与实验光谱对比）？

## 证据与论证标准补充

通用 analysis/fact 规则之外，本学科的证据标准：

### 可接受的证据类型

- **原始谱图数据**：XRD衍射图、FTIR/Raman光谱、NMR谱图、质谱（MS）、紫外-可见吸收光谱等必须以原图呈现（含坐标轴和标注），不能仅列出峰位数据表
- **电镜图像**：SEM/TEM图像必须标注比例尺（scale bar），同一论文中比例尺风格应统一；高倍图和低倍图配合使用以展示整体形貌和局部细节
- **电化学数据**：循环伏安曲线（CV）、电化学阻抗谱（EIS）、充放电曲线等须标注测试条件（扫描速率、电解液、电极面积、电位范围）
- **热分析数据**：TGA/DSC曲线须标注升温速率和气氛；质量损失百分比须与理论值对比
- **物料衡算表**：工艺设计类论文中物料衡算表作为核心定量证据，必须逐项列出各物流组分的质量流量
- **标准对比**：性能数据必须与文献报道值或国家标准值进行对比，不能仅声称"性能优异"而无参照基准

### 证据红线

- **仅列峰位不给谱图**：声称通过 FTIR 确认了官能团，但只写"在 3400 cm^-1 处有吸收峰"而不给出完整谱图——审阅者无法判断峰的形状、强度和基线质量
- **表征数据内部矛盾**：XRD 显示为非晶态但 FTIR 解析了晶体特征峰；TGA 显示有机组分含量 20% 但元素分析给出 40%——内部矛盾说明数据不可靠或解析有误
- **无对照的性能声称**：声称"催化活性优异"但没有与文献已知催化剂或商业催化剂进行对比；声称"吸附容量高"但没有对比已有吸附剂——无参照的"优异"不构成有效结论
- **新化合物表征不完整**：合成了文献未报道的新化合物，但只做了一两种表征（如只有 FTIR）而缺少确证结构的关键手段（NMR、MS、元素分析或单晶衍射）
- **他人数据冒充自有数据**：论文中的谱图/图像与已发表文献完全一致（分辨率、噪声模式、基线特征均相同），涉嫌直接复制他人数据

### 论证模式期望

- **合成表征型**：设计意图→合成路线→实验操作→表征确认→性能评价的完整研究闭环。每步合成必须有表征数据支撑，最终产物必须有完整的结构确认
- **工艺设计型**：工艺路线选择→流程设计→物料/能量衡算→设备设计→安全环保→技术经济评价的工程设计闭环。每个设计参数必须有计算依据
- **性能测试型**：材料制备→结构表征→性能测试→性能分析→与文献对比的评价闭环。测试结果必须与文献值进行定量对比并讨论差异原因
- **模拟计算型**：理论方法选择→模型构建→计算执行→结果分析→实验验证/文献对比的计算研究闭环。计算方法须经验证方可信赖

## 常见问题模式

### S级（论文是否成立）

1. **新化合物缺少关键表征**：
   - 表现：论文声称合成了新化合物，但只提供了1-2种表征数据（如仅有FTIR），缺少能够确证分子结构的关键手段（NMR、MS、元素分析、单晶衍射），无法排除产物为副产物或混合物的可能
   - 影响：论文的核心结论——"成功合成了目标化合物"——不成立；后续所有基于该化合物的性能讨论均失去基础
   - 修改方向：补充至少两种互补的结构确认手段；有机化合物须提供 ^1H NMR、^13C NMR 和高分辨 MS；无机/纳米材料须提供 XRD（与标准卡片对比）、TEM（晶格条纹）和元素分析（EDS或XPS）

2. **无对照实验导致因果归因不成立**：
   - 表现：论文声称"A因素提升了B性能"，但实验中未设置控制变量——没有空白对照、没有改变单一变量的对比实验，所有变量同时改变
   - 影响：无法判断性能变化究竟归因于所声称的变量还是其他因素，核心结论缺乏因果支撑
   - 修改方向：补充控制实验——至少设置空白对照和单因素变量对照；对于多因素影响的体系，采用正交实验或单因素逐步优化法

3. **数据造假或操控嫌疑**：
   - 表现：不同样品的谱图高度相似（噪声模式、基线漂移完全一致）；数据曲线异常光滑且无任何噪声或波动；图片存在明显的复制粘贴痕迹（如SEM图像中重复的颗粒排列）
   - 影响：数据真实性存疑，论文的全部结论不可信；涉及学术诚信问题
   - 修改方向：要求提供原始仪器数据文件（raw data）进行核查；重新进行实验并提供原始、未经处理的数据

4. **工艺设计无物料/能量衡算**：
   - 表现：工艺设计型论文给出了流程图和设备清单，但没有基于化学计量关系的物料衡算，也没有各单元操作的能量衡算；设备参数凭经验指定而非计算确定
   - 影响：工艺设计缺乏定量基础，设备选型无依据，设计方案不可实施
   - 修改方向：以主反应方程式为起点，逐步完成全流程物料衡算表；对关键放热/吸热操作进行能量衡算；设备参数须基于衡算结果进行设计计算

### A级（严重影响质量）

1. **缺少误差分析/无平行实验**：
   - 表现：所有定量数据（产率、吸附容量、催化转化率、电化学性能等）均为单次测量值，无重复实验，无误差棒，无标准偏差
   - 影响：数据的可靠性和可重复性无法评估，读者无法判断实验结果的置信水平
   - 修改方向：关键实验至少重复3次，给出均值±标准偏差（mean ± SD）；所有定量图表添加误差棒并注明其含义（SD或SEM）

2. **表征手段不足以支撑声称**：
   - 表现：声称制备了某种特定结构的材料但表征不充分——如声称合成了核壳结构但无TEM截面图或元素线扫描；声称掺杂成功但无XPS确认化学状态
   - 影响：结构声称缺乏直接证据支撑，结论的可信度大打折扣
   - 修改方向：针对每一项结构声称，列出必要的表征证据并补充缺失的数据

3. **性能数据无文献对比**：
   - 表现：论文给出了材料的性能数据（如比电容、催化转化率、吸附容量等），但没有与同类材料的文献报道值进行对比，仅用"性能良好""效果显著"等定性描述
   - 影响：无法判断论文成果在该领域中的水平和贡献度；评审无法评估工作的学术价值
   - 修改方向：补充与至少3-5篇同类工作的定量对比表格，列出各项性能指标的具体数值，讨论本文结果优于或不及已有工作的原因

4. **反应条件描述不完整**：
   - 表现：合成步骤中缺少关键反应条件——如"将混合物加热反应"但未标注温度和时间；"在惰性气氛下"但未说明是N_2还是Ar；溶剂用量未标注
   - 影响：实验无法被他人重复，违反化学研究的基本可重复性原则
   - 修改方向：逐步核查每个合成步骤，确保温度（±精度）、时间、气氛类型、溶剂名称及用量、试剂摩尔比、催化剂用量等信息完整

5. **产率计算不明确或不合理**：
   - 表现：产率数值过高（接近100%且未经过纯化）或过低（但未讨论原因）；未区分粗产率与分离纯化后产率；产率计算方法未说明
   - 影响：产率是评价合成路线优劣的核心指标，不准确的产率数据影响路线评价和可重复性
   - 修改方向：明确标注产率类型（分离产率/粗产率），给出计算公式；异常产率须讨论可能原因

### B级（影响说服力）

1. **谱图/图像质量差**：
   - 表现：XRD衍射图基线漂移严重、噪声大、峰分辨率不足以进行物相鉴定；SEM/TEM图像模糊、对比度低或放大倍数不恰当；NMR谱图中溶剂峰未标注或基线未校正
   - 影响：降低数据的可读性和说服力，评审可能质疑实验操作的规范性
   - 修改方向：重新采集数据或对原始数据进行规范化处理（基线校正、平滑去噪），确保图像分辨率满足期刊投稿要求（通常 300 dpi 以上）

2. **有效数字不一致**：
   - 表现：同一类数据的有效数字位数不统一——如产率有的报 87.3%、有的报 92.15678%；XRD衍射角有的保留1位小数、有的保留4位
   - 影响：暴露数据处理不严谨，降低论文专业性
   - 修改方向：根据测量仪器精度统一有效数字位数——天平0.0001 g精度报4位小数，产率保留1位小数（如87.3%），XRD衍射角保留2位小数

3. **化学结构式绘制不规范**：
   - 表现：化学结构式键角不标准（如苯环画成不规则六边形）、立体化学标记缺失（手性中心未标记R/S构型）、化合物编号与正文不对应
   - 影响：影响结构信息的准确传达，可能导致读者对结构产生误解
   - 修改方向：使用 ChemDraw 等专业软件绘制结构式，遵循 ACS Document 1996 或 IUPAC 推荐的绘制规范；手性中心必须用楔形键/虚线键标注

4. **常见试剂缺少CAS号**：
   - 表现：使用了名称可能产生歧义的化学试剂但未标注CAS登录号（如"PEG"未说明分子量，"硅烷偶联剂"未说明具体品种）
   - 影响：试剂信息不明确导致实验不可重复
   - 修改方向：对可能产生歧义的试剂标注CAS号；通用试剂（如乙醇、盐酸）可不标注，但非常用试剂和商品名试剂必须标注

5. **引用"标准方法"无出处**：
   - 表现：实验部分写"按标准方法操作""参照常规步骤"但未给出具体的标准编号或文献引用
   - 影响：读者无法获知具体操作细节，实验可重复性受损
   - 修改方向：将"标准方法"替换为具体引用——如"按GB/T 6682-2008纯化水标准"或"参照文献[X]中报道的方法"

6. **坐标轴标注不完整**：
   - 表现：图表缺少坐标轴标签、缺少单位、图例缺失或不清晰；多条曲线使用相同线型和颜色无法区分
   - 影响：数据可读性降低，影响论文的专业呈现
   - 修改方向：所有图表的坐标轴必须标注物理量名称和单位（格式：物理量/单位，如"波数/cm^-1"）；多条曲线使用不同颜色和线型并配图例

7. **化学方程式未配平或格式不规范**：
   - 表现：反应方程式未配平、反应条件未标注（温度/催化剂/溶剂）、物质状态未标注（s/l/g/aq）
   - 影响：影响论文的化学规范性
   - 修改方向：所有反应方程式必须配平，标注反应条件和物态符号；使用化学方程式排版工具确保格式统一

## 文献综述特殊要求

### 文献来源优先级

1. **国际顶级期刊**：
   - 综合化学类：JACS (Journal of the American Chemical Society)、Angew. Chem. Int. Ed.、Chemical Science、Chemical Communications
   - 材料类：Advanced Materials、ACS Nano、Nano Letters、Chemistry of Materials
   - 高分子类：Macromolecules、Polymer、Polymer Chemistry
   - 化工类：Chemical Engineering Journal、Chemical Engineering Science、AIChE Journal、Industrial & Engineering Chemistry Research
   - 催化类：ACS Catalysis、Journal of Catalysis、Applied Catalysis B
   - 电化学/能源类：Energy & Environmental Science、Electrochimica Acta、Journal of Power Sources

2. **国内核心期刊**：
   - 化学学报、高等学校化学学报、无机化学学报、有机化学、分析化学
   - 化工学报、化工进展、高分子学报、高分子材料科学与工程
   - 催化学报、物理化学学报

3. **标准与规范**：GB/T（国家标准）、HG/T（化工行业标准）、YS/T（有色金属行业标准）、国家药典（制药工程类）

4. **权威工具书**：《有机化学》（邢其毅）、《无机化学》（武汉大学）、《化工原理》（陈敏恒/柴诚敬）、《高分子化学》（潘祖仁）——可引用基础理论，但不能作为研究进展的来源

### 文献数量底线

- 合成表征型：至少25篇参考文献，其中近5年文献占比不低于40%，英文文献占比不低于50%
- 工艺设计型：至少20篇参考文献，其中须包含工艺路线对比文献和相关标准
- 性能测试型：至少25篇参考文献，须包含同类材料的性能对比文献
- 模拟计算型：至少20篇参考文献，须包含计算方法验证的文献
- 综述分析型：至少50篇参考文献，近5年文献占比不低于50%

### 综述组织方式

- **禁止**：逐篇罗列式综述（"张三[1]合成了…，李四[2]制备了…"），这不是综述
- **要求**：按研究思路或技术路线组织——如按合成方法分类（溶剂热法、共沉淀法、溶胶-凝胶法）对比优缺点，或按材料体系演进（第一代→第二代→第三代材料）梳理发展脉络
- **必须有**：综述末尾明确指出现有研究的不足或空白（如"现有方法合成温度高、产率低""该材料在某应用场景下的研究较少"），自然引出本文的研究切入点
- **表格辅助**：推荐使用对比表格总结同类工作的合成条件和性能数据，有助于定位本文工作的创新点

## 术语与规范

### 术语使用规范

- **IUPAC命名**：化合物命名必须符合IUPAC命名法；首次出现时给出系统名和常用名（如有），后续可使用缩写但须在首次出现时注明全称
- **化学式格式**：分子式中的下标必须正确排版（H_2O 而非 H2O）；离子电荷标注在右上角（Fe^3+ 而非 Fe3+）；同位素标记在左上角（^13C 而非 13C）
- **单位规范**：遵循SI制，但允许化学领域常用约定——浓度用 mol/L（正式写法）或 mol L^-1，不推荐使用"M"（仅在非正式场合使用）；温度用 ℃（实验条件）或 K（热力学数据）；压力用 Pa（或 kPa、MPa）
- **命名一致性**：同一化合物在全文中的名称必须一致；不能一处写"纳米二氧化钛"另一处写"TiO_2纳米颗粒"再一处写"钛白粉"——选定一种表述后统一使用，首次出现时注明等价名称

### 引用格式

- 遵循 GB/T 7714-2015 标准
- 期刊论文须完整标注作者、题名、刊名、年份、卷号、页码或DOI
- 专利引用须标注专利类型、专利号、公开日期
- 标准类引用格式：标准编号, 标准名称[S]. 发布地: 发布机构, 发布年份
- 数据库引用（如JCPDS卡片号、CAS号）须在首次使用时给出完整信息

### 图表规范

- **谱图**：所有谱图必须有清晰的坐标轴标注（物理量名称/单位），XRD须标注2θ角度和衍射强度，FTIR须标注波数和透过率/吸光度
- **电镜图像**：必须包含比例尺（scale bar），标注放大倍数或比例尺数值；插图（inset）的比例尺独立标注
- **性能曲线**：电化学曲线须标注扫描速率、电解液成分、参比电极类型；催化曲线须标注反应温度和空速
- **表格**：数据表格须在表头标注单位，同一列数据的有效数字位数一致；对比表格须注明文献来源

### 化学安全信息标注

- 使用剧毒物质（氰化物、砷化物、汞化合物等）时须在实验部分明确标注安全警示
- 使用高压反应（水热/溶剂热反应）时须注明安全操作规程

## 结论与建议特殊要求

### 结论章节要求

- **成果量化**：合成型论文须总结产物的关键表征结果（如"XRD确认产物为纯相某某，晶粒尺寸约15 nm"）和关键性能数据；工艺设计型须总结主要设计参数和经济指标
- **禁止空泛总结**："成功制备了某材料并进行了表征"——这种结论毫无信息量，必须具体到获得了什么结构、什么性能、达到什么水平
- **与文献的定位**：结论中应简要说明本文成果在该领域中的位置——优于/持平/劣于现有报道的哪些指标，以及可能的原因
- **局限性坦诚**：必须讨论实验的局限性——如"本文仅在实验室规模验证，放大生产可能面临散热问题""计算采用了某简化假设，可能导致某方面的偏差"

### 建议/展望部分的边界

- **合理展望**：基于本文实验发现提出下一步方向——如"未来可通过掺杂优化性能""可在不同底物上验证催化剂的普适性""放大实验以验证工业化可行性"
- **过度展望红线**：展望与本文工作无关（如论文做的是有机合成，展望里写"推动绿色化学产业发展"但全文未涉及绿色化学）；展望过于宏大（"为我国新材料强国战略做出贡献"）
- **合理粒度**：展望应具体到可操作的实验方案或研究方向，而非宏观产业愿景

## 学科专属审查维度

### 合成实验完整性评估框架

用于判断一篇合成表征型论文的实验工作是否充分：

**深度不足的信号**（多项命中则工作量存疑）：
- 仅合成了一个样品，无任何条件优化或系列实验
- 表征手段少于3种，且缺少确证结构的关键手段
- 所有性能数据为单次测量，无重复实验
- 无对照实验，无法排除其他因素的影响
- 与文献方法完全相同，无任何改进或创新
- 讨论部分仅描述现象，无机理分析或文献对比

**深度充足的信号**（至少命中2-3项）：
- 进行了系统的条件优化（如温度、时间、配比等系列实验）
- 表征手段覆盖结构-形貌-性能三个层面
- 关键实验有平行实验和误差分析
- 设置了合理的对照实验
- 与文献同类工作进行了定量对比讨论
- 对实验现象提出了合理的机理解释并有数据支撑

### 工艺设计完整性评估框架

用于判断一篇工艺设计型论文的设计工作是否充分：

**深度不足的信号**：
- 仅有工艺流程示意图，无详细的PFD/PID
- 物料衡算仅计算了主反应，未考虑副反应和循环物料
- 设备选型仅列出型号，无设计计算过程
- 无安全与环保章节
- 无技术经济分析

**深度充足的信号**：
- 工艺路线经过至少两种方案比选
- 物料衡算和能量衡算完整且自洽
- 关键设备有详细的设计计算（如精馏塔的理论板数计算、换热器的传热面积计算）
- 绘制了规范的PFD和PID图
- 包含安全分析（HAZOP或安全检查表）和环保措施
- 技术经济分析包含投资估算和成本核算

### 表征数据一致性交叉检验

审查时须检验不同表征手段的数据是否内部一致：

| 检查组合 | 一致性要求 | 常见矛盾 |
|---|---|---|
| XRD + FTIR | XRD显示的晶相应与FTIR的特征峰吻合 | XRD显示为某氧化物但FTIR出现碳酸盐峰（样品不纯） |
| XRD + TEM | XRD的晶粒尺寸（Scherrer公式）应与TEM观察的粒径量级一致 | Scherrer计算5 nm但TEM显示50 nm（可能为多晶或团聚） |
| TGA + 元素分析 | TGA的质量损失应与元素分析给出的有机/无机比例一致 | TGA显示有机含量20%但元素分析计算出40%（数据矛盾） |
| EDS/XPS + 化学式 | 元素比应与目标化合物的化学计量比一致 | 目标Fe_2O_3但EDS显示Fe:O远偏离2:3（产物可能不纯） |
| BET + SEM/TEM | 比表面积应与颗粒尺寸和形貌一致 | BET显示高比表面积但SEM显示大块颗粒（可能有介孔结构，需进一步确认） |
