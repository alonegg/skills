# 生物环境类 学科配置

> 本文件是 paper-review 技能的学科叠加层。它不替换通用规则，而是在通用规则基础上增加生物环境类学科特有的审查维度、调整权重、补充常见问题模式。

## 适用范围

- **覆盖学科**：生物科学、生物技术、生物工程、生态学、环境科学、环境工程、食品科学与工程、农学
- **典型论文类型**：
  - **实验研究型**：设计并执行生物学/环境学实验，验证假设或探究机制（如基因功能验证、污染物降解实验、发酵工艺优化）
  - **野外调查型**：在自然或半自然环境中进行采样调查，描述物种分布、群落结构、环境质量现状（如植被样方调查、水质监测、土壤污染调查）
  - **生物信息学分析型**：基于公共数据库或测序数据进行计算分析（如转录组分析、宏基因组分析、分子进化分析、蛋白质结构预测）
  - **环境评价型**：对特定区域或项目进行环境影响评价或环境风险评估（如建设项目环评、区域生态风险评价）
  - **工艺处理设计型**：设计或优化污水处理、废气治理、固废处理、食品加工等工艺流程（如A2O工艺优化、厌氧发酵产气优化）
- **识别线索**：论文中出现物种名称（拉丁学名）、基因/蛋白名称、引物序列、实验动物品系、采样点坐标、环境质量标准编号（GB 3838等）、生态指数（Shannon-Wiener、Simpson）、分子生物学实验流程（PCR、Western blot、ELISA）、统计分析软件（SPSS、R、GraphPad Prism）、生物信息学工具（BLAST、MEGA、DESeq2）

## 结构期望

### 标准章节结构

本学科论文结构因论文类型而异，审查时应先判断论文属于哪种类型，再对照对应结构：

**实验研究型**（最常见）：
```
第1章 绪论（研究背景、国内外研究现状、研究目的与意义、技术路线）
第2章 材料与方法（实验材料、试剂与仪器、实验方法、数据分析方法）
第3章 结果与分析（按实验顺序展示结果，配合统计分析）
第4章 讨论（结果解释、与前人研究的对比、机制探讨）
第5章 结论与展望
```

**野外调查型**：
```
第1章 绪论（研究背景、研究区概况、研究目的与意义）
第2章 研究区域与研究方法（研究区自然条件、采样设计、调查方法、数据处理方法）
第3章 结果与分析（物种组成/环境质量现状、时空变化特征、影响因素分析）
第4章 讨论（与其他区域对比、驱动因素解析、保护/治理建议）
第5章 结论与展望
```

**生物信息学分析型**：
```
第1章 绪论（研究背景、相关研究进展、研究目的）
第2章 材料与方法（数据来源与质控、分析流程与软件、统计方法）
第3章 结果与分析（差异分析、功能富集、网络构建、关键基因/通路筛选）
第4章 讨论（生物学意义解读、与已知研究的一致性/差异、局限性）
第5章 结论与展望
```

**环境评价型**：
```
第1章 绪论（评价背景、法规依据、评价目的）
第2章 评价区域概况与评价方法（区域自然社会条件、评价标准、评价方法选择与依据）
第3章 环境现状调查与监测（监测方案、监测结果）
第4章 环境影响预测与评价（预测模型、评价结果、达标分析）
第5章 环保措施与建议
第6章 结论
```

**工艺处理设计型**：
```
第1章 绪论（处理对象特征、现有工艺综述、研究目的）
第2章 工艺设计（工艺流程选择论证、设计参数、主要构筑物/设备设计计算）
第3章 实验验证（小试/中试实验设计、运行参数优化）
第4章 结果与分析（处理效果、影响因素、最优条件确定）
第5章 经济技术分析（投资估算、运行成本、技术经济比较）
第6章 结论与展望
```

### 必须有的章节

- **绪论**：必须包含国内外研究现状的系统梳理，明确指出现有研究的不足和本文的切入点；不能仅有空泛的"意义"描述
- **材料与方法/研究方法**：本学科的核心章节之一，必须详细到他人可重复实验——试剂厂家与货号、仪器型号、实验条件参数、统计方法及软件版本
- **结果与分析**：结果展示和统计分析必须分离呈现（先给数据，再做分析），图表必须规范且有统计检验标注
- **讨论**：必须独立存在（不能与结果合并后省略讨论），需对结果进行机制层面的解释，并与前人研究对比
- **结论**：必须基于本文实际数据得出，不能超出实验/调查范围进行推断

### 可选/特殊章节

- **伦理声明**：涉及实验动物时必须有伦理委员会批准编号；涉及人体样本时必须有知情同意说明
- **技术路线图**：复杂实验研究建议在绪论末尾提供技术路线图，帮助读者理解实验逻辑
- **附录**：原始数据表、引物序列汇总表、仪器参数详表、物种名录等可放入附录

### 结构红线

- **无"材料与方法"章节**：直接展示结果但不说明怎么做的——实验不可重复，S级问题
- **结果与讨论完全混为一段**：每展示一个数据就讨论一段，导致读者无法区分客观结果和主观解释
- **缺少统计分析**：结果章节只有定性描述（"A组明显高于B组"）而无统计检验结果
- **讨论章节缺失或空壳化**：仅重复结果数字而无机制解释、无与前人研究的对比

## 专项规则权重调整

基于通用6大专项规则，调整本学科的审查权重：

| 专项规则 | 权重调整 | 理由 |
|---|---|---|
| intro | 0 | 无特殊调整，按通用标准审查 |
| literature | +1 | 生物环境领域发展迅速，综述需体现对研究前沿的追踪，且需区分"背景知识介绍"和"研究进展综述" |
| method | +2 | 方法的可重复性是实验科学的基石，材料与方法章节的详细程度直接决定论文的学术价值 |
| analysis | +2 | 结果的统计分析规范性和讨论的深度是本学科论文质量的核心区分度 |
| conclusion | +1 | 生物环境类论文结论需严格限定在数据支撑范围内，过度推断是常见问题 |
| fact | 0 | 无特殊调整 |

## 方法审查补充维度

通用 method 规则之外，本学科需额外检查：

### 实验设计规范

1. **生物学重复与技术重复的区分**：检查论文是否正确区分了生物学重复（biological replicates，来自独立个体/独立培养物）和技术重复（technical replicates，同一样本的重复测量）。统计分析必须基于生物学重复数，技术重复不能作为独立样本纳入统计。本科论文要求每组至少3个生物学重复
2. **样本量论证**：动物实验或临床相关研究需说明样本量确定依据（如power analysis）；若无法进行功效分析，应说明样本量选择的文献依据或实际约束
3. **随机化与盲法**：检查是否描述了实验分组的随机化方法；涉及主观评价的实验（如病理切片评分、行为学评分）是否采用了盲法
4. **对照设置**：阳性对照（验证实验体系有效）和阴性对照（排除假阳性）是否齐全；空白对照、溶剂对照等是否根据实验需要设置

### 采样规范

5. **采样设计**：检查采样方式（随机、系统、分层）是否与研究目的匹配；样方大小、数量是否满足统计学要求；采样时间和频率是否覆盖了研究对象的时间变异性
6. **样本保存与运输**：是否描述了样本采集后的保存条件（温度、容器、固定液）和运输方式；环境样本是否说明了保存时限和链式保管（chain of custody）
7. **空间与时间覆盖度**：野外调查的采样范围是否足以支撑结论？单次采样能否代表该区域的一般状况？是否讨论了季节变异、年际变异等时间因素

### 生物统计方法

8. **统计检验选择**：数据是否满足所选统计方法的前提假设？生物学数据通常不服从正态分布，是否进行了正态性检验（Shapiro-Wilk）和方差齐性检验（Levene）？不满足参数检验前提时是否选用了非参数检验？
9. **多重比较校正**：涉及多组比较时是否进行了多重比较校正（Tukey HSD、Bonferroni、Dunn检验等）？组学数据的多重检验校正（FDR/BH校正）是否执行？
10. **生物多样性指数**：使用Shannon-Wiener指数、Simpson指数、Chao1估计量等时，是否说明了计算依据和适用条件？物种丰富度比较是否考虑了样本量的稀释曲线（rarefaction curve）校正？

### 分子生物学实验规范

11. **引物设计与验证**：PCR引物是否给出了完整序列、退火温度、产物大小？是否说明了引物设计软件和验证方式（如特异性BLAST比对、熔解曲线分析）？
12. **PCR条件优化**：是否给出了完整的PCR反应体系和扩增程序？qRT-PCR是否报告了扩增效率和内参基因的选择依据？
13. **凝胶电泳规范**：凝胶图像是否包含分子量标记（Marker）？条带大小是否与预期一致？图像是否有过度调整痕迹（过曝、裁切不当）？
14. **测序质量控制**：高通量测序数据是否报告了Q30值、测序深度、比对率等质控指标？数据过滤标准是否明确？

### 伦理审查

15. **动物实验伦理**：涉及实验动物的研究是否获得了实验动物伦理委员会（IACUC）的批准？是否注明了批准编号？是否说明了动物饲养条件和处死方式？
16. **生物安全**：涉及病原微生物的实验是否在相应生物安全等级的实验室中进行？转基因操作是否遵循相关法规？
17. **环境采样许可**：自然保护区、水源保护区等特殊区域的采样是否获得了相关许可？

### 环境监测与评价规范

18. **标准方法引用**：环境监测项目是否引用了国家标准方法（GB/HJ系列）、EPA方法或ISO方法？分析方法是否与监测目标匹配？
19. **检出限与定量限**：是否报告了各监测项目的方法检出限（MDL）和定量限（MQL）？低于检出限的数据如何处理是否说明？
20. **质量保证与质量控制（QA/QC）**：是否描述了现场空白、运输空白、实验室空白、加标回收、平行样等QA/QC措施？回收率和相对偏差是否在可接受范围内？
21. **空间插值方法**：使用克里金插值、反距离加权等空间分析方法时，是否论证了方法选择的依据？是否进行了交叉验证？

## 证据与论证标准补充

通用 analysis/fact 规则之外，本学科的证据标准：

### 可接受的证据类型

- **实验数据**：在明确实验条件下产生的可量化结果（基因表达量、酶活性、浓度、去除率等），必须附带统计检验结果（P值、置信区间）和效应量（effect size）。每组至少3个生物学重复
- **野外调查数据**：按规范采样方案获取的现场数据，必须说明采样时间、地点（坐标）、方法和环境条件。数据需经过质控筛选
- **高通量组学数据**：转录组、蛋白质组、代谢组等数据，必须提交至公共数据库（GEO、NCBI SRA等）并给出登录号。分析流程需完整描述软件版本和参数
- **环境监测数据**：按国家或国际标准方法获取的监测数据，必须附QA/QC结果。未经质控的原始数据不可直接作为证据
- **生物信息学预测结果**：可作为线索性证据，但不能单独支撑功能性结论；需标明为"预测结果"并建议实验验证
- **文献数据引用**：引用前人发表的数据作为对比依据，需注明数据来源和实验条件差异

### 证据红线

- **无生物学重复的实验结论**：每个条件只做了1次（n=1），无法进行统计检验，任何基于此的结论均不成立
- **仅凭生物信息学预测下功能性结论**：仅通过序列分析预测某基因具有某功能，未经任何实验验证（如基因敲除/过表达、酶活测定），不能声称"发现了XX基因的功能"
- **环境评价不引用标准方法**：监测数据的获取方法无标准依据，数据可信度无法保证
- **P值操纵**：选择性报告显著性结果、将P=0.06描述为"接近显著"并据此下结论、大量比较后不做多重校正
- **图像篡改**：Western blot、凝胶图像经过裁切拼接但未标注、图像对比度过度调整导致条带失真
- **数据点过少的曲线拟合**：用3-4个数据点拟合复杂模型（如Michaelis-Menten方程、Langmuir等温线），过拟合风险极高

### 论证模式期望

- **实验研究型**：假设→实验设计→结果→统计检验→机制解释→与前人对比的实证研究范式。假设必须先于实验提出，不能先看到结果再编造假设
- **野外调查型**：研究问题→采样设计→数据收集→模式识别→驱动因素分析→保护/管理建议的归纳论证。结论的推广范围必须限定在采样区域和时段内
- **生物信息学分析型**：数据获取→质控→计算分析→生物学解读→实验验证建议的分析论证。必须强调计算结果的预测性质，区分"预测"和"证实"
- **环境评价型**：法规依据→现状调查→影响预测→达标判断→措施建议的规范论证。评价结论必须有明确的标准依据
- **工艺设计型**：处理目标→工艺选择论证→参数优化→效果验证→经济分析的工程论证。工艺选择必须有对比论证，不能无理由选定

## 常见问题模式

### S级（论文是否成立）

1. **无生物学重复**：
   - 表现：每个实验条件仅做了1次实验（n=1），或将技术重复当作生物学重复报告（如同一培养皿取3次样声称"3次重复"）
   - 影响：无法进行有效的统计检验，所有实验结论均不可信，论文的核心数据基础不成立
   - 修改方向：重新设计实验，确保每组至少3个独立的生物学重复；明确区分生物学重复和技术重复，在方法部分清晰说明

2. **动物实验缺乏伦理批准**：
   - 表现：论文涉及小鼠、大鼠、兔等实验动物，但全文未提及实验动物伦理委员会（IACUC）审批，无伦理批准编号
   - 影响：违反《实验动物管理条例》和学术出版伦理要求，论文不具备发表资格，学位论文可能面临撤销风险
   - 修改方向：补充伦理审批信息；若实验确实未获批准，需在答辩前补办伦理审查手续

3. **环境评价未引用标准方法**：
   - 表现：水质、大气、土壤等环境监测数据的获取未引用任何国家标准或行业标准方法，分析方法来源不明
   - 影响：数据可信度无法验证，评价结论缺乏法规依据，整篇评价报告不具备有效性
   - 修改方向：逐项核实每个监测指标的分析方法，补充对应的标准编号（如GB 11893-1989用于总磷测定）；若部分指标确实无对应国标，需引用行业标准或国际标准并说明理由

4. **仅凭生物信息学预测声称基因功能**：
   - 表现：通过转录组差异分析或序列比对筛选出候选基因后，直接声称"发现了调控XX过程的关键基因"，未进行任何实验验证（如qRT-PCR验证、基因功能实验）
   - 影响：计算预测与实验证实之间存在本质区别，未验证的预测不能作为确定性结论
   - 修改方向：将结论表述修改为"预测/筛选了候选基因"；补充至少qRT-PCR验证关键基因的表达趋势；对功能性结论需补充基因敲除/过表达或蛋白互作等实验

5. **实验数据伪造或图像篡改**：
   - 表现：Western blot条带明显拼接（背景不连续）、同一张凝胶图在不同实验中复用、数据过于"完美"（所有重复值几乎相同且无异常值）
   - 影响：学术不端，论文面临撤销、学位撤回等严重后果
   - 修改方向：提供原始未处理图像和原始数据；若确为操作失误（如裁切未标注），需在图注中明确说明图像处理方式

### A级（严重影响质量）

1. **混淆生物学重复与技术重复**：
   - 表现：方法部分声称"实验重复3次"，但实际是对同一批培养物/同一组织样本重复检测3次，并以n=3进行统计分析
   - 影响：统计自由度被高估，P值不可信，实际上仍是伪重复（pseudoreplication），结论可靠性大幅下降
   - 修改方向：重新明确重复类型，如果确实只有技术重复，需补充独立的生物学重复实验；若无法补做，需在讨论中承认该局限

2. **统计方法与数据类型不匹配**：
   - 表现：对明显不服从正态分布的数据（如物种计数、等级评分）使用t检验或方差分析；对非独立样本（如同一个体不同时间点的重复测量）使用独立样本检验
   - 影响：统计结果无效，基于错误统计得出的"显著性差异"不可信
   - 修改方向：先检验数据分布（Shapiro-Wilk检验），不满足正态性则使用非参数检验（Mann-Whitney U、Kruskal-Wallis）；重复测量数据使用重复测量方差分析或混合效应模型

3. **分子实验缺乏阴性/阳性对照**：
   - 表现：PCR实验没有无模板对照（NTC）和阳性对照；Western blot没有阳性对照样本；转染实验没有空载体对照
   - 影响：无法排除假阳性（污染）或确认实验体系有效性，实验结果的可靠性受质疑
   - 修改方向：补做含完整对照的实验；至少在现有结果基础上补充对照实验数据并在图中展示

4. **采样设计不足以支撑结论**：
   - 表现：研究声称描述了某区域的植被/水质空间分布格局，但仅设置了3-5个采样点且集中在小范围内；或声称描述了年际变化但仅有单次采样数据
   - 影响：有限的采样点/时间无法代表整个研究区域的空间/时间变异性，结论的外推性严重不足
   - 修改方向：在讨论中明确限定结论的时空适用范围；补充采样或引用其他来源数据进行验证；建议增加采样点/频次

5. **环境标准引用过时或错误**：
   - 表现：水质评价引用已废止的旧版标准（如引用GB 3838-2002的过期前版本），或用地表水标准评价地下水，或用错功能区类别
   - 影响：评价结论的法规基础错误，整个评价可能需要推翻重做
   - 修改方向：核实所有引用标准的现行有效性（通过国家标准全文公开系统查询）；确认评价对象与标准的适用范围匹配

6. **高通量数据分析流程不透明**：
   - 表现：仅写"使用R语言进行差异分析"，未说明具体使用的R包名称和版本、过滤阈值（如log2FC > 1且padj < 0.05的依据）、归一化方法、质控步骤
   - 影响：分析过程不可重复，读者无法评估结果的可靠性
   - 修改方向：以流程图形式展示完整的分析管线（pipeline）；逐步注明软件/R包版本、关键参数设置、阈值选择依据

7. **讨论章节无机制解释**：
   - 表现：讨论部分仅重复结果数字（"实验组的SOD活性为XX U/mg，显著高于对照组"），未从生物学/化学机制角度解释原因，未与已发表研究对比
   - 影响：讨论丧失了应有的学术深度，论文沦为数据报告而非研究论文
   - 修改方向：对每个关键结果从机制层面进行解读（如氧化应激通路、微生物代谢途径）；与同类研究的结果进行对比并分析异同的可能原因

### B级（影响说服力）

1. **凝胶/blot图像呈现不规范**：
   - 表现：凝胶图像缺少分子量标记（Marker），或Marker标注不清晰；图像过曝导致条带饱和无法判断量差异；裁切后不标注裁切线
   - 影响：读者无法判断目的条带大小是否正确，降低实验结果的可信度
   - 修改方向：补充含清晰Marker标注的完整凝胶图像；保留原始图像作为补充材料；在图注中说明曝光条件和任何图像处理操作

2. **物种命名不规范**：
   - 表现：物种拉丁学名未斜体；首次出现时未给出命名人；属名缩写使用不一致（首次出现即缩写）；中文名与学名对应错误
   - 影响：不符合生物学命名法规（ICN/ICZN），降低论文的专业性
   - 修改方向：全文统一为标准命名格式——首次出现写全称并斜体（如 *Escherichia coli* Migula, 1895），此后可缩写属名（*E. coli*）；植物需附命名人

3. **生物信息学工具版本未注明**：
   - 表现：方法部分仅写"使用BLAST进行序列比对"、"使用MEGA进行进化分析"，未注明软件版本号和关键参数（如比对算法、进化模型选择）
   - 影响：不同版本的软件可能产生不同结果，缺少版本信息导致分析不可重复
   - 修改方向：为每个使用的软件/数据库注明版本号和访问日期（如"MEGA 11.0.13"、"NCBI nr数据库，2025年6月版"）；注明所有关键参数设置

4. **地图/采样点图缺乏基本要素**：
   - 表现：采样点分布图缺少比例尺、指北针、坐标信息、图例；底图来源未标注；采样点编号与正文不对应
   - 影响：读者无法判断采样的空间布局合理性和覆盖范围
   - 修改方向：使用GIS软件或专业制图工具重新制作采样点图，确保包含比例尺、指北针、经纬度坐标、图例、底图来源标注

5. **环境参数报告不完整**：
   - 表现：水质、土壤等环境参数仅报告浓度数值，未注明单位、检出限、测定方法；低于检出限的值标为"0"或空白而非"<MDL"
   - 影响：数据的精确含义不明确，无法与标准值或其他研究结果进行有效比较
   - 修改方向：在数据表中统一标注单位和检出限；低于检出限的值统一标注为"ND"或"<[检出限值]"并在表注中说明处理方式

6. **图表统计标注缺失**：
   - 表现：柱状图仅展示均值而无误差线（标准差/标准误）；组间比较未标注显著性符号（如 *P<0.05, **P<0.01）；图注中未说明统计方法
   - 影响：读者无法判断组间差异的统计显著性和数据离散程度
   - 修改方向：所有柱状图/折线图添加误差线（建议使用标准差SD，并在图注中注明）；显著性差异以星号或字母标注法标示；图注中注明统计方法和样本量

7. **化学物质/污染物命名不规范**：
   - 表现：有机污染物使用俗名而非IUPAC名称且未在首次出现时标注CAS号；重金属元素符号使用不一致；缩写如COD、BOD首次出现未给出全称
   - 影响：命名不统一导致阅读困难，可能引起歧义
   - 修改方向：化学物质首次出现时给出IUPAC名称或通用名称+CAS号；环境参数缩写首次出现时给出全称（如"化学需氧量（Chemical Oxygen Demand, COD）"）；此后全文统一使用缩写

## 文献综述特殊要求

### 文献来源优先级

1. **高影响力综合期刊**：Nature、Science、Cell（及其子刊如 Nature Ecology & Evolution、Nature Microbiology、Cell Reports）、PNAS
2. **学科顶级期刊**：
   - 生态学：Ecology Letters、Ecology、Journal of Ecology、Molecular Ecology
   - 环境科学：Environmental Science & Technology、Water Research、Environment International、Journal of Hazardous Materials
   - 分子生物学：Nucleic Acids Research、Molecular Biology and Evolution、Plant Cell
   - 微生物学：ISME Journal、mBio、Applied and Environmental Microbiology
   - 食品科学：Food Chemistry、Journal of Agricultural and Food Chemistry
3. **国内核心期刊**：生态学报、环境科学、中国环境科学、应用生态学报、生物工程学报、微生物学报、环境科学学报、农业环境科学学报
4. **学位论文与技术报告**：可作为方法参考引用，但不能替代同行评议期刊论文作为主要学术依据
5. **标准与法规**：国家标准（GB）、环保行业标准（HJ）、EPA方法、ISO标准——作为方法依据必须引用，但不计入"学术参考文献"

### 文献数量底线

- 实验研究型：至少20篇学术论文（不含标准和教材），其中外文文献不低于30%，近5年文献不低于40%
- 野外调查型：至少15篇学术论文，必须包含研究区域或相似区域的已有研究
- 生物信息学分析型：至少20篇学术论文，需包含分析方法的原始文献（如DESeq2、edgeR的原始论文）
- 环境评价型：至少10篇学术论文+所有相关法规和标准文件
- 工艺设计型：至少15篇学术论文，需包含同类工艺的已有研究和对比工艺的文献

### 综述组织方式

- **禁止**：逐篇罗列式综述（"张三[1]研究了XX，李四[2]发现了XX"）；将教科书知识大段复制作为文献综述
- **要求**：按研究主题、技术路线、发展阶段或争议焦点组织综述，形成逻辑清晰的分析框架
- **必须有**：综述末尾的"研究不足与本文切入点"段落，明确指出现有研究的空白、局限性或争议，以及本文如何针对这些问题开展工作
- **特殊关注**：生物环境类综述需注意区域差异性——国外研究结论不能直接套用于国内环境条件，需分析适用性

## 术语与规范

### 生物学命名规范

- **物种拉丁学名**：必须斜体；首次出现时写完整属名+种加词+命名人（如 *Arabidopsis thaliana* (L.) Heynh.）；后续可缩写属名（*A. thaliana*）
- **基因命名**：基因名称斜体（如 *BRCA1*、*nifH*），蛋白名称正体（如 BRCA1、NifH）；模式生物遵循各自的命名规范（如拟南芥基因全大写斜体 *AtSOD1*，小鼠基因首字母大写斜体 *Sod1*）
- **微生物菌株**：种名后标注菌株编号（如 *E. coli* BL21(DE3)）；新分离菌株需注明保藏中心编号

### 环境科学术语规范

- **标准缩写**：COD（化学需氧量）、BOD5（五日生化需氧量）、SS（悬浮物）、DO（溶解氧）、TN（总氮）、TP（总磷）、NH3-N（氨氮）等——首次出现时必须给出中英文全称，此后使用缩写
- **浓度单位**：统一使用SI单位制；水质指标一般用mg/L，微量污染物可用μg/L；气体浓度区分体积浓度（ppm/ppb）和质量浓度（mg/m³）并注明
- **环境标准引用格式**：引用标准时注明标准号和年份（如GB 3838-2002《地表水环境质量标准》）；注明所适用的功能区类别

### 引用格式

- 遵循GB/T 7714-2015标准
- 期刊论文引用需包含：作者、论文标题、期刊名（可缩写）、年份、卷（期）、页码
- 标准文件引用格式：标准编号-年份 标准名称 [S]
- 数据库引用：需注明数据库名称、登录号、访问日期

### 图表规范

- **实验结果图**：柱状图必须有误差线（标注SD或SEM）和显著性标记；折线图的数据点必须清晰可辨；热图需标注色标含义和聚类方法
- **凝胶/blot图**：必须包含Marker标注、样品标识、目的条带位置标注；裁切图需在图注中说明
- **采样点分布图**：必须包含比例尺、指北针、坐标系（推荐WGS84）、图例、底图来源
- **系统发育树**：必须标注bootstrap值或后验概率、分支长度含义（替代率/时间）、外群选择依据、序列登录号
- **工艺流程图**：按标准化工制图规范，标注流量、浓度等关键设计参数

### 数据呈现规范

- 所有定量数据需注明有效数字位数的确定依据（一般与仪器精度一致）
- 百分比保留1-2位小数；浓度值的有效数字不超过仪器精度
- 数据表格中的均值±标准差格式统一（如 12.3 ± 1.5，不要混用 12.3±1.5 和 12.3 +/- 1.5）

## 结论与建议特殊要求

### 结论章节要求

- **结论必须基于数据**：每条结论必须有对应的实验/调查数据支撑，不能出现未经验证的推测性结论
- **量化总结**：结论应包含关键定量结果（如"XX处理组的去除率达到95.3%，显著高于对照组的72.1%（P<0.01）"），不能只有定性描述
- **范围限定**：结论的适用范围必须与实验条件/采样范围一致——实验室条件下的结论不能推广到野外环境；单一区域的调查结论不能推广到全国尺度
- **禁止过度推断**：不能从相关性推断因果性；不能从体外实验结论推断体内效果；不能从单一物种的结论推广到整个类群

### 建议部分的合理边界

- **实验研究型**：建议应围绕"下一步需要验证什么"——如补充体内实验、扩大样本范围、探究上下游调控机制等
- **野外调查型**：管理建议必须基于调查数据且具有可操作性；不能提出远超数据支撑范围的宏大建议（如"建议建立全流域生态补偿机制"——除非论文确实进行了全流域评估）
- **环境评价型**：环保措施建议必须针对评价中识别的具体问题，且具有技术可行性和经济合理性
- **工艺设计型**：展望应聚焦于工艺优化方向和放大实验计划，不能脱离工程实际

### 展望部分的红线

- 展望内容必须与本文研究方向直接相关（如论文研究的是水稻抗旱基因，不应展望"在医学领域的应用前景"）
- 不能用展望弥补论文的不足（如"由于时间限制未能完成XX实验，未来将继续开展"——如果该实验是支撑核心结论所必需的，则属于S/A级问题，不能推到展望中）
- 展望应具体到可操作的研究计划，避免空泛表述（如"为生态文明建设做出贡献"）

## 学科专属审查维度

### 实验可重复性评估

审查论文是否提供了足够的信息使他人能重复实验：

| 检查项 | 必须包含的信息 | 常见缺失 |
|---|---|---|
| 实验材料 | 物种/品系、来源、培养条件 | 仅写"大肠杆菌"，未标注菌株 |
| 试剂 | 厂家、货号、纯度/浓度 | 仅写"胰蛋白酶"，未标注来源 |
| 仪器 | 型号、厂家、关键参数设置 | 仅写"离心机"，未写转速和时间 |
| 实验条件 | 温度、时间、pH、光照等 | 缺少关键条件参数 |
| 统计方法 | 软件名称+版本、具体检验方法 | 仅写"使用SPSS分析" |
| 生物信息学 | 软件版本、数据库版本、参数 | 仅写工具名称无版本 |

### 环境评价规范性检查

| 检查项 | 合格标准 | 常见问题 |
|---|---|---|
| 标准引用 | 引用现行有效标准，功能区匹配 | 标准已废止、功能区错误 |
| 监测布点 | 点位布设有规范依据 | 布点随意，无代表性论证 |
| QA/QC | 空白、平行、加标回收齐全 | 完全无质控描述 |
| 数据处理 | 检出限以下数据处理有说明 | ND值当0处理或直接忽略 |
| 评价方法 | 方法选择有论证 | 无理由选择单因子指数法 |

### 分子实验图像质量检查

| 图像类型 | 必须包含的要素 | 不可接受的情况 |
|---|---|---|
| 凝胶电泳 | Marker、样品标识、预期条带大小标注 | 无Marker、过曝、拼接未标注 |
| Western blot | 分子量标记、内参条带、曝光条件说明 | 内参缺失、条带饱和、明显拼接 |
| 荧光显微镜 | 比例尺、激发/发射波长、对照组图像 | 无比例尺、无对照、过度增强 |
| 流式细胞术 | 门控策略、阴性对照、坐标轴标签 | 门控策略不明、无阴性对照 |
| 组织切片 | 染色方法、放大倍数、比例尺 | 无比例尺、染色方法未标注 |
