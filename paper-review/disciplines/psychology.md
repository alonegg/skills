# 心理学 学科配置

> 本文件是 paper-review 技能的学科叠加层。它不替换通用规则，而是在通用规则基础上增加心理学科特有的审查维度、调整权重、补充常见问题模式。心理学是方法驱动型学科，方法章节的审查深度应显著高于其他章节。

## 适用范围

- **覆盖学科**：基础心理学、应用心理学、教育心理学、发展心理学、临床与咨询心理学、社会心理学
- **典型论文类型**：实验研究型、问卷调查型、质性研究型、量表编制/修订型、元分析型（本科少见但存在）
- **识别线索**：出现"被试""自变量""因变量""量表""信度""效度""中介效应""调节效应""Cronbach's α""Cohen's d""SPSS""AMOS""Mplus""PROCESS""问卷""实验范式""启动""反应时""眼动""ERP""fMRI""心理健康""人格""认知""情绪""动机""依恋""自我效能感""心理资本"等关键词；引用心理学期刊（《心理学报》《心理科学》等）；研究对象涉及心理变量的测量

## 结构期望

### 标准章节结构

心理学本科论文高度接近IMRAD范式，按论文类型有四种典型结构：

**实验研究型**
```
第1章 引言（研究背景、文献综述、研究问题与假设）
第2章 方法（被试、实验设计、实验材料/仪器、实验程序）
第3章 结果（描述统计、推断统计、图表呈现）
第4章 讨论（结果解释、与已有研究比较、理论贡献、局限性）
第5章 结论
```

**问卷调查型**
```
第1章 引言（研究背景、核心概念界定）
第2章 文献综述（各变量的相关研究、变量间关系的理论基础、研究假设）
第3章 研究方法（被试、研究工具、施测程序、数据分析策略）
第4章 结果（共同方法偏差检验、描述统计与相关分析、回归/中介/调节分析）
第5章 讨论（结果解释、与已有研究比较、实践启示、局限与展望）
第6章 结论
```

**质性研究型**
```
第1章 引言（研究背景、研究目的）
第2章 文献综述
第3章 方法论与方法（研究范式声明、具体方法选择理由、研究者反思性定位）
第4章 数据收集与分析过程（参与者选取、访谈/观察过程、编码过程详述）
第5章 研究发现/主题呈现（各主题的厚描述与原始资料引证）
第6章 讨论（与已有文献对话、理论意涵）
第7章 结论（研究总结、局限与未来方向）
```

**量表编制/修订型**
```
第1章 引言（研究背景、构念界定与操作化）
第2章 文献综述（构念的理论模型、已有测量工具评述）
第3章 量表编制过程（题目生成→专家评审→预测试→项目分析→探索性因子分析）
第4章 信效度检验（验证性因子分析→效标关联效度→重测信度→内部一致性信度）
第5章 结果（最终量表的因子结构、信效度指标汇总）
第6章 讨论（与理论模型的一致性、量表的适用范围与局限）
```

### 必须有的章节

- **引言**：必须包含研究背景、文献综述（或独立成章）、明确的研究问题和研究假设；心理学论文的假设必须具体到变量间关系的方向（如"自我效能感正向预测学业成绩"），不能仅写"探究X与Y的关系"
- **方法**：心理学论文的方法章节是核心章节，必须独立成章且内容详尽到他人可以据此复现研究。最低要求：被试描述、研究工具/材料、实施程序、数据分析方法
- **结果**：必须独立于讨论，纯粹报告数据分析结果，不掺入解释。心理学严格区分结果（Results）与讨论（Discussion）
- **讨论**：必须将本研究发现与已有文献进行逐一比较，不能只写"本研究发现与前人一致"而不指出具体是哪些前人研究
- **结论**：必须回应所有假设，逐条说明支持或不支持

### 可选/特殊章节

- **预研究/预实验**：量表编制型论文通常需要；实验研究在正式实验前可能有预实验确定参数
- **研究一/研究二/研究三**：多研究设计的论文可按子研究分章，每个子研究内部各自包含方法-结果-讨论
- **伦理声明**：涉及临床样本、未成年人、敏感话题（自杀、创伤、性行为等）时必须有独立伦理说明段落

### 结构红线

- 没有明确的研究假设，全文只写"探究X与Y的关系"——缺少理论驱动，论文沦为数据挖掘
- 结果与讨论混在一起——违反心理学论文的基本写作规范
- 方法章节缺失被试描述或研究工具来源——论文不可复现
- 实验研究没有描述实验程序的时间线和具体操作步骤
- 量表编制型论文跳过探索性因子分析直接做验证性因子分析——违反量表开发的基本程序

## 专项规则权重调整

基于通用6大专项规则，调整心理学科的审查权重：

| 专项规则 | 权重调整 | 理由 |
|---|---|---|
| intro | +1 | 心理学要求假设必须从文献中逻辑推导出来，引言的论证链直接决定假设的合理性 |
| literature | +1 | 文献综述需建构变量间关系的理论逻辑，而非简单罗列，是假设提出的基础 |
| method | +3 | 心理学是方法驱动型学科，方法章节的严谨程度直接决定论文的科学价值——这是最核心的审查对象 |
| analysis | +2 | 统计分析的正确性在心理学中至关重要：方法与设计是否匹配、假设检验是否恰当、效应量是否报告 |
| conclusion | 0 | 保持通用权重；心理学论文结论通常较为克制 |
| fact | -1 | 心理学论文事实描述段落较少，研究结果以数据形式呈现 |

## 方法审查补充维度

通用 method 规则之外，心理学科需额外检查以下维度。**方法审查是心理学论文审查的重中之重，审查者应在此投入最多精力。**

### 1. 实验设计审查

#### 1.1 变量定义与操作化

- **自变量（IV）**：是否明确定义？操作化是否恰当？例如，声称研究"焦虑"的影响，自变量是状态焦虑还是特质焦虑？通过什么方式操纵（情境诱发、指导语、视频材料、TSST压力测试）？操纵检验（manipulation check）是否实施？
- **因变量（DV）**：测量指标是否恰当？是行为指标（反应时、正确率）、生理指标（心率、皮电、ERP成分）还是自我报告指标？如果DV仅为自我报告，需讨论方法偏差
- **控制变量/协变量**：是否识别并控制了关键的混淆变量？如年龄、性别、受教育水平、基线水平等。是否通过随机分配、匹配或统计控制（ANCOVA）来处理？

#### 1.2 实验设计类型

- **被试间/被试内/混合设计**：设计类型是否明确说明？选择理由是否合理？
- **被试内设计的特殊要求**：是否说明了平衡措施（counterbalancing）？如拉丁方设计、ABBA顺序。未平衡的被试内设计存在顺序效应和练习效应的威胁
- **随机化**：被试是否随机分配到各条件？如果不是（如使用自然班级），是否明确承认为准实验设计并讨论效度威胁？随机化方式是否说明（随机数表、计算机随机）？
- **对照条件**：是否设置了恰当的控制条件？控制条件是空白对照、安慰剂对照还是积极对照？选择理由是否说明？

#### 1.3 样本

- **样本量与统计检力**：是否进行了先验统计检力分析（a priori power analysis）？推荐使用G*Power软件。如果没有做检力分析，样本量的确定依据是什么？以下为常见最低样本量参考：
  - 独立样本t检验：每组至少30人（中等效应量d=0.5, α=.05, power=.80时需每组64人）
  - 单因素方差分析（3组）：每组至少20人
  - 相关分析：N ≥ 80（中等效应量r=.30）
  - 多元回归：N ≥ 50 + 8k（k为预测变量数）
  - 结构方程模型：N ≥ 200（严格标准）；N ≥ 10 × 估计参数数（宽松标准）；N < 100的SEM结果不可接受
  - 探索性因子分析：N ≥ 题目数 × 5，且绝对值 N ≥ 150
  - 中介分析（Bootstrap）：N ≥ 200（Fritz & MacKinnon, 2007推荐）
- **被试招募与筛选**：招募方式（课程学分、报酬、志愿等）是否说明？纳入和排除标准是否明确？是否报告了排除的被试数量及原因？
- **人口统计学描述**：年龄（M, SD, range）、性别比例、教育水平、职业等是否报告？临床样本是否报告诊断标准和来源机构？

#### 1.4 实验材料与仪器

- **刺激材料**：实验材料的选取依据是否说明（如从哪个标准化情绪图片库中选取？IAPS、CAPS？）？材料的标准化属性（效价、唤醒度等）是否报告？
- **实验仪器**：使用的软件（E-Prime、PsychoPy、Inquisit等）和硬件（屏幕刷新率、眼动仪型号、EEG导联数等）是否具体说明？
- **时间参数**：刺激呈现时间、ISI（刺激间隔）、ITI（试次间隔）是否报告？这些参数的设置依据是否引用先前研究？

#### 1.5 实验程序

- **流程完整性**：从被试到达实验室到离开的完整流程是否描述？包括知情同意、练习试次、正式实验、操纵检验、人口统计学信息收集、事后解释（debriefing）
- **实验试次数**：总试次数、各条件试次数是否报告？是否有足够的试次保证测量的稳定性（通常每个条件至少20-30个试次）？
- **欺骗（deception）**：如果实验涉及欺骗，是否在事后对被试进行了充分的解释说明（debriefing）？是否在伦理审查中获得批准？

### 2. 问卷调查研究审查

#### 2.1 研究工具/量表

- **量表来源**：每个量表是否注明原始编制者和发表年份？如果使用中文版，是否注明中文修订者和修订年份？例如："采用Rosenberg（1965）编制、王孟成等（2010）修订的中文版自尊量表"
- **量表的文化适应**：国外量表是否经过正式的翻译-回译程序？是否有中国样本的信效度报告？直接翻译而未经信效度检验的量表不可接受
- **当前样本的信度**：**必须报告本研究样本中的Cronbach's α值**，仅引用原始研究的信度是不够的。α ≥ .70为可接受，α ≥ .80为良好。如果当前样本α < .70，必须讨论原因及对结果解释的影响
- **计分方式**：量表的计分方式是否说明（Likert几点计分、是否有反向计分题目）？总分/维度分的计算方式是否明确？

#### 2.2 共同方法偏差（Common Method Bias）

- **如果所有变量均通过自我报告问卷测量**：必须讨论共同方法偏差的潜在影响
- **程序控制**：是否采取了程序控制措施？如匿名施测、部分变量不同时间点施测、题目随机排列、设置反向计分题
- **统计检验**：是否进行了Harman单因子检验？第一个因子的方差解释率 < 40%可作为共同方法偏差不严重的初步证据（注意：Harman检验的敏感度较低，仅作为最低要求）。更好的做法是使用潜在方法因子法（unmeasured latent method factor）

#### 2.3 数据清洗

- 是否报告了有效问卷回收数与回收率？
- 无效问卷的剔除标准是否说明（如规律性作答、缺失值超过X%、作答时间过短）？
- 是否检查了数据的正态性分布？偏度和峰度的检验结果是否报告？

### 3. 统计分析审查（核心审查内容）

**审查原则：研究问题→研究设计→统计方法三者必须逻辑一致。**

#### 3.1 基础统计检验

| 分析方法 | 审查要点 |
|---|---|
| **独立样本t检验** | Levene方差齐性检验是否报告？不齐时是否使用校正df？是否报告Cohen's d？ |
| **配对样本t检验** | 前后测是否为同一批被试？时间间隔是否说明？是否报告Cohen's d？ |
| **单因素ANOVA** | 正态性检验和方差齐性检验是否报告？事后比较使用了哪种方法（Tukey、Bonferroni、Games-Howell）？选择理由是否合理？是否报告η²或偏η²？ |
| **重复测量ANOVA** | 球形检验（Mauchly's test）是否报告？违反球形假设时是否使用Greenhouse-Geisser或Huynh-Feldt校正？ |
| **卡方检验** | 期望频次是否满足要求（每格期望频次 ≥ 5）？不满足时是否使用Fisher精确检验？是否报告Cramér's V？ |
| **相关分析** | Pearson还是Spearman？选择依据（数据是否满足正态性和连续性）是否说明？是否报告置信区间？ |

#### 3.2 回归分析

- **多元回归**：是否检查了多重共线性？VIF > 10 或容忍度 < 0.1 表明严重多重共线性。是否报告了R²和调整R²？
- **回归假设**：残差的正态性、线性、同方差性是否检验？是否有残差图？
- **分层回归**：各步骤纳入变量的理论依据是否说明？ΔR²和ΔF是否报告？
- **逻辑回归**：模型拟合指标（-2LL、Nagelkerke R²、Hosmer-Lemeshow检验）是否报告？

#### 3.3 中介效应分析

- **必须使用现代方法**：推荐Hayes（2013）的PROCESS宏或Bootstrap方法。Baron & Kenny（1986）的逐步回归法已过时且检验力不足，如果论文仍使用该方法，应标记为A级问题
- **Bootstrap参数**：是否报告了Bootstrap抽样次数（推荐5000次）？是否报告了间接效应的95%置信区间？置信区间不包含0才能判定中介效应显著
- **因果序列的理论依据**：X→M→Y的因果顺序是否有理论支撑？横截面数据不能证明因果关系，论文是否承认了这一局限？
- **完全中介vs部分中介**：判断依据是否正确？直接效应不显著=完全中介，直接效应仍显著=部分中介

#### 3.4 调节效应分析

- **交互项**：是否对自变量和调节变量进行了中心化处理以减少多重共线性？
- **交互效应的呈现**：是否绘制了简单斜率图（simple slope plot）？是否进行了简单斜率检验（simple slope test）？
- **调节方向的解释**：是否正确解释了交互效应的含义？不能仅说"X的效应受M调节"，必须说明在M的不同水平（高/低）上X对Y的效应如何变化

#### 3.5 结构方程模型（SEM）

- **样本量**：N < 200时SEM结果的稳定性存疑，N < 100不可接受
- **模型拟合指标**：必须报告多个指标，最低要求：χ²/df（< 3为可接受）、CFI（≥ .90为可接受，≥ .95为良好）、RMSEA（≤ .08为可接受，≤ .06为良好）、SRMR（≤ .08为可接受）。仅报告一个指标不可接受
- **模型修正**：如果进行了模型修正（根据修正指数MI），修正是否有理论依据？纯粹根据MI数值修正模型属于数据驱动，需要标记
- **区分测量模型与结构模型**：是否先检验了测量模型（CFA）再检验结构模型？跳过CFA直接做SEM是方法缺陷
- **因子载荷**：标准化因子载荷是否 ≥ .50？低于.40的题目通常应删除
- **区分效度**：AVE的平方根是否大于各因子间相关系数？（Fornell-Larcker准则）

#### 3.6 探索性因子分析（EFA）

- **适切性检验**：KMO值是否报告？KMO ≥ .60为最低要求，≥ .80为良好。Bartlett球形检验是否显著？
- **因子提取**：使用什么方法（主成分分析PCA还是主轴因子法PAF）？心理学推荐PAF而非PCA（PCA严格来说不是因子分析）
- **因子数目确定**：依据是什么？仅用Kaiser标准（特征值>1）不够，应结合碎石图（scree plot）、平行分析（parallel analysis）
- **旋转方法**：正交旋转（Varimax）还是斜交旋转（Promax/Oblimin）？如果因子间存在相关（心理学构念通常如此），应使用斜交旋转
- **交叉载荷**：是否有题目在多个因子上载荷 > .40？这类题目的处理方式是否说明？

#### 3.7 验证性因子分析（CFA）

- **与EFA使用不同样本**：EFA和CFA是否使用了不同的样本（如将样本随机分半）？用同一批数据既做EFA又做CFA会导致模型拟合指标虚高
- **竞争模型比较**：是否与竞争模型（如单因子模型、双因子模型等）进行了比较，以证明目标模型优于替代模型？
- **拟合指标**：要求同SEM

### 4. 质性研究审查

#### 4.1 方法论声明

- **必须指明具体的质性研究方法**：不能笼统地说"采用质性研究方法"。是扎根理论（Grounded Theory）、解释性现象学分析（IPA）、主题分析（Thematic Analysis）、叙事分析（Narrative Analysis）还是个案研究（Case Study）？每种方法有不同的哲学基础和分析程序
- **方法选择的理由**：为什么选择这种质性方法而非其他方法？与研究问题的匹配性是否说明？

#### 4.2 参与者

- **目的性抽样**：是否说明了参与者选取的标准和理由（最大差异抽样、典型个案抽样、滚雪球抽样等）？
- **样本量**：IPA通常3-8人，扎根理论需要达到理论饱和（通常15-30人），主题分析6-15人。样本量与所选方法是否匹配？
- **参与者描述**：每位参与者的基本信息是否以表格形式呈现（使用化名）？

#### 4.3 数据收集

- **访谈**：半结构化访谈提纲是否附在附录中？访谈时长是否报告？是否全程录音并转录？
- **观察**：观察的频次、时长、记录方式是否说明？是参与式观察还是非参与式观察？
- **数据充分性**：是否讨论了数据饱和（data saturation）的判断标准？

#### 4.4 数据分析

- **编码过程**：是否逐步描述了从原始数据到主题的分析过程？是否给出编码示例（原始语句→初始编码→子主题→主题）？
- **编码者间信度**：是否有独立编码者？一致性系数（如Cohen's Kappa ≥ .60）是否报告？单人编码需说明可信性保障措施（同伴审核、参与者检验、审计追踪）
- **扎根理论的特殊要求**：是否遵循了开放编码→轴心编码→选择性编码的三级编码程序？是否呈现了核心范畴？
- **IPA的特殊要求**：是否对每位参与者进行了逐一深度分析，再跨参与者进行主题整合？

#### 4.5 质性研究的质量标准

- **可信性（credibility）**：是否采用了三角验证、成员检验（member checking）、负面个案分析等策略？
- **可转移性（transferability）**：是否提供了足够的厚描述（thick description）使读者能判断结论的适用范围？
- **可靠性（dependability）**：是否保留了审计追踪（audit trail），即编码过程的完整记录？
- **可确认性（confirmability）**：研究者是否反思了自己的预设和立场对研究的可能影响？

### 5. 量表编制/修订审查

#### 5.1 构念界定

- **理论基础**：量表测量的构念是否有明确的理论定义？理论维度的划分依据是什么（来自理论模型还是经验概括）？
- **与相近构念的区分**：是否说明了该构念与相近概念的区别？（如"学业倦怠"与"学业疲劳"、"学业压力"的异同）

#### 5.2 题目生成

- **题目来源**：题目来自文献分析、开放式调查、访谈，还是已有量表改编？
- **专家评审**：是否邀请了领域专家评审题目内容效度？专家人数（建议≥5人）、评审标准、修改情况是否说明？
- **内容效度指数（CVI）**：I-CVI和S-CVI是否报告？

#### 5.3 项目分析

- 题目的区分度（题总相关，corrected item-total correlation ≥ .30）是否报告？
- 极端组法（高低27%分组的t检验）是否使用？
- 删除题目的依据是否说明？

#### 5.4 因子分析程序

- **样本分割**：是否将样本随机分半，一半用于EFA，一半用于CFA？
- **EFA和CFA要求**：见上文3.6和3.7节
- **跨性别/跨年龄的测量不变性（measurement invariance）检验**：如果量表需要在不同群体中使用，是否检验了测量不变性（形态不变性→弱不变性→强不变性）？

#### 5.5 信效度报告

- **内部一致性信度**：Cronbach's α，分量表和总量表均需报告
- **重测信度**：间隔时长（建议2-4周）、重测样本量、重测系数是否报告？
- **聚合效度**：与测量相同/相似构念的已有量表的相关系数（应显著正相关）
- **区分效度**：与测量不同构念的量表的相关系数（应相关较低或不显著），或AVE的平方根大于因子间相关
- **效标关联效度**：与外部效标变量的相关是否符合理论预期？

### 6. 伦理审查

- **伦理审批**：是否获得了所在院校伦理审查委员会/IRB的审批？审批编号是否注明？心理学研究必须经过伦理审查
- **知情同意**：是否获取了被试的知情同意？未成年被试是否同时获取了法定监护人的同意？知情同意书是否附在附录中？
- **欺骗与事后解释**：如果实验涉及欺骗（如虚假反馈、掩盖真实目的），事后解释（debriefing）程序是否描述？
- **敏感内容处理**：涉及心理健康、创伤经历、自杀意念等敏感话题时，是否有风险预案（如发现高危被试后的转介机制）？
- **数据匿名化**：是否说明了个人信息的去识别化处理方式？原始数据的保管和销毁计划？
- **被试报酬**：报酬金额/形式是否说明？报酬水平是否恰当（不能过高以至于构成不当诱因）？

## 证据与论证标准补充

通用 analysis/fact 规则之外，心理学科的证据标准：

### 可接受的证据类型

- **实验数据**：在控制条件下收集的行为数据（反应时、正确率、选择偏好）、生理数据（ERP、fMRI、皮电、心率）、自我报告数据。使用条件：必须报告描述统计（M, SD）和推断统计，所有推断检验必须报告效应量
- **问卷数据**：经信效度检验的标准化量表得分。使用条件：量表来源和当前样本信度必须报告，共同方法偏差必须检验（所有变量均为自我报告时）
- **质性数据**：访谈转录文本、观察记录、日记/叙事材料。使用条件：必须有系统的编码分析过程，直接引用参与者原话并标注编号，不能仅用研究者的概括代替原始资料
- **行为观察数据**：结构化观察记录、视频编码数据。使用条件：编码系统和观察者间一致性必须报告
- **二手数据**：公开数据库（如CGSS、CFPS、CHARLS等）。使用条件：注明数据来源、年份、样本特征，变量的操作化定义必须说明

### 证据红线

- **自编量表未经任何信效度检验即使用**：这是心理学论文的根本性缺陷——没有心理测量学证据支持的工具，测量结果无意义
- **统计方法与研究设计根本不匹配**：如用相关分析的结果做因果推论、用参数检验处理明显非正态分布的数据且不做任何说明、用卡方检验处理连续变量
- **样本量严重不足**：如SEM分析的样本量 N < 100，或多元回归中预测变量数多于样本量的1/10
- **选择性报告**：只报告显著结果，不报告或隐藏不显著的发现（publication bias在论文层面的体现）
- **p-hacking迹象**：大量勉强显著的p值（如多个p = .04x），或者在没有先验假设的情况下进行大量两两比较而不做多重比较校正
- **伪造/篡改数据**：描述统计值明显不一致（如报告的均值不可能产生报告的t值），或数据模式过于完美

### 论证模式期望

- **实验研究型**：假设-验证模式。理论推导→提出假设→实验检验→接受/拒绝假设→理论修正。论证链条必须闭合
- **问卷调查型**：理论建模-实证检验模式。文献综述→理论模型/假设→数据收集→统计检验→模型修正。每个假设的推导必须有文献支撑
- **质性研究型**：归纳建构模式。从原始数据中涌现主题→跨主题整合→与已有理论对话→提出概念框架。必须展示从数据到理论的完整推理路径
- **量表编制型**：理论-实证迭代模式。理论构念界定→题目生成→统计检验→题目删改→再检验。每一轮迭代的决策依据必须说明

## 常见问题模式

### S级（论文是否成立）

1. **没有研究假设，只有笼统的"探究"**：表现为"本研究旨在探究X与Y的关系"，没有根据理论提出方向性预测 → 研究缺乏理论驱动，沦为无目的的数据挖掘，结果的解释缺乏锚点 → 修改方向：基于文献综述中梳理的理论逻辑，提出具体的方向性假设（如"H1: 自我效能感正向预测学业成绩"）

2. **统计方法与研究设计根本性错配**：如用Pearson相关分析声称变量间的因果关系、对5点Likert量表的单题得分做参数检验而不做任何正态性说明、对有序分类变量做方差分析 → 统计结论效度丧失，结果不可信 → 修改方向：根据变量类型（连续/分类/有序）和研究设计（被试间/被试内/相关）选择正确的统计方法

3. **样本量严重不足以支撑所用分析**：如用N=80做包含5个潜变量的SEM、用N=60做包含10个预测变量的多元回归、用N=100做探索性因子分析（量表有40个题目） → 参数估计不稳定，模型拟合指标不可信，结果几乎一定会在重复中失败 → 修改方向：增加样本量或简化模型（减少潜变量/预测变量），并进行检力分析说明当前样本量能检测到多大效应

4. **使用自编问卷但无任何心理测量学证据**：自行编写问卷，未经专家审查、未做预测试、未报告信度效度，直接用于正式施测 → 工具的测量质量完全未知，所有基于该工具的分析结果都没有意义 → 修改方向：至少完成预测试（N≥30）→项目分析→EFA→信度检验的基本流程，或改用已有的成熟量表

5. **实验设计存在根本性缺陷导致内部效度丧失**：如没有控制条件、存在明显的混淆变量未控制（实验组和控制组使用不同材料且实验者不同）、被试内设计未做顺序平衡 → 实验结论不成立 → 修改方向：补充控制条件，消除混淆变量，或明确承认设计缺陷并大幅弱化结论措辞

### A级（严重影响质量）

1. **文献综述未能建构假设的逻辑基础**：文献综述罗列了X的研究和Y的研究，但没有论证X→Y或X→M→Y的理论逻辑。假设突然出现在综述末尾，仿佛凭空产生 → 假设缺乏理论根基，读者无法判断假设的合理性 → 修改方向：在综述中明确论证变量间关系的理论机制（如通过什么理论框架解释X为何影响Y），使假设成为综述的自然推论

2. **只报告p值，不报告效应量**：结果全部以"差异显著（p < .05）""相关显著（p < .01）"呈现，没有Cohen's d、η²、R²、f²等效应量 → 读者无法判断效应的实际大小和实践意义，p值受样本量影响，大样本下微小差异也会显著 → 修改方向：所有推断检验均伴随报告效应量，并参照Cohen（1988）的标准解释效应大小

3. **结果与讨论混淆**：在"结果"章节中出现"这可能是因为……""这说明……"等解释性语言；或在"讨论"章节中出现新的统计分析结果 → 违反心理学论文的基本写作规范，影响论文的清晰性和逻辑性 → 修改方向：结果章节严格只呈现数据分析结果（包括描述统计和推断统计），所有解释和与文献的比较放到讨论章节

4. **不报告不显著结果或对其进行解释性回避**：如假设H2未得到支持，论文对此只字不提或一句"未达到显著水平"带过，不做任何讨论 → 选择性报告损害论文的科学性和诚信度 → 修改方向：如实报告所有分析结果（包括不显著的），在讨论中认真分析不显著结果的可能原因（检力不足？理论模型需修正？测量工具不敏感？）

5. **中介分析缺乏因果序列的理论依据**：问卷调查收集的横截面数据，直接做X→M→Y的中介分析，但未讨论为什么X在因果上先于M、M先于Y → 横截面数据无法确定因果方向，中介效应可能是Y→M→X → 修改方向：（a）提供理论依据说明因果序列的合理性；（b）检验替代模型（如M→X→Y）并比较拟合度；（c）明确承认横截面数据的因果推断局限

6. **Cronbach's α引用原始研究而非当前样本**：写"该量表信度良好（α = .85，张三，2015）"，但不报告本研究样本中的α值 → 信度是样本依赖的指标，原始样本的信度不能代表当前样本的信度 → 修改方向：必须报告当前样本的α值，可同时引用原始研究的α值作为参照

7. **中介分析使用Baron & Kenny的逐步回归法**：仍然采用"三步法"或"四步法"检验中介效应（第一步检验X→Y……） → 该方法检验力低、无法直接检验间接效应、已被学界广泛弃用 → 修改方向：改用Bootstrap方法（Hayes PROCESS宏或Mplus），报告间接效应的Bootstrap 95%置信区间

8. **SEM模型拟合不佳但强行解释**：CFI < .90、RMSEA > .08，论文承认"拟合度不够理想"但仍然解释路径系数并得出结论 → 模型拟合不佳意味着理论模型与数据不匹配，路径系数的解释缺乏基础 → 修改方向：修正模型（需有理论依据）或简化模型或调整测量模型直至拟合达标

### B级（影响说服力）

1. **文献综述过长且缺乏逻辑递进**：综述章节占全文1/3以上，但只是文献的堆砌，没有清晰的逻辑主线引向假设 → 读者无法把握研究脉络 → 修改方向：按"已知→未知→本研究如何填补"的逻辑重构综述，删减与假设无直接关系的文献

2. **人口统计学信息缺失**：被试描述只写"某大学本科生200人"，缺少年龄（M, SD）、性别比例、专业分布等 → 读者无法评估样本特征和研究的外部效度 → 修改方向：编制人口统计学信息表，包含年龄、性别、年级、专业等关键变量的描述统计

3. **图表格式不符合APA规范**：表格有竖线/网格线、图表没有标题、坐标轴没有标签、p值报告为"0.000"而非"< .001" → 影响论文的专业性 → 修改方向：严格遵循APA第7版格式——表格使用三线表（无竖线）、报告精确p值（如p = .034）、p < .001时报告为"< .001"而非"= .000"、统计符号斜体

4. **讨论未与已有文献对话**：讨论只是重述结果，没有将发现与引言中综述的文献进行比较 → 讨论缺乏深度 → 修改方向：对每个主要发现，讨论其与前人研究的一致性或不一致性，并分析原因

5. **局限性讨论过于笼统**：只写"样本量不够大""只选取了某地区样本" → 这些局限是所有研究的共性问题，缺乏针对性 → 修改方向：讨论与本研究设计直接相关的具体局限（如"本研究使用横截面设计无法推断因果关系""所有变量均为自我报告可能存在共同方法偏差""样本为大学生群体，结论能否推广至职业人群有待验证"）

6. **伦理审查未提及**：全文没有任何伦理审批或知情同意的说明 → 心理学研究必须经过伦理审查 → 修改方向：在方法章节补充伦理审批信息和知情同意程序描述

7. **统计结果报告不完整**：如只报告F值和p值，不报告自由度；只报告"回归分析显著"，不报告回归系数B和β → 读者无法评估结果 → 修改方向：遵循APA统计报告标准，如t(58) = 2.34, p = .023, d = 0.61；F(2, 87) = 4.56, p = .013, η² = .095

## 文献综述特殊要求

### 文献来源优先级

1. **第一梯队（必须引用）**：与论文变量直接相关的经典文献和理论原著
   - 中文核心期刊：《心理学报》《心理科学》《心理发展与教育》《心理科学进展》《中国临床心理学杂志》《心理与行为研究》《应用心理学》
   - 外文顶刊：*Psychological Bulletin*, *Psychological Review*, *Journal of Personality and Social Psychology (JPSP)*, *Developmental Psychology*, *Journal of Abnormal Psychology*, *Journal of Experimental Psychology* 系列, *Psychological Science*, *Journal of Consulting and Clinical Psychology*
   - 经典著作/量表原著：如Bandura（1977/1997）自我效能感、Bowlby（1969）依恋理论、Deci & Ryan（2000）自我决定理论、Lazarus & Folkman（1984）压力应对理论等

2. **第二梯队（推荐引用）**：SSCI/CSSCI来源期刊论文、元分析论文、权威综述论文、领域内高被引论文

3. **第三梯队（可以引用）**：其他学术期刊论文、博士论文、学术专著

4. **不可接受**：百度百科、知乎回答、微信公众号、心理学科普文章、没有同行评审的网络内容

### 文献数量参考标准

- 本科论文参考文献一般不少于40篇，其中外文文献不少于10篇
- 近5年文献应占总量的40%以上
- 每个核心变量至少引用3-5篇直接相关的实证研究

### 综述组织方式

- **推荐**：按变量/理论逻辑组织。典型结构为：变量X的相关研究→变量Y的相关研究→X与Y关系的已有研究→理论机制（如中介变量M）→边界条件（如调节变量W）→总结与假设提出
- **可接受**：按理论流派组织（适用于某一构念有多种理论解释的情况）
- **不推荐**：按作者逐一罗列、按"国外研究""国内研究"简单二分

### 各方向关键理论家/框架

审查者可据此判断文献综述的覆盖度：

| 研究方向 | 应知应引的关键人物/框架 |
|---|---|
| 认知心理学 | Kahneman（双加工理论）、Baddeley（工作记忆模型）、Posner（注意网络）、Craik & Lockhart（加工深度）、Anderson（ACT-R） |
| 发展心理学 | Piaget（认知发展阶段）、Vygotsky（最近发展区）、Erikson（心理社会发展）、Bowlby & Ainsworth（依恋理论）、Bronfenbrenner（生态系统理论）、Kohlberg（道德发展） |
| 社会心理学 | Bandura（社会学习理论/自我效能感）、Festinger（认知失调）、Tajfel & Turner（社会认同理论）、Petty & Cacioppo（精细加工可能性模型）、Cialdini（社会影响六原则） |
| 人格心理学 | Costa & McCrae（大五人格）、Eysenck（人格三维度）、Mischel（认知-情感系统理论）、Rogers（自我理论）、Maslow（需要层次） |
| 教育心理学 | Deci & Ryan（自我决定理论）、Dweck（心智模式/成长型思维）、Zimmerman（自我调节学习）、Pintrich（学习动机）、Hattie（可见的学习） |

## 术语与规范

### 术语使用规范

- **核心变量必须在引言中给出操作性定义**，且全文使用统一术语。如确定用"自我效能感"则不能时而写"自信心"时而写"胜任感"
- 区分日常用语与学术术语：不能用"心态好"代替"心理韧性"；不能用"抑郁症"指代"抑郁症状"（前者是临床诊断，后者是自评量表得分）；不能用"性格"与"人格"混用而不加说明
- 外来术语首次出现时标注英文原文，如"自我决定理论（Self-Determination Theory, SDT）""执行功能（Executive Function, EF）"
- **量表名称必须使用正式全称**：首次出现时写全称并标注英文缩写，如"正性负性情感量表（Positive and Negative Affect Schedule, PANAS）"，之后可用缩写
- **统计术语使用规范**：显著性水平用α表示（非a），标准差用SD（非std），均值用M（非X̄，APA推荐），效应量符号使用正确格式（Cohen's *d*, η², *R*², *f*²）

### 引用格式

- **推荐APA第7版格式**（心理学的通行标准）
- 正文中引用使用"作者-年份"制：
  - 单作者：Bandura（1997）或（Bandura, 1997）
  - 双作者：Deci和Ryan（2000）或（Deci & Ryan, 2000）
  - 三人及以上：张三等（2020）或（Zhang et al., 2020）
- 直接引用需标注页码：（Bandura, 1997, p. 36）
- 参考文献列表按作者姓氏字母顺序排列

### 统计报告规范（APA格式）

以下为各类统计结果的标准报告格式，审查时逐一核对：

- **t检验**：*t*(df) = X.XX, *p* = .XXX, *d* = X.XX（或95% CI [X.XX, X.XX]）
- **方差分析**：*F*(df₁, df₂) = X.XX, *p* = .XXX, η² = .XXX（或偏η²）
- **相关**：*r*(df) = .XX, *p* = .XXX（或*r*s表示Spearman）
- **回归**：*B* = X.XX, *SE* = X.XX, *β* = .XX, *t* = X.XX, *p* = .XXX
- **卡方**：χ²(df, *N* = XXX) = X.XX, *p* = .XXX, *V* = .XX
- **p值报告**：报告精确值（如*p* = .034），不写"*p* = .000"（改为*p* < .001），不写"*p* > .05"或"*p* < .05"来代替精确值
- **数字格式**：统计量保留两位小数，p值保留三位小数，百分比保留一位小数
- **表格中的统计符号一律斜体**

### 图表规范

- **表格**：使用APA格式三线表（无竖线、无网格线），只有顶线、栏目线和底线
- **相关矩阵表**：对角线上方或下方呈现相关系数，对角线上可放Cronbach's α值，表下注明*p < .05, **p < .01, ***p < .001
- **回归结果表**：包含B、SE、β、t、p、VIF；模型摘要包含R²、ΔR²、F
- **CFA/SEM路径图**：标注标准化路径系数和显著性水平，报告模型拟合指标
- **所有图表编号连续**，表标题在表上方，图标题在图下方
- **效应量报告**：建议在表格中增加效应量列，而非仅在文中提及

## 结论与建议特殊要求

### 结论的要求

- **逐一回应假设**：结论应逐条回应每个研究假设，明确说明"H1得到支持/未得到支持"。假设的支持与否应基于统计检验结果，不能含糊其辞
- **结论的语气与设计匹配**：
  - 实验研究（尤其随机实验）可使用因果性语言（"X导致了Y的提升"）
  - 相关/问卷研究只能使用关联性语言（"X与Y存在正向关联""X正向预测Y"），**绝不能**写"X导致/引起/促进了Y"
  - 质性研究使用理解性语言（"参与者体验到……""研究发现……的主题"）
- **不显著结果同样报告**：不能只在结论中列出得到支持的假设而回避未得到支持的假设
- **研究局限必须具体**：不接受"本研究存在一定局限性"这样的空话。必须明确指出的局限包括：
  - 研究设计层面（横截面/纵向、实验/准实验/相关）
  - 样本层面（样本特征、抽样方式、样本量）
  - 测量层面（所有变量均为自我报告？量表的某些维度信度偏低？）
  - 分析层面（未控制的变量、替代解释的可能性）

### 实践启示的合理边界

- 心理学论文的"启示"通常包括理论贡献和实践建议两部分
- **理论贡献**：本科论文一般不具备提出新理论的能力，可表述为"本研究为XX理论提供了新的实证证据""本研究拓展了XX理论在中国文化背景下的适用性"
- **实践建议**：必须从研究发现中逻辑推出，不能泛泛而谈
  - 合格示例："本研究发现教师自主支持正向预测学生的内在动机（β = .42），建议教师在课堂教学中增加选择机会、减少控制性语言，以促进学生内在动机的发展"
  - 不合格示例："建议加强学生心理健康教育"——与研究发现无具体关联
- **干预建议**：如果研究发现了某心理变量的保护/风险因素，可提出干预建议，但需注明证据等级（本研究仅为相关证据/实验证据）

## 各子方向特殊审查要点

### 基础心理学

- 实验范式是否为该领域的经典范式（如Stroop范式、Flanker任务、Go/No-Go任务、IAT内隐联想测验）？如果是自创范式，是否有充分的效度论证？
- 反应时数据的预处理是否规范：是否剔除了错误试次的反应时？是否剔除了极端值（如±3SD或±2.5SD）？剔除标准是否事先确定？
- 如果涉及眼动/ERP/fMRI等技术，数据预处理流程（滤波、伪迹剔除、基线校正等）是否详细报告？

### 应用心理学

- 如果研究涉及组织行为/人力资源管理主题（如工作满意度、组织承诺、领导力），是否界定了与管理学视角的区别——心理学视角应聚焦个体心理机制
- 消费心理学、广告心理学等方向需注意研究的生态效度——实验室情境能否推广到真实消费场景

### 教育心理学

- 与教育学论文的区别：教育心理学论文应聚焦心理变量及其机制（如学习动机、自我调节、认知负荷），而非教学方法本身的效果
- 常用理论框架：自我决定理论、自我调节学习理论、成就目标理论、期望-价值理论、认知负荷理论
- 研究对象通常为学生，需注意发展阶段的差异对变量关系的影响

### 发展心理学

- 研究设计是否匹配发展研究的需要：横断研究只能描述年龄差异（不是发展变化），纵向追踪研究才能描述发展轨迹
- 如果涉及儿童被试：测量工具是否适合该年龄段？自我报告量表通常不适用于10岁以下儿童，需使用他评（教师/家长评定）或行为观测
- 年龄分组的依据是否说明？是否有理论或实证基础？

### 临床与咨询心理学

- **临床样本vs一般样本**：是否明确区分？如果研究对象为临床样本（如抑郁症患者），诊断标准（DSM-5/ICD-11）和来源机构必须说明
- **量表的临界分值（cut-off score）**：如果使用量表筛选临床群体，临界分值的选择依据是否说明？（如PHQ-9 ≥ 10为中度抑郁）
- **伦理要求更高**：涉及自杀意念、自伤行为、创伤经历、物质滥用等敏感话题时，伦理审批、知情同意、风险预案、转介机制缺一不可
- **干预研究**：是否有基线期测量？是否有随访（follow-up）数据？干预方案是否标准化（手册化）？
- **个案研究**：临床个案报告是否征得来访者书面同意？是否彻底去识别化？

### 社会心理学

- 实验中的欺骗（deception）是否经过伦理审批？事后解释（debriefing）是否充分？
- 内隐测量（如IAT）与外显测量（如问卷）的结果是否同时报告？如果仅有外显测量，是否讨论了社会赞许性偏差（social desirability bias）的影响？
- 跨文化研究是否讨论了文化差异对理论适用性的影响？（如集体主义vs个人主义文化背景下的差异）
- 启动研究（priming study）需特别注意效应的可重复性——社会心理学的部分经典启动效应在重复危机中未能复现

## 心理学特有审查维度

### 复现危机意识

- 心理学正处于复现危机（replication crisis）时代，审查者应对效应的可靠性保持适度怀疑
- **加分项**（非强制要求，但值得鼓励）：研究预注册、公开数据和分析代码、使用大样本、报告检力分析
- **警惕信号**：大量勉强显著的p值（p = .04x多次出现）、对空假设的大量否认（所有假设都得到支持，无一例外）、效应量异常大且未讨论

### 共同方法偏差的系统性处理

当论文的所有变量均通过自我报告问卷测量时（这是本科心理学论文的最常见情况），必须审查以下内容：

1. **是否在论文中讨论了共同方法偏差的潜在威胁**——完全不提及=B级问题
2. **是否采取了程序控制措施**——如匿名施测、变量间设置无关题目间隔、不同时间点施测不同变量
3. **是否进行了Harman单因子检验**——最低要求的统计检验。未做=A级问题（当所有变量均为自我报告时）
4. **更高级的检验方法**（加分项）：潜在方法因子法、标记变量法

### 测量模型意识

- 审查论文作者是否理解所用量表的因子结构：量表是单维度还是多维度？如果是多维度，作者是使用总分还是维度分？选择依据是否合理？
- 如果量表原始版本为多维度结构，但作者只使用总分做分析，需审查总分的使用是否有先例支持
- 区分反映性指标（reflective）和形成性指标（formative）：大部分心理量表属于反映性指标模型，但某些构念（如社会经济地位SES）可能是形成性指标，此时Cronbach's α不适用
